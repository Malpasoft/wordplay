How to upload on GitHub (iPhone / mobile)?
Desktop: Download the file → go to the correct folder on github.com → drag & drop, or use Add file → Upload files.

iPhone / any mobile (no upload button):
  1. Tap Download file below — it saves to your Files app.
  2. Open github.com in Safari → sign in → go to your repo.
  3. Navigate into each folder shown in the path above (tap each folder name in turn).
  4. Tap the pencil / + icon (top right) → Create new file.
  5. In the filename field, type only the last part of the path (e.g. flashcards.html) — you’re already inside the right folder.
  6. Go to your Files app, open the downloaded file, select all the text and copy it.
  7. Tap inside the file content area on GitHub and paste.
  8. Tap Commit changes → commit directly to main (or your branch).
Tip: If the folder doesn’t exist yet, type the full path in the filename field (e.g. en/b/b1/vocabulary/new-chapter/flashcards.html) — GitHub creates the folders automatically.
