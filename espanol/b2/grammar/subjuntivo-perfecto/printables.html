El Subjuntivo Perfecto y Pluscuamperfecto — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Subjuntivo Perfecto

Concepto Forma Ejemplo
Subj. perfecto: haya + part.haya/hayas/haya/hayamos/hayais/hayan + participioEspero que hayas llegado bien.
Cuando usar subj. perfectoSubord. anterior al presente, en contexto de subj.Es una lastima que no hayas venido.
Subj. pluscuamperfecto: hubiera + part.hubiera/hubieras… + participioNo creia que hubiera llegado tan pronto.
Condicional tipo 3Si + hubiera + part. → habria + part.Si hubiera estudiado, habria aprobado.
NUNCA: si + condicional compuestosi + NUNCA habria/habrias…Incorrecto: *Si habria sabido…
Discurso indirecto en pasadodijo que esperaba que + subj. pluscuamperf.Dijo que esperaba que hubiera comido.
Hubiera = hubieseFormas equivalentes del subj. imperf.Si lo hubiera/hubiese sabido, habria venido.

Errores comunes

Incorrecto Correcto Explicación
Si hubiera sabido, habria venido → Si habria sabido, habria venidoSi hubiera sabido, habria venidoEn el condicional tipo 3, la clausula con si nunca lleva condicional. Lo correcto es siempre si + hubiera + participio.
Espero que has terminado → con indicativoEspero que hayas terminadoTras verbos de esperanza y deseo, la subordinada va en subjuntivo: hayas terminado, no el perfecto de indicativo «has terminado».
No creo que ha llegadoNo creo que haya llegadoLa negacion de certeza exige subjuntivo: haya llegado (subj. perfecto). «Ha llegado» es perfecto de indicativo y no puede usarse tras «no creo que».
Si hubiera sabido, habria sabido (tipo 3 con mismo verbo)Si hubiera sabido, habria venido (otro verbo en la consecuencia)En el tipo 3, la consecuencia expresa que habria ocurrido otra cosa diferente: si hubiera sabido → habria venido (no: si hubiera sabido → habria sabido, que es una tautologia).
Hubiera o hubiese: son diferentesHubiera y hubiese son equivalentesLas formas en -ra (hubiera) y en -se (hubiese) del subjuntivo imperfecto son intercambiables en todos los contextos.

Ejercicio 1 — Elige la forma correcta del subjuntivo compuesto

1. Espero que tu ______ (recibir) mi mensaje de ayer. [has recibido / hayas recibido / hubiera recibido / recibieras]
2. Es una lastima que ellos no ______ (poder) venir a la boda. [han podido / hayan podido / hubieran podido / podrian]
3. Si ______ (saber, yo) la verdad antes, habria actuado de otra forma. [hubiera sabido / habria sabido / haya sabido / supiera]
4. Dijo que no creia que el jefe ______ (firmar) el contrato. [haya firmado / hubiera firmado / habia firmado / firmara]
5. No creo que todavia ______ (terminar, ellos) el informe. [han terminado / hayan terminado / hubieran terminado / terminaran]
6. Avísame en cuanto ______ (leer, tu) el documento. [leas / hayas leido / leyeras / hayas leido]

Ejercicio 2 — Completa con el subjuntivo compuesto correcto

1. Si lo ______ (decir, tu) antes, habria podido ayudarte.  
2. Me alegra que ______ (resolver, vosotros) el problema.  
3. No habia nadie que ______ (ver) lo que paso.  
4. Si ellos ______ (llegar) a tiempo, habrian encontrado un sitio.  
5. Es increible que tu ______ (olvidar) nuestro aniversario.  

Ejercicio 3 — Condicional tipo 3 y subjuntivo compuesto

1. ¿Cual forma el condicional tipo 3 correctamente? [Si habria llegado antes, lo habria visto. / Si hubiera llegado antes, lo habria visto. / Si llegara antes, lo habria visto. / Si hubiera llegado antes, lo hubiera habido visto.]
2. ¿En que se diferencia el subj. perfecto del pluscuamperfecto? [No hay diferencia, son identicos. / El perfecto usa haya y se orienta al presente; el pluscuamperfecto usa hubiera y se orienta al pasado. / El perfecto usa hubiera; el pluscuamperfecto usa haya. / El perfecto es solo para verbos regulares.]
3. ¿Cual usa el subjuntivo perfecto correctamente? [Espero que has descansado. / Espero que hayas descansado. / Espero que hubieras descansado. / Espero que habias descansado.]
4. En discurso indirecto pasado, «Espero que hayas llegado» se transforma en: [Dijo que esperaba que hayas llegado. / Dijo que esperaba que hubieras llegado. / Dijo que espera que hayas llegado. / Dijo que esperaba que has llegado.]
5. ¿Cual de estas frases es equivalente a «Si lo hubiese sabido, habria venido»? [Si lo hubiera sabido, habria venido. / Si lo habria sabido, habria venido. / Si lo supiera, vendria. / Si lo hubiera sabido, vendria.]

Clave de respuestas

Ejercicio 1: 1. hayas recibido 2. hayan podido 3. hubiera sabido 4. hubiera firmado 5. hayan terminado 6. hayas leido

Ejercicio 2: 1. hubieras dicho 2. hayais resuelto 3. hubiera visto 4. hubieran llegado 5. hayas olvidado

Ejercicio 3: 1. Si hubiera llegado antes, lo habria visto. 2. El perfecto usa haya y se orienta al presente; el pluscuamperfecto usa hubiera y se orienta al pasado. 3. Espero que hayas descansado. 4. Dijo que esperaba que hubieras llegado. 5. Si lo hubiera sabido, habria venido.
