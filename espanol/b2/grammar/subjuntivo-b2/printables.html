El Subjuntivo en Oraciones Sustantivas, Adjetivas y Adverbiales — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Subjuntivo: tres clausulas

Concepto Forma Ejemplo
Sustantivas (deseo/emocion/necesidad)querer/esperar/alegrarse de que + subj.Espero que llegues pronto.
Adjetivas (antecedente indefinido)un/una + sust. + que + subj.Busco un piso que tenga terraza.
Adjetivas (antecedente negativo)no hay/nadie/nada + que + subj.No hay nadie que sepa la respuesta.
Temporales con cuando (futuro)cuando + pres. subj. (accion futura)Llama cuando llegues.
Finales con para quepara que + subj. (siempre)Te lo explico para que entiendas.
Concesivas con aunqueaunque + subj. (posibilidad) / indic. (hecho)Aunque llueva, saldre.
Concordancia temporalpres./fut. → subj. pres.; pasado/cond. → subj. imperf.Queria que vinieras.

Errores comunes

Incorrecto Correcto Explicación
Busco un piso que tiene terrazaBusco un piso que tenga terrazaEl antecedente «un piso» es indefinido (no se sabe si existe ese piso especifico): la oracion de relativo exige subjuntivo. El indicativo «tiene» implicaria que el piso ya existe y se conoce.
Llama cuando llegas → referencia al futuroLlama cuando lleguesLas oraciones temporales con cuando y referencia al futuro requieren subjuntivo: cuando llegues. El indicativo «llegas» solo se usa para acciones habituales o pasadas.
Te lo explico para que entiendesTe lo explico para que entiendasLa conjuncion para que siempre exige subjuntivo. Nunca va seguida de indicativo.
Aunque llueve, saldre (queriendo indicar posibilidad)Aunque llueva, saldreSi la lluvia es una posibilidad (no un hecho confirmado), se usa subjuntivo: aunque llueva. El indicativo aunque llueve afirma que en efecto esta lloviendo.
Queria que vienesQueria que vinierasLa concordancia temporal exige que un verbo principal en pasado (queria) vaya seguido de subjuntivo imperfecto, no de subjuntivo presente.

Ejercicio 1 — Elige el modo correcto

1. Busco una secretaria que ______ tres idiomas. [habla / hable / hablara / hablaria]
2. Llama en cuanto ______ (llegar, tu) al hotel. [llegas / llegaras / llegues / llegaras]
3. Queria que todos ______ (venir) a la cena. [vienen / vengan / vinieran / vendrian]
4. No hay nadie en la oficina que ______ (saber) donde esta el jefe. [sabe / sepa / supiera / sabria]
5. Te presto el coche con tal de que lo ______ (cuidar) bien. [cuidas / cuidaras / cuides / cuidabas]
6. Aunque ______ (estar) cansado, termino el trabajo. [este / estoy / estuviera / estaria]

Ejercicio 2 — Escribe la forma correcta del subjuntivo

1. Espero que vosotros ______ (leer) el informe antes de la reunion.  
2. Cuando ______ (terminar, tu) los deberes, puedes salir.  
3. Necesitaba a alguien que ______ (poder) trabajar los fines de semana.  
4. Te lo cuento para que ______ (entender) la situacion.  
5. No conozco a nadie que ______ (hablar) cinco idiomas.  

Ejercicio 3 — Identifica el uso correcto del subjuntivo

1. ¿Que oracion usa el subjuntivo correctamente en una relativa? [Conozco a un medico que habla ruso. / Busco un medico que hable ruso. / Conozco a un medico que hable ruso. / Busco al medico que habla ruso.]
2. ¿Cual es la oracion temporal correcta con referencia al futuro? [Llame cuando llego. / Llamame cuando llegas. / Llamame cuando llegues. / Llamame cuando llegabas.]
3. ¿Cual usa aunque correctamente para indicar posibilidad? [Aunque llueve, ire. / Aunque llueva, ire. / Aunque lloviera, ire. / Aunque ha llovido, ire.]
4. ¿Que estructura sigue obligatoriamente la final para que? [para que + futuro / para que + indicativo / para que + subjuntivo / para que + condicional]
5. Si el verbo principal esta en indefinido, ¿que tiempo de subjuntivo se usa en la subordinada? [Presente de subjuntivo / Futuro de subjuntivo / Imperfecto de subjuntivo / Pluscuamperfecto]
