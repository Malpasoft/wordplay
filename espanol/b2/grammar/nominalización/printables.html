Referencia: La Nominalización

Concepto Forma Ejemplo
-ción / -siónverbo → sustantivo (acción/proceso)producir → producción
-mientoverbo → sustantivo (proceso)crecer → crecimiento
-dad / -tadadj → sustantivo (cualidad abstracta)libre → libertad
-ezaadj → sustantivo (cualidad)rico → riqueza
lo + adjnominalización neutra del adjetivolo importante
infinitivo como sust.el + infinitivo (concepto abstracto)el saber, el poder
de en la nominalizaciónintroduce complemento o agentela aprobación de la ley por el gobierno

Errores comunes

Incorrecto Correcto Explicación
*la decidirla decisiónEl infinitivo como sustantivo lleva artículo masculino: el decidir. Pero para una acción concreta usa el sustantivo nominalizado: la decisión.
*subimiento de preciossubida de preciosEl sustantivo de subir es subida, no *subimiento. Consulta el diccionario si tienes dudas sobre el sustantivo correcto.
*libertedadlibertadEl sufijo es -tad (no -tedad): libre → libertad. Igualmente: lealtad, voluntad, mitad.
*el aprobamiento de la leyla aprobación de la leyAprobar → aprobación (no *aprobamiento). Los verbos en -ar terminados en -ar con vocal larga suelen dar -ción.
nominalizar en excesousa el verbo cuando sea más naturalLa nominalización excesiva hace el texto pesado. Úsala cuando aporte precisión o sea requerida por el registro, no de forma mecánica.

Ejercicio 1 — Nominalización: elige el sustantivo correcto

1. ¿Cuál es el sustantivo de «crecer»? [crecimiento / cresción / crecida / crecedura]
2. ¿Cuál nominaliza correctamente «El gobierno decidió»? [La decisión del gobierno / El decidir del gobierno / La decidura del gobierno / El decidimiento del gobierno]
3. ¿Cuál es el sustantivo de «libre»? [libertad / librez / librería / libertación]
4. Completa: «La _____ de los precios afecta a todos.» (subir) [subida / subición / suba / subimiento]
5. ¿Cuál es la nominalización formal de «los precios subieron»? [la subida de los precios / el subimiento de precios / lo de los precios / el subir de precios]
6. ¿Cuál nominaliza el adjetivo «amargo»? [amargura / amargedad / amargía / amargución]

Ejercicio 2 — Nominaliza las siguientes oraciones

1. Transforma en nominalización: «El director aprobó el proyecto.»  
2. ¿Qué significa «el saber» en «el saber no ocupa lugar»?  
3. Transforma: «El ejército retiró sus tropas.»  
4. ¿Cuál es el sufijo para «valiente → valentía»?  
5. Identifica la nominalización en: «El aumento de la producción fue inesperado.»  

Ejercicio 3 — Nominalización en contexto formal

1. ¿Cuál es el registro más formal? [Ante el crecimiento económico, se revisará la política. / Como la economía creció, se cambiará la política. / Cuando creció la economía, cambiarán la política. / Ya que la economía fue creciendo, cambian la política.]
2. ¿Qué sufijo da sustantivos abstractos de cualidad? [-dad / -tad / -miento / -da / -ida / -ción]
3. «La existencia de este problema…» ¿De qué verbo procede «existencia»? [existir / existenciar / existar / existicionar]
4. ¿Cuál es el sustantivo de «abrir»? [apertura / abertura / ambas son correctas / abrimiento]
5. ¿Qué estructura nominaliza adjetivos de forma neutra? [lo + adjetivo / el + adjetivo / un + adjetivo / muy + adjetivo]

Clave de respuestas
