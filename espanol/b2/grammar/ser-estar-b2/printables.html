SER y ESTAR: Usos Avanzados y Diferencias de Significado — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: SER y ESTAR avanzado

Concepto Forma Ejemplo
ser (identidad/permanente)ser + adj. = cualidad inherente o permanenteEs inteligente. Es espanol. Es medico.
estar (estado/condicion)estar + adj. = estado o condicion temporalEsta cansado. Esta enfermo. Esta listo (= preparado).
Adjetivos que cambianlisto/malo/rico/seguro/vivo/aburridoEs rico (dinero) vs. esta rico (sabor).
Sorpresa/cambio con estarestar + adj. normalmente de ser = cambio notableQue alta esta la nina — cuanto ha crecido.
Ubicacion de eventosevento + ser (no estar) + lugarLa boda es en la catedral.
Ubicacion de cosas/personasobjeto/persona + estar + lugarEl libro esta en la mesa.

Errores comunes

Incorrecto Correcto Explicación
Esta listo = es inteligenteEs listo = es inteligente; esta listo = esta preparadoEl adjetivo listo cambia de significado: es listo (inteligente, cualidad permanente) vs. esta listo (preparado para hacer algo).
El concierto esta en el teatroEl concierto es en el teatroLos eventos se localizan con ser: el concierto es en el teatro. Estar se reserva para la ubicacion de personas y objetos.
Esta malo = es malvadoEs malo = es malvado; esta malo = esta enfermoEs malo describe el caracter o la calidad (malvado, de mala calidad). Esta malo describe un estado temporal de enfermedad.
Es rico el pastel = sabe bienEl pastel esta ricoEl sabor agradable de una comida o bebida se expresa con estar: el pastel esta rico. Ser rico se refiere a la riqueza economica o a la abundancia.
Que guapa es hoy (expresando sorpresa)Que guapa esta hoyPara expresar sorpresa o un cambio notable respecto a lo habitual, se usa estar: que guapa esta hoy. Ser expresa la cualidad de forma neutra y permanente.

Ejercicio 1 — SER o ESTAR: elige la forma correcta

1. La pelicula ______ muy aburrida. (es una pelicula sin interes) [esta / es / era / fue]
2. El nino ______ muy listo; siempre saca las mejores notas. [esta / es / estuvo / fue]
3. ¿Ya ______ lista la cena? Tengo mucha hambre. [es / esta / fue / sera]
4. Este pastel ______ buenísimo; ¿puedo repetir? [es / fue / esta / sera]
5. El festival de jazz ______ en la Plaza Mayor este sabado. [esta / estara / es / sera]
6. Que elegante ______ tu hoy, ¿vas a alguna fiesta? [eres / seras / estas / fuiste]

Ejercicio 2 — Explica la diferencia de significado

1. ¿Que significa «El guarda de seguridad es muy vivo»?  
2. Transforma: «Ese hombre es malo» (malvado) al sentido de «esta enfermo».  
3. ¿Que frase ubica correctamente el evento? El partido ______ en el estadio.  
4. ¿Como se dice que algo sabe delicioso con ser/estar?  
5. ¿Que significa «Estas muy segura de eso»?  

Ejercicio 3 — Elige el significado correcto de ser/estar + adjetivo

1. ¿Que significa «Mi jefe esta muy aburrido hoy»? [Mi jefe causa aburrimiento. / Mi jefe siente aburrimiento. / Mi jefe es una persona aburrida. / Mi jefe ha muerto de aburrimiento.]
2. ¿Cual usa ser/estar correctamente para ubicar personas y eventos? [El museo esta cerrado y la exposicion esta en el museo. / El museo esta cerrado y la exposicion es en el museo. / El museo es cerrado y la exposicion es en el museo. / El museo es cerrado y la exposicion esta en el museo.]
3. ¿Que comunica «Que alto estas» en vez de «Que alto eres»? [Que la persona es siempre muy alta. / Que la persona ha crecido o el hablante nota un cambio. / Que la persona es de origen alto. / No hay diferencia entre las dos frases.]
4. ¿Cual es la diferencia entre «es seguro» y «esta seguro»? ["Es seguro" describe un estado; "esta seguro" es una cualidad permanente. / "Es seguro" describe un lugar sin peligro; "esta seguro" expresa certeza o confianza. / No hay diferencia. / "Es seguro" se usa solo con personas; "esta seguro" solo con cosas.]
5. ¿Cual describe el sabor de una comida correctamente? [La paella es muy rica. / La paella esta muy rica. / La paella fue muy rica. / La paella sera muy rica.]

Clave de respuestas

Ejercicio 2: 1. es vivo = es astuto o listo 2. esta malo 3. es 4. esta rico 5. tiene certeza o confianza
