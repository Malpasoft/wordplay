Las Expresiones Temporales Avanzadas — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Expresiones Temporales

Concepto Forma Ejemplo
Hace + tiempo + que + presenteduración hasta ahora (acción sigue)Hace cinco años que trabajo aquí.
Hace + tiempo + que + indefinidoindica cuándo ocurrió algo (hace X tiempo)Hace tres días que llegó de viaje.
Llevaba + tiempo + gerundioduración de acción en el pasado hasta otro puntoLlevaba una hora esperando cuando llamó.
desde hace + cantidadorigen de la acción: periodo de tiempoEstudio español desde hace dos años.
desde + fecha/eventoorigen de la acción: punto concretoVivo aquí desde enero / desde que me casé.
dentro de + tiempoen X tiempo a partir de ahora (futuro)Estará listo dentro de media hora.
en + tiempoduración para completar una tareaTerminó el informe en menos de una hora.

Errores comunes

Incorrecto Correcto Explicación
Vivo aquí desde un añoVivo aquí desde hace un añoCon una cantidad de tiempo, siempre se usa desde hace: «desde hace un año». «Desde» sola va con fechas o eventos: «desde enero», «desde que llegué».
Hace tres años que llegué (para decir que sigo aquí)Hace tres años que vivo aquíSi la acción continúa en el presente, el verbo debe estar en presente: «Hace tres años que vivo aquí». El indefinido indicaría que la acción ya terminó.
Vendrá en una hora (= a partir de ahora)Vendrá dentro de una horaPara señalar un momento futuro a partir de ahora, se usa dentro de. En mide la duración de una tarea completada: «Lo hice en una hora».
Llevaba esperando dos horas / llevaba dos horas esperadoLlevaba dos horas esperandoEl orden correcto es: llevaba + periodo de tiempo + gerundio. El gerundio nunca se sustituye por el participio en esta construcción.
A partir de ahora todo cambiará → desde ahora en adelanteA partir de ahora todo cambiará (correcto)A partir de + punto temporal indica que algo comienza desde ese momento y continúa. Es correcto; no confundir con «a partir de» en sentido matemático (dividir entre).

Ejercicio 1 — Elige la expresión temporal correcta

1. Lleva aquí ______ tres años. (cantidad de tiempo, acción continua) [desde / desde hace / hace / por]
2. ______ dos horas que esperamos y el técnico no aparece. [Desde / Llevamos / Hace / En]
3. El médico ______ media hora hablando con el paciente antes de darle el diagnóstico. [hacía / había / llevaba / tenía]
4. La conferencia empezará ______ veinte minutos; aún hay tiempo de tomar un café. [en / desde hace / dentro de / hace]
5. ¿Cuánto tardaste en terminar el examen? — Lo hice ______ dos horas exactas. [dentro de / desde hace / en / hace]
6. Trabaja en esa empresa ______ que la fundaron, hace más de diez años. [desde hace / en / desde / hace]

Ejercicio 2 — Completa con la expresión temporal adecuada

1. ______ cinco años que no nos vemos; ¡cuánto tiempo!  
2. Vivo en este barrio ______ quince años; lo conozco mejor que nadie.  
3. ______ tres meses estudiando este tema cuando por fin lo entendí.  
4. El paquete llegará ______ dos días; ya lo han enviado.  
5. Resolvió el acertijo ______ apenas diez minutos; es muy inteligente.  

Ejercicio 3 — Identifica el uso temporal correcto

1. ¿Cuál expresa una acción que comenzó en el pasado y continúa ahora? [Hace dos años llegué aquí. / Hace dos años que vivo aquí. / Llegué aquí en dos años. / Vivo aquí dentro de dos años.]
2. ¿Cuál usa "desde hace" correctamente? [Vivo aquí desde un año. / Vivo aquí desde hace un año. / Vivo aquí hace un año. / Vivo aquí en un año.]
3. ¿Qué significa "Llevaba una hora estudiando cuando sonó el teléfono"? [Estudió durante una hora después de que sonó el teléfono. / Había estado estudiando durante una hora cuando el teléfono interrumpió. / Estudia una hora todos los días. / Estudiará una hora más después de la llamada.]
4. ¿Cuál usa "dentro de" correctamente? [Lo hice dentro de una hora. / Llegó dentro de dos días antes. / El vuelo sale dentro de dos horas. / Vivimos aquí dentro de tres años.]
5. ¿Cuál es la diferencia entre "en" y "dentro de" con expresiones de tiempo? ["En" apunta al futuro; "dentro de" mide la duración de una tarea. / "En" mide cuánto tarda algo en completarse; "dentro de" señala cuándo ocurrirá algo (futuro). / Son sinónimos intercambiables. / "En" solo se usa con acciones pasadas; "dentro de", con futuras.]
