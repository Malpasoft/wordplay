Referencia: Construcciones Absolutas

Concepto Forma Ejemplo
gerundio simultáneoacción simultánea o de modoEntró corriendo.
habiendo + participioacción anterior a la principalHabiendo terminado, salió.
participio absolutosujeto propio + participio — acción anteriorTerminada la reunión, salimos.
al + infinitivosimultaneidad / causa — equivale a cuando/porqueAl llegar, llamó.
de + infinitivo compuestocondición formal — equivale a si hubieraDe haberlo sabido, actuaría distinto.
gerundio como adjsolo en formas lexicalizadas (agua hirviendo)Agua hirviendo (excepción).
restricción de gerundio postnominalusa cláusula relativael libro que contiene errores (no *conteniendo).

Errores comunes

Incorrecto Correcto Explicación
*Un libro conteniendo erroresun libro que contiene erroresEl gerundio en español no se usa como adjetivo postnominal. Usa que + verbo conjugado.
*Habiendo llegado, voy al mercado (futuro)Al llegar, iré al mercadoEl gerundio compuesto (habiendo + participio) expresa anterioridad en el pasado, no en el futuro. Para futuro: al llegar.
*Terminado la reunión…Terminada la reunión…El participio absoluto concuerda con su sujeto en género: la reunión (fem.) → terminada.
Gerundio como sujetoincorrecto en español formalEl gerundio NO es sujeto en español formal: *«Fumar es malo». En registro formal: Fumar es incorrecto pero muy frecuente; se prefiere «El tabaco es malo».
*de haber sabido → de habiendo sabidode haber sabidoLa construcción condicional formal es de + infinitivo compuesto: de haber sabido. No se usa gerundio aquí.

Ejercicio 1 — Construcciones absolutas: elige la correcta

1. «_____ la reunión, todos se fueron.» (terminar) [Terminada / Terminando / Al terminar / Habiendo terminando]
2. «_____ al mercado, recuerda comprar pan.» (ir) [Al ir / Habiendo ido / Ido / Yendo]
3. ¿Cuál es incorrecta? [Un libro conteniendo errores. / Un libro que contiene errores. / Habiendo terminado, salió. / Terminada la clase, salimos.]
4. «_____ los contratos, comenzaron las obras.» (firmar) [Firmados / Firmando / Al firmar / Habiendo firmando]
5. «_____ el trabajo, salió a cenar.» (terminar — acción anterior) [Habiendo terminado / Terminando / Al terminar / Terminado él]
6. «_____ haberlo sabido, habría actuado de otra manera.» [De / Si / Al / Por]

Ejercicio 2 — Reformula con construcción absoluta

1. Reemplaza: «Cuando llegó, llamó al médico.»  
2. Reformula: «Una vez firmados los contratos, comenzaron las obras.»  
3. ¿Cómo corriges «un señor hablando ruso»?  
4. Completa: «_____ este problema, podemos pasar al siguiente.» (resolver — acción anterior, sujeto: el problema)  
5. Transforma en condición formal: «Si lo hubiera visto, te lo habría contado.»  

Ejercicio 3 — Gerundio y participio: uso correcto

1. ¿Cuál es la construcción de participio absoluto correcta? [Aprobado el proyecto, comenzaron las obras. / Aprobando el proyecto, comenzaron las obras. / Al aprobar el proyecto, comenzaron las obras. / Habiendo aprobado el proyecto, comenzaron las obras.]
2. ¿Qué expresa «al + infinitivo»? [Simultaneidad o causa temporal / Acción anterior con sujeto propio / Condición irreal / Modo de hacer la acción]
3. ¿Cuándo se usa el gerundio compuesto (habiendo + participio)? [Para una acción claramente anterior a la principal / Para una acción simultánea / Para modificar un sustantivo / Para condiciones formales]
4. ¿Cuál es correcta? [Dicho esto, se fue. / Diciendo esto, se fue. / Al decir esto, se fue. / Habiendo dicho esto, se fue.]
5. «De haberlo sabido…» equivale a: [Si lo hubiera sabido… / Cuando lo supo… / Al saberlo… / Sabiéndolo…]
