Los Tiempos del Futuro y el Condicional Avanzado — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: El Futuro Avanzado

Concepto Forma Ejemplo
futuro simplepredicción, conjetura (presente), mandato enfático¿Dónde estará? Estará en casa.
futuro perfectohabrá + participio — anterior a otra acción futuraCuando llegues, ya habrá comido.
condicional simplehipótesis tipo 2, futuro del pasado, conjetura pasadaDijo que vendría. / ¿Qué hora sería?
condicional perfectohabría + participio — hipótesis irreal en el pasadoSi hubiera sabido, habría venido.
ir a + infintención/plan inmediatoVoy a llamarle ahora.
futuro concesivoSerá + adj/sust + peroSerá difícil, pero posible.
condicional de cortesíaquerría, podría, debería (atenuación)¿Podría ayudarme?

Errores comunes

Incorrecto Correcto Explicación
*si vendríasi vinieraNUNCA se usa el condicional después de si condicional. Lo correcto es si + subjuntivo imperfecto: si viniera.
*habría venido → hipótesis presentehabría venido = hipótesis pasadaEl condicional perfecto (habría venido) expresa una hipótesis irrealizable en el PASADO. Para el presente: vendría.
confundir futuro de conjetura con predicciónambos usan futuro simpleTanto la predicción (mañana lloverá) como la conjetura (estará en casa) usan futuro simple. El contexto distingue.
*habrá llegado cuando lleguescuando llegues, habrá llegadoEn la oración temporal «cuando + subj», el futuro perfecto expresa la acción completada en ese momento futuro: cuando llegues, ya habrá comido.
*querría = queríaquerría = condicional; quería = imperfectoQuerría es el condicional de querer (atenuación/cortesía). Quería es el imperfecto. No los confundas.

Ejercicio 1 — Futuro y condicional: elige la opción correcta

1. ¿Dónde estará Juan? Creo que _____ en el trabajo. [estará / está / estaría / esté]
2. Si hubiera tenido tiempo, _____ a verte. [habría ido / iría / fui / iré]
3. Para cuando llegues, ya _____ los deberes. [habré terminado / terminaré / habría terminado / termine]
4. Dijo que _____ a las ocho. [vendría / vendrá / venía / venga]
5. ¿Qué hora sería cuando llamaste? _____ las dos de la madrugada. [Serían / Serán / Sean / Eran]
6. _____ caro, pero merece la pena probarlo. [Será / Es / Sea / Sería]

Ejercicio 2 — Forma los tiempos futuros y condicionales

1. Forma el futuro perfecto de «ella / terminar».  
2. ¿Por qué es incorrecto «si vendría»?  
3. ¿Cómo se usa el condicional para la conjetura en el pasado? Da un ejemplo.  
4. Forma el condicional perfecto de «nosotros / poder hacer».  
5. ¿Cuál es la diferencia entre «voy a llamarte» y «te llamaré»?  

Ejercicio 3 — Futuro avanzado: identificación y uso

1. «Serán las cinco.» ¿Qué expresa el futuro aquí? [Conjetura sobre el tiempo presente / Predicción del futuro / Mandato enfático / Valor concesivo]
2. ¿Cuál es correcto en estilo indirecto (pasado)? [Dijo que vendría. / Dijo que vendrá. / Dijo que venía. / Dijo que venga.]
3. «Habrás podido hacerlo mejor.» ¿Qué expresa? [Reproche o lamento / Predicción futura / Conjetura del presente / Hipótesis irreal]
4. ¿Cuál hipótesis es irreal en el pasado? [Si hubiera estudiado, habría aprobado. / Si estudias, aprobarás. / Si estudiara, aprobaría. / Si estudiaste, aprobaste.]
5. ¿Qué forma expresa una acción futura completada antes de otra futura? [El futuro perfecto (habré + participio) / El futuro simple / El condicional perfecto / El subjuntivo perfecto]
