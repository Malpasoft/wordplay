El Subjuntivo con Expresiones de Valoracion y Emocion — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Subjuntivo: valoracion y emocion

Concepto Forma Ejemplo
Valoracion → subjuntivoes importante/necesario/bueno/malo + que + subj.Es importante que llegues a tiempo.
Probabilidad/duda → subjuntivoes posible/probable/dudoso/raro + que + subj.Es posible que llueva manana.
Certeza → indicativoes cierto/verdad/evidente/obvio/claro + que + indic.Es evidente que tiene razon.
Negacion de certeza → subjuntivono es cierto/verdad/evidente + que + subj.No es cierto que venga hoy.
Emocion → subjuntivome alegra/sorprende/preocupa/molesta/gusta + que + subj.Me alegra que estes mejor.
Creer: afirm. → indic., neg. → subj.creo que + indic. / no creo que + subj.No creo que sea tan facil.

Errores comunes

Incorrecto Correcto Explicación
Es posible que vieneEs posible que vengaEs posible que expresa probabilidad, no certeza → exige subjuntivo: venga, no indicativo.
Es cierto que haya un problemaEs cierto que hay un problemaEs cierto que presenta un hecho como verdad → indicativo: hay. El subjuntivo solo aparece cuando se niega: «No es cierto que haya…»
Me alegra que estas bienMe alegra que estes bienLos verbos de emocion con cambio de sujeto exigen siempre subjuntivo: estes, no indicativo.
Es dudoso que vieneEs dudoso que vengaEs dudoso que expresa duda, no certeza → subjuntivo: venga.
No es verdad que tienen → tienen con indicativoNo es verdad que tenganAl negar una expresion de certeza, el modo cambia a subjuntivo: no es verdad que tengan (no «tienen»).

Ejercicio 1 — ¿Indicativo o subjuntivo? Elige la forma correcta

1. Es importante que vosotros ______ (leer) las instrucciones. [leis / leais / leereis / leais]
2. Es cierto que el cambio climatico ______ (afectar) a todo el planeta. [afecte / afecta / afectara / afectaria]
3. Me preocupa que los ninos no ______ (dormir) lo suficiente. [duermen / durmieran / duerman / dormiran]
4. Es probable que el tren ______ (llegar) con retraso. [llega / llegara / llegue / llegaria]
5. No es verdad que ella ______ (mentir) en esa declaracion. [miente / mintio / mienta / mentira]
6. Es evidente que el equipo ______ (necesitar) apoyo. [necesite / necesita / necesitara / necesitaria]

Ejercicio 2 — Completa con indicativo o subjuntivo

1. Es una lastima que no ______ (poder, tu) asistir a la ceremonia.  
2. Es obvio que ellos ______ (tener) mucha experiencia en este campo.  
3. Me sorprende que tu jefe ______ (tomar) esa decision sin consultar.  
4. Es raro que nadie ______ (saber) donde esta el director.  
5. Es verdad que muchas personas ______ (sufrir) las consecuencias de la crisis.  

Ejercicio 3 — Identifica la expresion correcta

1. ¿Cual usa subjuntivo correctamente tras una expresion de valoracion? [Es importante que llega a tiempo. / Es importante que llegue a tiempo. / Es importante que llegara a tiempo. / Es importante que llegaria a tiempo.]
2. ¿Cual usa indicativo correctamente tras una expresion de certeza? [Es evidente que tenga razon. / Es evidente que hubiera tenido razon. / Es evidente que tiene razon. / Es evidente que tuviera razon.]
3. ¿Que ocurre cuando se niega «es cierto que»? [Sigue usando indicativo. / Usa condicional. / Usa subjuntivo. / Usa infinitivo.]
4. ¿Cual expresa emocion correctamente con subjuntivo? [Me alegra que estas aqui. / Me alegra que estes aqui. / Me alegra que estuvieras aqui. / Me alegra que habias venido.]
5. ¿Con que expresion es correcto el indicativo? [Es probable que hay trafico. / Es posible que hay trafico. / Es claro que hay trafico. / Es dudoso que hay trafico.]

Clave de respuestas

Ejercicio 2: 1. puedas 2. tienen 3. haya tomado 4. sepa 5. sufren
