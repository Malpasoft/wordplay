Referencia: La Carta Formal B2

Concepto Forma Ejemplo
Apertura reclamaciónMe veo en la necesidad de...Me veo en la necesidad de reclamar la factura incorrecta.
Exponer datosfecha, referencia, número de pedidoEl día 3 de junio, pedido n.º 45678, por importe de 120 €.
PeticiónLes ruego que... / Solicito que...Les ruego que procedan a la devolución del importe.
InstanciaEXPONE... SOLICITA...EXPONE que ha finalizado el grado. SOLICITA el título.
Vocabulario claveadjuntar, acreditar, expedienteAdjunto la documentación acreditativa de mis méritos.
Cierre formalEn espera de su respuesta...En espera de su resolución favorable, le saluda atentamente.

Errores comunes

Incorrecto Correcto Explicación
Quiero reclamar que el producto no funciona.Me dirijo a ustedes para reclamar que el producto adquirido el día X presenta un defecto de fabricación.La reclamación formal concreta el problema con fecha y detalle
EXIJO que me devuelvan el dinero.Les ruego que procedan a la devolución del importe abonado.En la carta formal se evita el imperativo exigente; se usa el subjuntivo de petición
Le mando esta carta para pedir una beca.Me dirijo a usted con el fin de solicitar la concesión de una beca de estudios.El lenguaje formal usa «solicitar» y evita «pedir» y «mandar»
Mando los documentos para que los vea.Adjunto la documentación requerida para su consideración.«Adjuntar» es el verbo formal para enviar documentos junto a una carta
Espero su respuesta.En espera de su respuesta, aprovecho la ocasión para saludarle atentamente.El cierre formal combina la expectativa de respuesta con la despedida protocolar

Ejercicio 1 — Vocabulario formal

Elige la expresión más adecuada en cada contexto de carta formal.

1. ¿Cómo se dice formalmente «quiero pedir una beca»? [Solicito la concesión de una beca / Quiero que me den una beca / Pido una beca]
2. ¿Qué fórmula se usa para enviar documentos junto a una carta? [Mando los papeles / Adjunto la documentación requerida / Te envío los documentos]
3. ¿Cómo se pide una solución educadamente en una reclamación? [Exijo la devolución / Les ruego que procedan a la devolución / Devuélvanme el dinero]
4. ¿Qué documento contiene el historial académico de un estudiante? [La instancia / El expediente / La resolución]
5. ¿Qué es una resolución institucional? [Una reclamación del ciudadano / Una decisión oficial de la institución / Un plazo de entrega]

Ejercicio 2 — Completa la carta

Elige la palabra correcta para completar cada fragmento de carta formal.

1. ______ la documentación acreditativa de mis méritos académicos.  
2. Les ruego que ______ a la devolución del importe en el plazo más breve posible.  
3. El ______ para presentar la solicitud de beca vence el 15 de julio.  
4. En espera de su ______ favorable, le saluda atentamente.  

Ejercicio 3 — Carta formal: registro correcto

Elige la versión con el registro más apropiado para una carta formal.

1. Para pedir información sobre un proceso de selección: [¿Puedes decirme cómo funciona el proceso? / Me dirijo a usted para solicitar información sobre el proceso de selección. / Hola, quiero saber más sobre el proceso.]
2. Para indicar que envías documentos: [Te mando los papeles. / Adjunto la documentación requerida para su consideración. / Aquí van los documentos.]
3. Para cerrar una carta de reclamación: [Espero que me contestéis. / En espera de su resolución, aprovecho la ocasión para saludarle atentamente. / Hasta luego.]
4. Para exponer un problema: [Mi pedido llegó tarde y estoy muy enfadado. / Me veo en la necesidad de comunicarles que el pedido n.º 3421 llegó con siete días de retraso. / El pedido tardó mucho.]
