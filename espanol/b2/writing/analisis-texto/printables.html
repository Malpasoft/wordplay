Referencia: El Análisis de Texto

Concepto Forma Ejemplo
Tema¿de qué trata el texto?El texto trata sobre la desigualdad educativa.
Idea principal¿qué dice el autor sobre el tema?El autor sostiene que la brecha digital agrava la desigualdad.
Estructuradeductiva / inductiva / dialécticaLa estructura es deductiva: la tesis aparece en el primer párrafo.
Recursosmetáfora / comparación / ironía / enumeraciónEl autor usa la metáfora «el conocimiento es poder» para...
Intención¿qué quiere conseguir el autor?El autor pretende sensibilizar al lector sobre la urgencia del problema.
Cita textualcomo se aprecia en la línea N...Como se aprecia en la línea 5: «la educación no es un privilegio».

Errores comunes

Incorrecto Correcto Explicación
El texto habla de la contaminación.El texto aborda el problema de la contaminación atmosférica en las ciudades.Usa verbos precisos: abordar, tratar, analizar, exponer; evita el vago «hablar de»
El autor usa una metáfora muy bonita.El autor emplea una metáfora para intensificar el efecto dramático de la imagen.Describe el efecto del recurso, no solo su presencia o belleza
En conclusión el texto es interesante.En conclusión, el texto argumenta de manera persuasiva que la acción climática es urgente.La conclusión del análisis interpreta el texto, no opina vagamente sobre él
El significado literal es...El sentido literal es..., pero el autor emplea la ironía para sugerir lo contrario.Distingue siempre entre sentido literal y sentido connotado o irónico
El autor quiere convencer al lector.El autor apela a la emoción del lector mediante el uso de ejemplos personales.Especifica qué estrategia usa el autor para conseguir su propósito

Ejercicio 1 — Elementos del análisis

Elige la respuesta correcta para cada pregunta sobre análisis textual.

1. ¿Qué pregunta responde el tema de un texto? [¿Qué opina el lector? / ¿De qué trata el texto? / ¿Cómo termina el texto?]
2. ¿Cuál es la diferencia entre metáfora y comparación? [La metáfora usa «como» / La comparación usa «como»; la metáfora, no / Son sinónimos exactos]
3. ¿Qué es una estructura inductiva? [La idea principal va al inicio / Los ejemplos preceden a la idea principal / El texto no tiene conclusión]
4. ¿Cuál es la función de una cita textual en el análisis? [Sustituir el argumento propio / Apoyar la interpretación con evidencia del texto / Resumir el texto]
5. ¿Qué verbo es más preciso para referirse al contenido de un texto? [Hablar de / Abordar / Decir]

Ejercicio 2 — Completa el análisis

Escribe la palabra o expresión adecuada para completar el análisis.

1. El texto ______ el problema de la brecha digital en zonas rurales.  
2. La ______ del texto es deductiva: la tesis aparece en el primer párrafo.  
3. El autor emplea la ______ «la ciudad es una jungla» para destacar el caos urbano.  
4. Como se aprecia en la línea 4: «ningún niño debería quedarse sin educación». Esta ______ muestra la posición del autor.  

Ejercicio 3 — Identifica el recurso

Identifica qué recurso lingüístico se usa en cada ejemplo.

1. «El tiempo vuela cuando estás feliz.» [Ironía / Metáfora / Enumeración]
2. «Llegó cansado, hambriento, solo y desesperado.» [Comparación / Metáfora / Enumeración]
3. «¡Qué magnífico servicio! Llevamos tres horas esperando.» [Metáfora / Ironía / Comparación]
4. «Sus ojos brillan como estrellas.» [Metáfora / Ironía / Comparación]
5. «La tierra llora cuando la olvidamos.» [Enumeración / Ironía / Metáfora]

Clave de respuestas

Ejercicio 2: 1. aborda 2. estructura 3. metáfora 4. cita textual

Ejercicio 3: 1. Metáfora 2. Enumeración 3. Ironía 4. Comparación 5. Metáfora
