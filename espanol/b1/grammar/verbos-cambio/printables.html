Los Verbos de Cambio — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Verbos de Cambio

Concepto Forma Ejemplo
ponerse + adjcambio emocional/físico temporal y repentinoSe puso nervioso antes del examen.
volverse + adj/sustcambio profundo de carácter (involuntario)Se volvió muy introvertido.
quedarse + adj/participioestado resultante, a menudo permanenteSe quedó dormido en el sofá.
hacerse + sust/adjcambio gradual y voluntario (profesión/ideología)Se hizo vegetariana el año pasado.
llegar a serlogro alcanzado con esfuerzoLlegó a ser presidente de la empresa.
convertirse entransformación total (también voluntaria)Se convirtió en la mejor del equipo.
resultar + adjresultado de una evaluaciónLa experiencia resultó muy útil.

Errores comunes

Incorrecto Correcto Explicación
*Se volvió rojo de vergüenzaSe puso rojo de vergüenzaPonerse se usa para cambios físicos visibles y temporales como los colores o estados emocionales repentinos. Volverse es para cambios de personalidad/carácter.
*Se puso médicoSe hizo médicoPara cambios de profesión, ideología o religión (graduales y voluntarios) se usa hacerse.
*Se quedó muy nervioso en la reuniónSe puso muy nerviosoReacción emocional repentina → ponerse. Quedarse implica un estado resultante más permanente o involuntario.
*Llegó a ser nerviosoSe puso / se volvió nerviosoLlegar a ser se usa para logros o cambios positivos tras esfuerzo, no para estados emocionales.
*Se hizo dormidoSe quedó dormidoQuedarse dormido es la expresión fija. Hacerse no se usa con participios de este tipo.

Ejercicio 1 — Verbos de cambio: elige la opción correcta

1. Cuando recibió la mala noticia, _____ muy triste. [se puso / se volvió / se quedó / se hizo]
2. Después de años de práctica, _____ un experto en su campo. [llegó a ser / se puso / se quedó / se volvió]
3. Con los años, _____ muy desconfiado — antes era abierto y simpático. [se volvió / se puso / se quedó / se hizo]
4. _____ médica después de estudiar durante diez años. [Se hizo / Se puso / Se quedó / Se volvió]
5. Durante la reunión, _____ dormido sin darse cuenta. [se quedó / se puso / se hizo / llegó a ser]
6. La película _____ muy aburrida a partir de la mitad. [se volvió / se puso / se quedó / llegó a ser]

Ejercicio 2 — Distingue los verbos de cambio

1. ¿Cuál es la diferencia entre «ponerse nervioso» y «volverse nervioso»?  
2. Usa «quedarse» en una oración que exprese un resultado involuntario.  
3. ¿Cuándo usarías «llegar a ser» en vez de «hacerse»?  
4. ¿«Se convirtió en»? ¿Cuándo se usa?  
5. ¿Es correcto «se puso médico»?  

Ejercicio 3 — Verbos de cambio en contexto

1. ¿Cuál es correcto para un cambio físico repentino? [Se puso rojo. / Se volvió rojo. / Se quedó rojo. / Se hizo rojo.]
2. «Se quedó sorprendida.» ¿Qué implica? [Quedó en estado de sorpresa como resultado de algo. / Decidió sorprenderse. / Se sorprendió voluntariamente. / Fue un cambio de carácter.]
3. ¿Cuál NO usa «hacerse»? [Se hizo dormido. / Se hizo vegetariana. / Se hizo rico. / Se hizo famoso.]
4. Para un cambio de ideología (de conservador a progresista), ¿qué verbo es más adecuado? [hacerse / ponerse / quedarse / volverse]
5. «Con el tiempo llegó a ser una referencia en su campo.» ¿Qué implica esto? [Logró ese estatus tras años de esfuerzo. / El cambio fue repentino. / Fue un cambio involuntario. / Cambió de profesión.]
