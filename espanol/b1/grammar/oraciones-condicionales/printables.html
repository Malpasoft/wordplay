Las Oraciones Condicionales — Si real e hipotético — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Oraciones Condicionales

Concepto Forma Ejemplo
Tipo 1 (real/posible)Si + presente indicativo → futuro/imperativoSi estudias, aprobarás el examen.
Tipo 2 (hipotético)Si + imperfecto subjuntivo → condicionalSi tuviera coche, iría en coche.
Regla absolutaNUNCA: si + condicionalNUNCA: *Si tendría… SIEMPRE: Si tuviera…
Subjuntivo imperfecto -AR-ara, -aras, -ara, -áramos, -arais, -aranSi hablara más, aprendería más.
Subjuntivo imperfecto -ER/-IR-iera, -ieras, -iera, -iéramos, -ierais, -ieranSi comiera menos, estaría mejor.
Como sicomo si + subj. imperfecto (comparación irreal)Habla como si supiera todo.

Errores comunes

Incorrecto Correcto Explicación
Si tendría más tiempo, estudiaría más.Si tuviera más tiempo, estudiaría más.Nunca se usa el condicional después de «si». Para la hipótesis, la estructura correcta es si + imperfecto de subjuntivo: si tuviera.
Si tienes tiempo, me llamas. (usando futuro)Si tienes tiempo, llámame o me llamas.Tras «si + presente», la consecuencia puede ser futuro, imperativo o también presente (hechos generales). El futuro «llamarás» también es posible.
Como si sabe / como si sabe todoComo si supiera todo«Como si» exige siempre subjuntivo imperfecto: como si supiera, nunca indicativo.
Si yo sería rico…Si yo fuera rico…La forma correcta para hipótesis con SER es el subjuntivo imperfecto: si fuera. Nunca «si sería».
Si llueva mañana…Si llueve mañana…Para condiciones reales o posibles, se usa el presente de indicativo tras «si», no el subjuntivo.

Ejercicio 1 — Elige la forma correcta en oraciones condicionales

1. Si ______ (tener, tú) fiebre, debes ir al médico. [tendrías / tengas / tienes / tuvieras]
2. Si tuviera más dinero, ______ (comprar, yo) un apartamento. [compro / compraré / compraría / comprara]
3. ______ un error usaría el condicional después de «si». [Si vendría, / Si viniera, / Si vendrá, / Si vendrías,]
4. Si ______ (poder) elegir, viviría cerca del mar. [podré / puedo / pudiera / podría]
5. Habla ______ (como si) lo supiera todo. [como que / como si / si como / como que si]
6. Si ______ (llover) mañana, cancelaremos el partido. [lloviera / llovería / llueve / lloverá]

Ejercicio 2 — Completa la oración condicional

1. Si (tener, yo) ______ tiempo esta tarde, iré a verte.  
2. Si (saber, tú) ______ la respuesta, escríbela en el papel.  
3. Si yo (ser) ______ el director, cambiaría muchas cosas.  
4. Ella habla (como si / conocer) ______ a todo el mundo.  
5. Si (estudiar, vosotros) ______ más, obtendríais mejores notas.  

Ejercicio 3 — Identifica la oración correcta

1. ¿Cuál es una condicional de tipo 1 correctamente formada? [Si tendría dinero, viajaría. / Si tienes hambre, come algo. / Si tuvieras tiempo, vendrías. / Si comería más, estaría mejor.]
2. ¿Cuál es una condicional de tipo 2 correctamente formada? [Si tuviera más experiencia, conseguiría ese trabajo. / Si tendría más experiencia, conseguiría ese trabajo. / Si tengo más experiencia, conseguiré ese trabajo. / Si tenga más experiencia, conseguiría ese trabajo.]
3. ¿Cuál contiene el error más típico con las condicionales? [Si llueve mañana, no saldremos. / Si tuviera más tiempo, estudiaría más. / Si tendría más tiempo, estudiaría más. / Si hace frío, me abrigaré.]
4. ¿Qué estructura sigue «como si»? [como si + futuro / como si + indicativo presente / como si + condicional / como si + subjuntivo imperfecto]
5. ¿Cuál expresa una condición hipotética? [Si llueve, me mojo. / Si llueve, me mojaré. / Si lloviera, me mojaría. / Si llueve, mójate.]

Clave de respuestas

Ejercicio 1: 1. tienes 2. compraría 3. Si viniera, 4. pudiera 5. como si 6. llueve

Ejercicio 2: 1. tengo 2. sabes 3. fuera 4. como si conociera 5. estudiarais

Ejercicio 3: 1. Si tienes hambre, come algo. 2. Si tuviera más experiencia, conseguiría ese trabajo. 3. Si tendría más tiempo, estudiaría más. 4. como si + subjuntivo imperfecto 5. Si lloviera, me mojaría.
