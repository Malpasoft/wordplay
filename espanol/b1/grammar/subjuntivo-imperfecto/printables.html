Referencia: Subjuntivo Imperfecto

Concepto Forma Ejemplo
FormaciónEllos (indefinido) − ron + -ra/-ras/-ra/-ramos/-rais/-ranhablaron → hablara, hiciéron → hiciera
Tilde en nosotrosSiempre tilde: -áramos / -iéramoshabláramos, comiéramos, tuviéramos
Condicional tipo 2Si + subj. imp. + condicionalSi pudiera, viviría en el campo.
Ojalá + subj. imp.Deseo poco probable o imposibleOjalá tuviera más tiempo.

Subordinada en pasadoVerbo principal pasado → subj. imp.Quería que lo hicieras tú.

Errores comunes

Incorrecto Correcto Explicación
Si tengo → Si tuviera (confundir condicional tipo 1 y tipo 2)Tipo 1: si + presente; Tipo 2: si + subjuntivo imperfecto«Si tengo tiempo, voy» (tipo 1: probable). «Si tuviera tiempo, iría» (tipo 2: hipotético/irreal). No mezcles: nunca «si tendría» o «si tenía» en la condición.
Nosotros habláramos sin tildehabláramos (con tilde)La forma nosotros del imperfecto de subjuntivo siempre lleva tilde: habláramos, comiéramos, fuéramos.
Ojalá + indicativo para deseos poco probablesOjalá + subjuntivo imperfecto«Ojalá tenía más tiempo» es incorrecto. Usa el subjuntivo: «Ojalá tuviera más tiempo» (poco probable) o «Ojalá tenga más tiempo» (posible).
Dijera → deciera (regularizar el irregular)dijera (raíz del indefinido: dij-)DECIR es irregular: ellos dijeron → imperfecto subj.: dijera. Nunca «deciera», ya que la raíz en subjuntivo viene del indefinido, no del infinitivo.
Como si + indicativoComo si + subjuntivo imperfecto«Habla como si es el director» es incorrecto. «Como si» siempre exige subjuntivo: «Habla como si fuera el director».

Ejercicio 1 — Elige la forma correcta de subjuntivo imperfecto

Selecciona la forma adecuada del imperfecto de subjuntivo.

1. Si ______ (tener, yo) más tiempo, leería más. [tengo / tuviera / tendría]
2. Ojalá ______ (poder, tú) venir a la fiesta. [puedes / pudieras / podrías]
3. El profesor quería que nosotros ______ (estudiar) más. [estudiamos / estudiáramos / estudiábamos]
4. Se comporta como si ______ (ser) el dueño del lugar. [es / fuera / sería]
5. ¿Cuál es el imperfecto de subjuntivo de DECIR para él? [deciera / dijera / diría]
6. Le pedí que me ______ (llamar) antes de salir. [llama / llamara / llamaría]

Ejercicio 2 — Escribe la forma de subjuntivo imperfecto

Conjuga el verbo entre paréntesis en pretérito imperfecto de subjuntivo.

1. Si (saber, yo) ______ la respuesta, te lo diría.  
2. Ojalá (venir, ellos) ______ a visitarnos este verano.  
3. Me alegró mucho que tú (estar) ______ bien.  
4. Habla como si (conocer) ______ a todo el mundo.  
5. Quería que nosotros (ir) ______ juntos a la conferencia.  

Ejercicio 3 — Subjuntivo imperfecto en contexto

Elige la oración que usa correctamente el imperfecto de subjuntivo.

1. ¿Cuál expresa una hipótesis irreal? [Si tengo dinero, viajo. / Si tuviera dinero, viajaría. / Si tendría dinero, viajaría.]
2. ¿Cuál usa «ojalá» correctamente para un deseo poco probable? [Ojalá tengo tiempo. / Ojalá tendré tiempo. / Ojalá tuviera tiempo.]
3. ¿Cuál usa «como si» correctamente? [Gasta como si es millonario. / Gasta como si fuera millonario. / Gasta como si será millonario.]
4. ¿Cuál es la forma correcta de QUERER para nosotros en subj. imp.? [quisiéramos / queramos / quisiéremos]
5. ¿Cuál reporta correctamente una orden en pasado? [Me dijo que venir pronto. / Me dijo que viniera pronto. / Me dijo que vendría pronto.]

Clave de respuestas

Ejercicio 2: 1. supiera 2. vinieran 3. estuvieras 4. conociera 5. fuéramos
