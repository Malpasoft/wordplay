Referencia: Pronombres Indefinidos

Concepto Forma Ejemplo
algo / nadacosas (afirm. / neg.)¿Quieres algo? — No, no quiero nada.
alguien / nadiepersonas (afirm. / neg.)¿Llamó alguien? — No, no llamó nadie.
algún(o)/a / ningún(o)/aparte o cero de un grupo¿Tienes algún libro? — No tengo ninguno.
algunos/algunasvarios de un grupoAlgunos estudiantes ya terminaron.
todo/a/os/asla totalidadTodos los días estudio media hora.
cadadistribución individual (invariable)Cada semana hay una clase nueva.
Doble negaciónno + verbo + negativoNo he visto a nadie en toda la tarde.

Errores comunes

Incorrecto Correcto Explicación
No hay nadie → Nadie no hayNo hay nadie (orden correcto)Con doble negación, el «no» va antes del verbo: No hay nadie. «Nadie no hay» es incorrecto en español estándar.
Ningunas personas vinieronNinguna persona vinoNinguno/ninguna se usa casi siempre en singular. Para expresar «ni una sola persona», se dice «ninguna persona».
Algo persona / alguien cosaalguien (personas) / algo (cosas)Algo se refiere a cosas; alguien, a personas. No se intercambian.
Cada días / cadas semanacada día / cada semanaCada es invariable: nunca lleva -s. Se dice «cada día», «cada semana», «cada estudiante».
¿Tienes ningún problema?¿Tienes algún problema?En preguntas sin negación explícita, se usan los afirmativos: algún, algo, alguien. Los negativos (ningún, nada, nadie) requieren «no» antes del verbo.

Ejercicio 1 — Elige el indefinido correcto

1. ¿Quieres ______ de comer? Hay bocadillos. [nada / algo / nadie / ninguno]
2. No hay ______ en la sala de espera; está vacía. [alguien / algo / nadie / ningún]
3. ______ de mis amigos habla cuatro idiomas. [Ningún / Nada / Alguien / Alguno]
4. No tengo ______ duda sobre el resultado. [alguna / nada / ninguna / nadie]
5. Viene ______ día a visitarnos; no sé exactamente cuándo. [ningún / algún / nada / nadie]
6. ______ los participantes recibieron un certificado al final del curso. [Alguno / Cada / Todos / Ninguno]

Ejercicio 2 — Completa con el indefinido adecuado

1. No vi a ______ conocido en la fiesta; no había nadie de mi clase.  
2. ¿Sabes si llamó ______ mientras yo estaba fuera?  
3. ______ semana hay reunión del equipo; es una norma fija.  
4. No entendí ______ de lo que dijiste; habla más despacio, por favor.  
5. Tengo ______ preguntas sobre el tema; ¿puedo hacértelas ahora?  

Ejercicio 3 — Uso correcto de los indefinidos

1. ¿Cuál usa la doble negación correctamente? [Nadie no vino a la reunión. / No vino nadie a la reunión. / Vino no nadie a la reunión. / No nadie vino a la reunión.]
2. ¿Cuál es incorrecto? [No tengo ningún problema. / No tengo ningunas ideas. / No tengo ninguna duda. / No tengo ningún dinero.]
3. "Algo" se refiere a… [personas desconocidas / cosas o hechos no especificados / todos los miembros de un grupo / una cantidad exacta]
4. ¿Qué significa «Ambos equipos jugaron bien»? [Todos los equipos jugaron bien. / Ningún equipo jugó bien. / Los dos equipos jugaron bien. / Cada equipo jugó bien.]
5. ¿Cuál usa «algún» correctamente? [Tengo algún libros interesantes. / Tengo algún idea nueva. / Tienes algún momento libre esta tarde? / ¿Tienes algún momento libre esta tarde?]
