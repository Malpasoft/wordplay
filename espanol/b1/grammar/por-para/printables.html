POR y PARA — Dos preposiciones, muchos usos — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: POR y PARA

Concepto Forma Ejemplo
POR: causalo hice por ti / gracias porGracias por tu ayuda en el proyecto.
POR: duraciónpor + periodo de tiempoViví en Londres por tres años.
POR: movimientopasar/caminar porPaseamos por el centro histórico.
POR: agente (pasiva)fue escrito/hecho porLa sinfonía fue compuesta por Mozart.
PARA: propósitoestudiar para / para + infinitivoEstudio para mejorar mi nivel.
PARA: destino/plazosalir para / entregar paraEl informe es para el viernes.
PARA: destinatario/opiniónun regalo para / para míPara mí, es la mejor opción.

Errores comunes

Incorrecto Correcto Explicación
Salgo para Madrid por el trenSalgo para Madrid en trenEl medio de transporte usa en (en tren, en avión), no «por». POR indica el medio de comunicación: por teléfono, por correo.
Lo hice para ti → causaLo hice por ti (causa), para ti = para tu beneficio«Por ti» expresa la causa («a causa de ti»). «Para ti» indica el destinatario («para tu beneficio/uso»).
Estudié para dos horasEstudié por dos horasLa duración usa POR: estudié por dos horas. PARA marca el plazo límite: necesito esto para las dos.
Trabajo por una empresaTrabajo para una empresaEl empleo (quién te contrata) usa PARA. POR se usa para intercambio: trabajo por un salario.
Para mí, creo que… / Para mi opiniónPara mí, … (sin «creo que» adicional)«Para mí» ya introduce la opinión. No es necesario añadir «creo que» a continuación — sería redundante.

Ejercicio 1 — POR o PARA: elige la preposición correcta

1. Gracias ______ tu ayuda con el proyecto. [por / para / en / de]
2. El avión sale ______ Buenos Aires a las seis. [por / para / hacia / en]
3. Esta novela fue escrita ______ una autora colombiana. [para / de / por / en]
4. Necesito terminar el informe ______ el jueves. [por / en / para / hasta]
5. Paseamos ______ la orilla del río durante una hora. [para / por / hasta / hacia]
6. Compré estas flores ______ mi madre, es su cumpleaños. [por / en / de / para]

Ejercicio 2 — Completa con por o para

1. Estudio español ______ trabajar en Latinoamérica.  
2. Lo hice todo ______ amor, no ______ dinero.  
3. Hablamos ______ teléfono durante media hora.  
4. Este regalo es ______ ti; espero que te guste.  
5. Gana veinte euros ______ hora de trabajo.  

Ejercicio 3 — Identifica el uso correcto

1. ¿Cuál expresa duración con POR correctamente? [Estudié para dos horas. / Estudié por dos horas. / Estudié en dos horas. / Estudié durante para dos horas.]
2. ¿Cuál usa PARA para indicar propósito? [Trabajo por esta empresa. / Viajo por Madrid. / Compré un diccionario para aprender vocabulario. / Gracias para tu ayuda.]
3. ¿Qué significa «Lo hice por ti»? [Lo hice con el objetivo de dártelo. / Lo hice a causa de ti. / Te lo di a ti. / Salí hacia tu casa.]
4. ¿Cuál usa POR para indicar el agente de la voz pasiva? [El cuadro fue pintado para Picasso. / El cuadro fue pintado por Picasso. / El cuadro fue pintado en Picasso. / El cuadro fue pintado de Picasso.]
5. ¿Cuál expresa opinión personal con PARA? [Según mí, es correcto. / Para mí, es la mejor solución. / Por mí, es la mejor solución. / En mí opinión, es correcto.]
