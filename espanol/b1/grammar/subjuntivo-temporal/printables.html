Referencia: Subjuntivo Temporal

Concepto Forma Ejemplo
cuando + futurosubjuntivoCuando llegues, avísame.
hasta que + futurosubjuntivoEspera hasta que termine.
antes de quesiempre subjuntivoLlama antes de que sea tarde.
en cuanto / tan pronto comosubjuntivo (futuro)En cuanto puedas, ven.
después de quesubj. (futuro) / indic. (pasado)Hablaremos después de que termines.
mientras + futurosubjuntivoLlámame mientras estés allí.
habitual/pasadoindicativoCuando llego, como. / Cuando llegué, comí.

Errores comunes

Incorrecto Correcto Explicación
Cuando llegaráscuando lleguesCon referencia futura, la oración temporal usa subjuntivo, no futuro. «Cuando llegues» (no: *cuando llegarás).
Hasta que llego (futuro)hasta que llegueExpresando un límite futuro, se usa subjuntivo: hasta que llegue. Indicativo se reserva para hábito o pasado.
antes de que llegaantes de que llegueAntes de que usa siempre subjuntivo, nunca indicativo.
En cuanto puedes, ven (orden para el futuro)en cuanto puedasLa orden/petición hacia el futuro requiere subjuntivo en la temporal: en cuanto puedas.
después de que + futurodespués de que + subjuntivoEn la temporal futura se usa subjuntivo, no indicativo ni futuro: después de que termines.

Ejercicio 1 — Subjuntivo o indicativo en temporales

1. «Cuando _____ (llegar, tú), avísame.» [llegues / llegas / llegarás / llegaste]
2. «Espera hasta que _____ (terminar, yo).» [termine / termino / terminaré / terminé]
3. «Cuando era niño, siempre _____ (jugar) en el parque.» [jugaba / juegue / jugará / jugara]
4. «Llama antes de que _____ (ser) tarde.» [sea / es / será / fue]
5. «En cuanto _____ (poder, tú), mándame el documento.» [puedas / puedes / podrás / pudiste]
6. «Después de que _____ (comer, ellos), saldremos.» [coman / comen / comerán / comieron]

Ejercicio 2 — Completa con la forma correcta

1. Completa: «Mientras _____ (estar, tú) aquí, hablaremos.»  
2. ¿Cuándo se usa indicativo con «cuando»? Da un ejemplo.  
3. Transforma usando «antes de que»: «Llama. Luego llego yo.»  
4. Completa: «En cuanto _____ (terminar, él) el trabajo, vendrá.»  
5. ¿Cuál es la diferencia entre «antes de» y «antes de que»?  

Ejercicio 3 — Identificación del modo temporal

1. ¿Cuál es correcta? [Llámame cuando llegues. / Llámame cuando llegarás. / Llámame cuando llegas. / Llámame cuando llegaste.]
2. «Cuando tengo hambre, como.» ¿Por qué está en indicativo? [Porque expresa un hábito general. / Porque es futuro. / Porque es un error. / Porque sigue a «cuando» siempre.]
3. ¿Qué conjunción usa SIEMPRE subjuntivo? [antes de que / cuando / mientras / hasta que]
4. «Esperaré hasta que _____.» ¿Cuál es correcta? [termines / terminarás / terminas / terminaste]
5. ¿Cuál es el modo en «Después de que llegaron, comimos»? [indicativo (pasado ya ocurrido) / subjuntivo imperfecto / indicativo futuro / condicional]

Clave de respuestas

Ejercicio 1: 1. llegues 2. termine 3. jugaba 4. sea 5. puedas 6. coman

Ejercicio 2: 1. estés 2. cuando expresa hábito o pasado: Cuando llegué, comí. 3. Llama antes de que llegue. 4. termine 5. Antes de + infinitivo (mismo sujeto); antes de que + subjuntivo (distinto sujeto)

Ejercicio 3: 1. Llámame cuando llegues. 2. Porque expresa un hábito general. 3. antes de que 4. termines 5. indicativo (pasado ya ocurrido)
