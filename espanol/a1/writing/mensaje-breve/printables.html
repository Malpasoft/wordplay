Referencia: El Mensaje Breve

Concepto Forma Ejemplo
SaludoHola / Oye / Buenos díasHola, ¿cómo estás?
Estadoestar + adjetivoEstoy cansado.
Ubicaciónestar en + lugarEstoy en el trabajo.
Info. clave¿qué? + ¿cuándo? + ¿dónde?Llego a las 5, en la puerta.
Pregunta¿Vienes? / ¿Dónde estás?¿Quedamos a las 6?
DespedidaBesos / Un abrazo / Hasta luegoHasta luego. Besos.

Errores comunes

Incorrecto Correcto Explicación
Hola, soy bien.Hola, estoy bien.Los estados usan estar, no ser
¿Cuándo vienes? escribir mañanaEscribe con mayúscula: Mañana voy.Empieza la frase con mayúscula
Hasta luego! (sin apertura)¡Hasta luego!El signo de exclamación de apertura ¡ es obligatorio en español
Estamos de casa.Estamos en casa.Ubicación: estar en + lugar
Cuídate., (coma al final)Cuídate. (sin coma)El punto cierra la frase, la coma no

Ejercicio 1 — Elige la forma correcta

Elige la mejor opción para cada hueco.

1. ¿Cómo ______ ? Bien, gracias. [estás / eres / tienes]
2. Mamá, ______ en casa de Ana. [estoy / soy / hay]
3. ______ , María. ¿Quedamos mañana? [Hola / Hasta luego / Besos]
4. Llego tarde. ______ en el metro. [Estoy / Soy / Tengo]
5. Un abrazo, ______ (cierre afectuoso) [Ana / Hola Ana / ¿Ana?]
6. ______ de casa a las 8. (salgo) [Salgo / Estoy / Soy]

Ejercicio 2 — Completa el mensaje

Escribe la palabra que falta. No hacen falta tildes.

1. ______ , Pedro. ¿Cómo estás? (saludo informal)  
2. ______ en casa. No puedo salir hoy. (ubicación)  
3. Llegamos ______ las 7. (preposición de hora)  
4. Hasta ______ , besos. (despedida)  
5. ______ mañana a las 5 en el parque? (pregunta de propuesta)  

Ejercicio 3 — Verdadero o falso

Lee las afirmaciones y elige la opción correcta.

1. En español se puede escribir "¿Vienes?" sin el signo de apertura ¿ [Falso: el ¿ de apertura es obligatorio / Verdadero: el ¿ es opcional]
2. Para decir dónde estás usas el verbo SER. [Falso: se usa ESTAR / Verdadero: se usa SER]
3. «Cuídate» es un saludo de apertura. [Falso: es una despedida / Verdadero: es un saludo]
4. Un mensaje breve debe responder a ¿qué?, ¿cuándo? y ¿dónde? [Verdadero / Falso: solo a ¿qué?]
5. El verbo ESTAR se usa para estados temporales. [Verdadero / Falso: ESTAR es solo para ubicaciones]
