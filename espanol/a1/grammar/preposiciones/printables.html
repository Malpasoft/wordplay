Preposiciones básicas — En, de, a, con, por, para, entre — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Preposiciones básicas

Concepto Forma Ejemplo
enlugar / medio de transporte / tiempo parcial del díaVivo en Madrid. Voy en metro. Estudio en verano.
deorigen / posesión / material + de + el = delSoy de México. El libro de Ana. La mesa es de madera.
adirección / hora / a personal + a + el = alVoy al trabajo. A las ocho. Llamo a mi madre.
concompañía / instrumento / modoVoy con Ana. Escribo con el bolígrafo.
por / parapor = causa / duración · para = finalidad / destinatario / plazoGracias por todo. Esto es para ti.
entreposición entre dos elementos / acción colectivaEstá entre la tienda y el banco. Entre tú y yo.

Errores comunes

Incorrecto Correcto Explicación
Voy a el trabajo.Voy al trabajo.A + el = AL (contracción obligatoria). No se puede decir «a el»; siempre «al».
Vengo de el banco.Vengo del banco.De + el = DEL (contracción obligatoria). Nunca «de el»; siempre «del».
Estudio para hablar con los locales.Estudio para hablar con los locales.Esta frase es CORRECTA. Para + infinitivo expresa finalidad. Es un error frecuente dudar de esta estructura.
Gracias para tu ayuda.Gracias por tu ayuda.Con «gracias», siempre usamos POR (causa del agradecimiento), nunca para. «Gracias por tu ayuda» es la expresión fija.
El libro es de el profesor.El libro es del profesor.De + el = DEL. La contracción es obligatoria excepto cuando el artículo forma parte de un nombre propio (Vengo de El Salvador).

Ejercicio 1 — Elige la preposición correcta

Selecciona la preposición más adecuada para cada oración.

1. Vivo ______ un barrio muy tranquilo ______ Madrid. [en / de / a / en / de / en]
2. Voy ______ trabajo en metro todos los días. [al / a el / del]
3. Este regalo es ______ mi madre. [para / por / de]
4. Gracias ______ todo lo que has hecho. [por / para / de]
5. El banco está ______ la farmacia y la zapatería. [entre / con / en]
6. Llamo ______ mi abuelo todos los domingos. [a / de / para]

Ejercicio 2 — Escribe la preposición correcta

Completa con la preposición adecuada.

1. Vengo ______ el supermercado. Tengo mucha comida. (de + el = ?)  
2. La clase de español empieza ______ las diez de la mañana.  
3. Esta mochila es ______ cuero, no ______ plástico. (material x2)  
4. Normalmente estudio ______ las mañanas, no por las tardes.  
5. ¿Quieres venir al parque ______ nosotros?  

Ejercicio 3 — POR o PARA: el contraste clave

Elige la preposición correcta: por o para.

1. Estudio español ______ hablar con la familia de mi novio. [para / por / de]
2. Te llamo ______ preguntarte algo importante. [para / por / a]
3. Pagué veinte euros ______ esta camisa. [por / para / de]
4. Necesito el informe ______ el lunes. [para / por / en]
5. Muchas gracias ______ la información. [por / para / de]

Clave de respuestas
