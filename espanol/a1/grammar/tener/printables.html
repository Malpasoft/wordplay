El verbo TENER — Posesión, edad y sensaciones — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: El verbo TENER

Concepto Forma Ejemplo
Posesióntener + sustantivoTengo un libro nuevo.
Edadtener + número + añosElla tiene doce años.
Hambre / sedtener hambre · tener sedTenemos hambre. ¿Tienes agua?
Frío / calortener frío · tener calorLos niños tienen frío hoy.
Sueño / prisatener sueño · tener prisaTengo mucho sueño. Date prisa.
Negaciónno tener + sustantivoNo tengo tiempo ahora mismo.

Errores comunes

Incorrecto Correcto Explicación
Soy veinte años.Tengo veinte años.La edad se expresa con TENER, nunca con ser. Recuerda: tener + número + años.
Estoy hambre.Tengo hambre.Las sensaciones físicas usan TENER, no estar. Tengo hambre / sed / frío / calor / sueño.
Yo tengo caliente.Yo tengo calor.Se dice «tener calor», no «tener caliente». Calor es el sustantivo correcto en esta expresión.
Ellos tiene un coche.Ellos tienen un coche.Con ellos/ellas/ustedes la forma es «tienen» (con -n final), no «tiene».
¿Cuántos años tienes tú? — Tengo años veinte.— Tengo veinte años.El número va antes de «años»: tengo VEINTE AÑOS, no «años veinte».

Ejercicio 1 — Elige la forma correcta de TENER

Selecciona la opción adecuada para completar cada oración.

1. Mi hermano ______ diecisiete años. [tiene / tengo / tienen]
2. Nosotros ______ mucha hambre después del partido. [tenemos / tienen / tenéis]
3. ¿Vosotros ______ las entradas del concierto? [tenéis / tienen / tenemos]
4. Ellas ______ frío porque no llevan abrigo. [tienen / tiene / tenéis]
5. Yo ______ mucho sueño. Son las dos de la madrugada. [tengo / tienes / tiene]
6. ¿Tú ______ sed? Hay zumo en la nevera. [tienes / tiene / tengo]

Ejercicio 2 — Escribe la forma correcta de TENER

Conjuga el verbo tener según el sujeto indicado.

1. ¿Cuántos años ______ (tú)?  
2. Mi abuelo ______ setenta y dos años y ______ muy buena salud. (él / él)  
3. Los estudiantes no ______ tiempo para descansar. (ellos)  
4. Nosotras ______ prisa; el tren sale en cinco minutos. (nosotras)  
5. ¿Vosotros ______ miedo de los exámenes? (vosotros)  

Ejercicio 3 — TENER en contexto

Elige la opción correcta según el significado de la oración.

1. Son las once de la noche y los niños ______ mucho sueño. [tienen / tienen hambre / tienen calor]
2. Hace 35 grados. Yo ______. [tengo calor / tengo frío / tengo sueño]
3. ¿Cuántos años ______ tu profesora? [tiene / es / está]
4. Mi padre ______ una tienda de ropa en el centro. [tiene / hay / está]
5. Hemos corrido diez kilómetros. ______ mucha sed. [Tenemos / Somos / Estamos]
