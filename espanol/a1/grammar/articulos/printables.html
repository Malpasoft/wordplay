Referencia: Los Artículos

Concepto Forma Ejemplo
el / losmasc. determinadoel libro / los libros
la / lasfem. determinadola casa / las casas
un / unosmasc. indeterminadoun perro / unos perros
una / unasfem. indeterminadouna silla / unas sillas
de + el = delcontracción obligatoriael coche del profesor
a + el = alcontracción obligatoriavoy al parque

Errores comunes

Incorrecto Correcto Explicación
El agua está fría. (correcto?)¡Sí, el agua es correcto!"Agua" es femenino pero usa "el" en singular para evitar el hiato (a-a). En plural: las aguas.
Soy una profesora.Soy profesora.Después de SER, la profesión va sin artículo indefinido, a menos que lleve adjetivo.
Voy a el mercado.Voy al mercado."A + el" se contrae obligatoriamente en "al".
Es libro de el estudiante.Es el libro del estudiante."De + el" se contrae obligatoriamente en "del".
La problema es difícil.El problema es difícil."Problema" es masculino aunque termina en -a. Muchos sustantivos de origen griego terminados en -ma son masculinos.

Ejercicio 1 — Elige el artículo correcto

Selecciona el artículo adecuado para cada oración.

1. ¿Dónde está ______ llave? (yo sé de qué llave hablamos) [la / una / un]
2. Tengo ______ gato muy simpático. [un / el / la]
3. ______ niños juegan en el parque. [Los / Unos / El]
4. Voy a comprar ______ manzanas. [unas / los / el]
5. Mi hermano es ______ médico. [médico (sin artículo) / un / el]
6. Necesito hablar con ______ director. [el / un / la]

Ejercicio 2 — Completa con el artículo correcto

Escribe el artículo determinado o indeterminado correcto.

1. ______ problema es muy difícil. (un problema específico)  
2. Voy ______ cine esta tarde. (a + el)  
3. Este es el libro ______ profesor. (de + el)  
4. Tengo ______ pregunta. (una pregunta, no específica)  
5. ______ agua está muy fría. (el agua conocida)  

Ejercicio 3 — Contracciones y concordancia

Elige la opción correcta.

1. Voy a comprar el pan. Voy ______ panadería. [a la / al / a el]
2. Hablo ______ director del colegio. [con el / con al / con del]
3. Este es el coche ______ jefe. [del / de el / de la]
4. Necesito ______ información importante. [una / un / el]
5. ______ exámenes son mañana. [Los / Las / Unos]

Clave de respuestas

Ejercicio 1: 1. la 2. un 3. Los 4. unas 5. médico (sin artículo) 6. el

Ejercicio 2: 1. El 2. al 3. del 4. una 5. El

Ejercicio 3: 1. a la 2. con el 3. del 4. una 5. Los
