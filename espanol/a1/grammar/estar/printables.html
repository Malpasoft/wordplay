Referencia: El Verbo ESTAR

Concepto Forma Ejemplo
yoestoyEstoy en casa.
túestás¿Estás bien?
él / ella / ustedestáEl museo está cerrado.
nosotros/asestamosEstamos contentos.
vosotros/asestáis¿Estáis listos?
ellos / ellas / ustedesestánEstán en el parque.

Errores comunes

Incorrecto Correcto Explicación
La oficina es en el centro.La oficina está en el centro.La ubicación siempre usa ESTAR, nunca SER.
Yo estoy médico.Yo soy médico.La profesión usa SER, no ESTAR.
Ella esta cansada.Ella está cansada."Está" lleva tilde: é-s-t-á. Sin tilde es la demostración "esta" (this).
Estamos feliz.Estamos felices.El adjetivo debe concordar con el sujeto. "Nosotros" → "felices" (plural).
¿Donde estas?¿Dónde estás?"Dónde" interrogativo lleva tilde. "Estás" también lleva tilde en la segunda persona singular.

Ejercicio 1 — Elige la forma correcta de ESTAR

Selecciona la forma correcta de ESTAR.

1. Yo ______ muy cansado hoy. [estoy / soy / estás]
2. ¿Dónde ______ el supermercado? [está / es / estoy]
3. Mis amigos ______ en la biblioteca. [están / estamos / estáis]
4. Nosotros ______ muy contentos con el resultado. [estamos / somos / están]
5. ¿Cómo ______ tú? [estás / eres / está]
6. El agua ______ muy fría. [está / es / estoy]

Ejercicio 2 — Conjuga el verbo ESTAR

Escribe la forma correcta de ESTAR.

1. Ella ______ en el hospital. (ESTAR)  
2. Nosotros ______ muy cansados. (ESTAR)  
3. ¿Vosotros ______ bien? (ESTAR)  
4. Los niños ______ en el colegio. (ESTAR)  
5. Yo ______ listo para empezar. (ESTAR)  

Ejercicio 3 — SER o ESTAR: ¿cuál es correcto?

Elige el verbo correcto para cada oración.

1. El libro ______ encima de la mesa. [está / es / soy]
2. Mi padre ______ abogado. [es / está / estoy]
3. ¿Cómo ______ la sopa? — Está deliciosa. [está / es / sois]
4. Yo ______ de España. [soy / estoy / eres]
5. Los estudiantes ______ nerviosos antes del examen. [están / son / sois]
