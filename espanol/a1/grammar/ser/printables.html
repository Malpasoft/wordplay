Referencia: El Verbo SER

Concepto Forma Ejemplo
yosoyYo soy español.
túeresTú eres muy amable.
él / ella / ustedesElla es profesora.
nosotros/assomosSomos estudiantes.
vosotros/assoisVosotros sois de Madrid.
ellos / ellas / ustedessonSon muy inteligentes.

Errores comunes

Incorrecto Correcto Explicación
Yo so estudiante.Yo soy estudiante.La primera persona singular de SER es "soy", no "so".
Ella es en Madrid.Ella está en Madrid.SER no se usa para ubicación. Usa ESTAR + en para indicar dónde está alguien.
Somos cansados.Estamos cansados.Los estados temporales (cansado, enfermo, triste) usan ESTAR, no SER.
Carlos es médico muy bueno.Carlos es un médico muy bueno.Cuando el sustantivo va calificado por un adjetivo, necesita el artículo indefinido.
Ellos es simpáticos.Ellos son simpáticos.Con sujeto plural (ellos/ellas/ustedes) la forma es "son", no "es".

Ejercicio 1 — Elige la forma correcta de SER

Selecciona la forma adecuada de SER para cada oración.

1. Mi madre ______ doctora. [es / eres / soy]
2. ¿De dónde ______ tú? [eres / soy / es]
3. Nosotros ______ de Colombia. [somos / sois / son]
4. Los estudiantes ______ muy trabajadores. [son / somos / es]
5. Vosotros ______ mis mejores amigos. [sois / son / somos]
6. Yo ______ ingeniero. [soy / es / eres]

Ejercicio 2 — Conjuga el verbo SER

Escribe la forma correcta de SER para cada oración.

1. Ella ______ muy simpática. (SER)  
2. ¿Tú ______ de Perú? (SER)  
3. Nosotros ______ profesores. (SER)  
4. Los chicos ______ altos. (SER)  
5. Yo ______ estudiante de español. (SER)  

Ejercicio 3 — SER para describir: ¿correcto o incorrecto?

Elige la oración correcta.

1. ¿Cuál oración es correcta? [Ana es de Madrid. / Ana está de Madrid. / Ana soy de Madrid.]
2. ¿Qué describe el verbo SER? [Una ubicación temporal / Una característica permanente / Un estado de ánimo]
3. Carlos ______ médico. ¿Qué forma va? [es / está / hay]
4. ¿Cuál oración usa SER correctamente? [Somos cansados. / Somos estudiantes. / Somos en casa.]
5. Mi hermana ______ alta y rubia. ¿Qué verbo? [es / está / tiene]
