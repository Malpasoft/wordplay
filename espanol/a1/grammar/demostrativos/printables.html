Referencia: Los Demostrativos

Concepto Forma Ejemplo
este/estacerca del hablanteeste libro / esta mesa
estos/estascerca del hablante (plural)estos libros / estas mesas
ese/esacerca del oyenteese coche / esa silla
esos/esascerca del oyente (plural)esos coches / esas sillas
aquel/aquellalejos de ambosaquel edificio / aquella montaña
aquellos/aquellaslejos de ambos (plural)aquellos árboles / aquellas casas
esto / eso / aquelloneutros (sin sustantivo)¿Qué es esto? / Eso es verdad.

Errores comunes

Incorrecto Correcto Explicación
Este libro o este libro o esta libro?Este libro (masculino)."Libro" es masculino → "este" (masculino singular), no "esta".
Estos sillas son nuevas.Estas sillas son nuevas."Sillas" es femenino plural → "estas".
¿Que es esto? (sin tilde)¿Qué es esto?"¿Qué?" interrogativo siempre lleva tilde.
Aquel chica es mi prima.Aquella chica es mi prima."Chica" es femenino → "aquella" (no "aquel").
Este es correcto siempre con objetos cercanos?Sí, este = cerca del hablante."Este" señala lo que está cerca de quien habla; "ese" cerca del oyente; "aquel" lejos de ambos.

Ejercicio 1 — Elige el demostrativo correcto

Selecciona el demostrativo adecuado.

1. ______ café está muy caliente. (el café está aquí, cerca de mí) [Este / Ese / Aquel]
2. ______ chica de allí es mi amiga. (lejos de los dos) [Aquella / Esta / Esa]
3. ¿Puedes pasarme ______ sal? (la sal está cerca de ti) [esa / esta / aquella]
4. ______ libros son muy interesantes. (los libros están aquí, conmigo) [Estos / Esos / Aquellos]
5. ______ montañas de allá son los Pirineos. [Aquellas / Esas / Estas]
6. ¿Qué es ______? — No sé, parece un insecto. (objeto desconocido) [esto / este / esta]

Ejercicio 2 — Completa con el demostrativo correcto

Escribe el demostrativo correcto.

1. ______ zapatos son muy caros. (los zapatos están aquí, plural masculino)  
2. ______ película es muy aburrida. (la película la estamos viendo juntos, femenino singular)  
3. Mira ______ casa de allí. (lejos, femenino singular)  
4. ¿Puedes coger ______ carpeta? (la carpeta está cerca de ti, femenino singular)  
5. ______ es lo que me preocupa. (referencia a una situación, neutro)  

Ejercicio 3 — Identifica el uso correcto

Elige la oración correcta.

1. ¿Cuál es correcta? [Este ordenador es mío. / Esta ordenador es mío. / Estos ordenador es mío.]
2. ¿Cuál es correcta? [Aquellas flores son preciosas. / Aquel flores son preciosas. / Aquella flores son preciosas.]
3. ¿Cuándo usamos "esto", "eso" o "aquello"? [Cuando no sabemos el género del objeto o nos referimos a una idea / Siempre antes de un sustantivo / Solo en preguntas]
4. Quiero ______ libro que está en el estante de allí. [aquel / este / ese]
5. ______ son mis amigos. (están aquí conmigo, masculino plural) [Estos / Esos / Aquellos]

Clave de respuestas

Ejercicio 2: 1. Estos 2. Esta 3. aquella 4. esa 5. Eso
