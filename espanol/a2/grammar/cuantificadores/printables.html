Referencia: Los Cuantificadores

Concepto Forma Ejemplo
mucho/a + sust.cantidad grandemucho tiempo / mucha gente / muchos libros
mucho (adv.)modifica verbosTrabaja mucho. Me gusta mucho.
muy + adj./adv.intensificadormuy alto, muy bien, muy rápido
poco/a + sust.cantidad pequeñapoco dinero / poca agua
demasiado/aexcesodemasiado ruido / demasiada sal
bastante/scantidad suficiente o considerablebastante comida / bastantes personas
todo el / toda latotalidadtodo el día / toda la semana
algún / ningúnexistencia / ausencia¿Algún problema? / Ningún problema.

Errores comunes

Incorrecto Correcto Explicación
Es muy alto mucho.Es muy alto."Muy" y "mucho" no se combinan. "Muy" modifica adjetivos; "mucho" modifica verbos o sustantivos.
Trabajo muy.Trabajo mucho."Muy" no puede ir solo ante un verbo. Usa "mucho" para modificar verbos.
Hay muchos gente.Hay mucha gente."Gente" es femenino singular → "mucha" (no "muchos").
Tengo bastante libros.Tengo bastantes libros."Bastante" concuerda en número: "bastantes" ante sustantivo plural.
No tengo algún problema.No tengo ningún problema.En oraciones negativas, "algún" pasa a ser "ningún".

Ejercicio 1 — Elige el cuantificador correcto

Selecciona el cuantificador adecuado.

1. Hay ______ personas en la cola. [muchas / mucho / muy]
2. Habla ______ rápido. No le entiendo. [muy / mucho / muchos]
3. No tengo ______ dinero este mes. [mucho / muchos / muy]
4. ¿Tienes ______ bolígrafo para prestarme? [algún / ningún / mucho]
5. Este trabajo es ______ difícil. [muy / mucho / demasiados]
6. Como ______ por las noches. No es bueno. [demasiado / demasiados / muy]

Ejercicio 2 — Muy o mucho

Escribe la forma correcta del cuantificador.

3. Tengo ______ (mucho, fem.) hambre.  
4. Hay ______ tráfico hoy.

Ejercicio 3 — Todo, alguno, ninguno

Elige la forma correcta.

1. ______ los estudiantes han aprobado. [Todos / Todo / Toda]
2. No tengo ______ problema con eso. [ningún / algún / mucho]
3. Pasé ______ la noche estudiando. [toda / todas / todo]
4. ¿Tienes ______ idea de dónde está? [alguna / ninguna / mucha]
5. ¿Hay ______ sillas libres? [algunas / algún / varios]
