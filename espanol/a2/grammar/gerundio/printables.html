El Gerundio y Estar + Gerundio — Acciones en progreso — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Gerundio

Concepto Forma Ejemplo
Formación -ARraíz + -andotrabajar → trabajando / hablar → hablando
Formación -ER/-IRraíz + -iendocomer → comiendo / vivir → viviendo
Irregulares con cambio vocálicoe→i / o→u en raízpedir → pidiendo / dormir → durmiendo
Estar + gerundioacción en progreso ahoraEstoy estudiando para el examen.
Seguir + gerundioacción que continúaSigue lloviendo — no salgas.
Llevar + tiempo + gerundioduración de la acción hasta ahoraLlevo dos horas esperando el autobús.

Errores comunes

Incorrecto Correcto Explicación
Estoy sabiendo la respuesta.Sé la respuesta.Los verbos de estado como saber, conocer, tener, querer, poder no se usan normalmente con estar + gerundio porque ya expresan un estado. Se usa el presente simple.
Llevo esperando tres horas.Llevo tres horas esperando.La estructura correcta de llevar + gerundio es: llevar + expresión de tiempo + gerundio. El tiempo va entre llevar y el gerundio.
Estoy comido.Estoy comiendo.Estar para acciones en progreso requiere el gerundio (-ando/-iendo), no el participio (-ado/-ido). Estoy comido significaría algo como "estoy consumido/gastado", no "estoy comiendo".
Leyendo el libro, me dormí. → "Leyiéndo el libro..."LeyendoEl gerundio de leer es leyendo (no leyiendo). Cuando la raíz termina en vocal, la terminación es -yendo sin la i inicial adicional.
Sigo a estudiar español.Sigo estudiando español.Seguir en perífrasis verbal se construye con gerundio, no con infinitivo precedido de a. La estructura correcta es seguir + gerundio.

Ejercicio 1 — Ejercicio 1 — Selección múltiple: Formación del gerundio

Elige la forma correcta del gerundio.

1. ¿Cómo se forma el gerundio de leer? [leyendo / leendo / leyiendo]
2. Ahora mismo mi jefe _____ (hablar) por teléfono. [está hablando / es hablando / estaba hablando]
3. ¿Cómo se forma el gerundio de pedir? [pidiendo / pediendo / pidendo]
4. Los estudiantes _____ (seguir) preparando el proyecto — lo entregarán la semana que viene. [siguen preparando / siguen a preparar / están siguiendo preparar]
5. El gerundio de dormir es: [durmiendo / dormiendo / durmendo]
6. _____ (llevar) dos años _____ (estudiar) español y ya hablo bastante bien. [Llevo... estudiando / Llevo... a estudiar / Sigo... estudiando dos años]

Ejercicio 2 — Ejercicio 2 — Escribe el gerundio o la forma verbal correcta

Completa las oraciones con la forma correcta del gerundio o la perífrasis verbal.

1. El niño _____ (dormir) plácidamente cuando llegamos a casa.  
2. Mis padres _____ (llevar) diez años _____ (vivir) en esta ciudad.  
3. Todavía _____ (seguir, yo) _____ (buscar) trabajo — el mercado está muy difícil.  
4. ¿Qué _____ (hacer, vosotros) este fin de semana? — _____ (visitar) a los abuelos.  
5. María entró en la habitación _____ (cantar) una canción muy bonita.  

Ejercicio 3 — Ejercicio 3 — Corrección de errores con el gerundio

Identifica el error en cada oración y elige la versión correcta.

1. "Llevo esperando media hora aquí." [Llevo media hora esperando aquí. / Llevo esperando a media hora aquí. / Estoy esperando media hora aquí.]
2. "Estoy sabiendo la respuesta ahora mismo." [Sé la respuesta. / Estoy conociéndola ahora mismo. / Estoy sabiéndola.]
3. "Siguen a trabajar hasta las nueve todas las noches." [Siguen trabajando hasta las nueve todas las noches. / Siguen trabajar hasta las nueve todas las noches. / Están siguiendo trabajar hasta las nueve.]
4. "El gerundio de decir es dicendo." [El gerundio de decir es diciendo. / El gerundio de decir es deciendo. / El gerundio de decir es dicho.]
5. "Leyiéndo el periódico, me enteré de la noticia." [Leyendo el periódico, me enteré de la noticia. / Leiendo el periódico, me enteré de la noticia. / Leendo el periódico, me enteré de la noticia.]
