Referencia: Comparativos

Concepto Forma Ejemplo
Superioridadmás + adj/adv + queEste hotel es más cómodo que el anterior.
Inferioridadmenos + adj/adv + queEl tren es menos rápido que el avión.
Igualdad (adj)tan + adj + comoHoy hace tan frío como ayer.
Igualdad (sust)tanto/a/os/as + sust + comoElla tiene tanta experiencia como él.
Comparativo irregularmejor / peor / mayor / menor + queEsta solución es peor que la primera.
Superlativo relativoel/la/los/las + más/menos + adj + deEs el museo más visitado de España.
Superlativo absolutoadj + -ísimo/a/os/asLa obra de teatro estuvo buenísima.

Errores comunes

Incorrecto Correcto Explicación
más mejormejorMejor ya es un comparativo; añadir más es redundante e incorrecto. Di simplemente: Este método es mejor que el anterior.
tan grande como yo esperabatan grande como yo esperaba (correcto, pero cuidado con tanto)Con adjetivos usa tan + adj + como. Con sustantivos usa tanto/a/os/as + sust + como. El error común es usar tan con sustantivos: *tan libros como — incorrecto.
el más bueno de la claseel mejor de la claseCon los comparativos irregulares (mejor, peor, mayor, menor) no se usa más. La forma correcta en superlativo relativo es el mejor, no el más bueno.
muy altísimoaltísimo o muy alto-ísimo ya expresa el grado máximo; combinar con muy es una redundancia incorrecta. Usa uno u otro, no los dos juntos.
Es el mejor estudiante en la claseEs el mejor estudiante de la claseDespués del superlativo relativo se usa la preposición de, no en, para introducir el grupo de referencia: el mejor de la clase, el más alto del equipo.

Ejercicio 1 — Ejercicio 1 — Selección múltiple: Comparativos

Elige la opción correcta para completar cada oración.

1. El metro es _____ rápido _____ el autobús. [más... que / tan... como / tanto... como]
2. Mi hermana es _____ alta _____ yo. [tan... como / más... de / tan... que]
3. Esta película es la _____ interesante _____ todas. [más... de / más... que / mejor... de]
4. El nuevo profesor es _____ que el anterior — explica mucho mejor. [más bueno / mejor / buenísimo]
5. Hoy tengo _____ trabajo _____ ayer — estoy agotado. [más... que / tanto... como / tan... como]
6. La sopa que hizo mi abuela estaba _____. [muy buenísima / buenísima / más buena]

Ejercicio 2 — Ejercicio 2 — Escribe la forma correcta

Completa las oraciones con la forma comparativa o superlativa correcta del adjetivo entre paréntesis.

1. El Aconcagua es la montaña _____ (alto) de América del Sur.  
2. Este examen fue _____ (fácil) — terminé en diez minutos.  
3. Mi abuela es _____ (mayor) que mi abuelo — tiene tres años más.  
4. Este camino es _____ (largo) que el otro — tardamos dos horas más.  
5. La nueva versión del programa es _____ (malo) que la anterior — tiene muchos errores.  

Ejercicio 3 — Ejercicio 3 — Corrección de errores

Cada oración contiene un error. Elige la versión corregida.

1. "Esta novela es más mejor que la anterior." [Esta novela es mejor que la anterior. / Esta novela es más buena que la anterior. / Esta novela es la mejor que la anterior.]
2. "Tengo tan problemas como tú." [Tengo tantos problemas como tú. / Tengo tan muchos problemas como tú. / Tengo más problemas que tú.]
3. "Es el restaurante más bueno en la ciudad." [Es el mejor restaurante de la ciudad. / Es el más bueno restaurante de la ciudad. / Es el restaurante mejor en la ciudad.]
4. "Esta tarea es muy facilísima." [Esta tarea es facilísima. / Esta tarea es muy fácil y facilísima. / Esta tarea es la más fácil.]
5. "Mi hermano es menor que yo — solo tiene seis años." [Mi hermano es menor que yo — solo tiene seis años. / Mi hermano es más pequeño que yo — solo tiene seis años. / Mi hermano es más menor que yo — solo tiene seis años.]

Clave de respuestas

Ejercicio 1: 1. más... que 2. tan... como 3. más... de 4. mejor 5. más... que 6. buenísima

Ejercicio 2: 1. más alta 2. facilísimo 3. mayor 4. más largo 5. peor
