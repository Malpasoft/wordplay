El Futuro Simple — Hablar del futuro — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Futuro Simple

Concepto Forma Ejemplo
Verbos regularesinfinitivo + -é/-ás/-á/-emos/-éis/-ánMañana hablaré con el director.
TENERtendr- + terminacionesTendré una reunión importante.
PODERpodr- + terminaciones¿Podrás venir mañana?
HACERhar- + terminacionesElla hará un viaje el verano que viene.
IRir- + terminacionesIremos a la playa en agosto.
Predicción / suposiciónFuturo para adivinar¿Dónde estará mi móvil?

Errores comunes

Incorrecto Correcto Explicación
yo hablare (sin tilde)yo hablaréLa forma yo del futuro lleva tilde en la última sílaba: hablaré. Sin tilde es subjuntivo imperfecto en algunos contextos o simplemente incorrecto.
él tendrá → tendaráél tendráTENER → raíz irregular tendr-, no «tendrá» derivado de «tener» completo. El infinitivo entero no se mantiene.
yo iré → yo ireyo iréIR en futuro: iré. La raíz es ir- + terminaciones normales. Lleva tilde.
nosotros hablaremos (con tilde)nosotros hablaremos (sin tilde)La forma nosotros es la única sin tilde en futuro simple: hablaremos, comeremos.
yo hacere / yo haceréyo haréHACER tiene raíz irregular en futuro: har-. Por eso es haré, no «haceré».

Ejercicio 1 — Elige la forma de futuro correcta

Selecciona el futuro simple correcto.

1. Mañana (hablar, yo) ______ con mi jefe. [hablo / hablé / hablaré]
2. El año que viene ellos (vivir) ______ en París. [vivieron / vivirán / viven]
3. ¿(Poder, tú) ______ ayudarme esta tarde? [pudiste / puedes / podrás]
4. Nosotros (hacer) ______ una fiesta el sábado. [hacemos / haremos / hicimos]
5. Ella (tener) ______ que estudiar mucho para el examen. [tiene / tuvo / tendrá]
6. ¿Adónde (ir, vosotros) ______ de vacaciones? [vais / fuisteis / iréis]

Ejercicio 2 — Escribe el verbo en futuro simple

Conjuga el verbo entre paréntesis en futuro simple.

1. Pronto (abrir) ______ el nuevo museo de arte.  
2. Este verano (viajar, nosotros) ______ por toda España.  
3. ¿Cuándo (venir, tú) ______ a visitarme?  
4. Mañana (salir, yo) ______ temprano de casa.  
5. Los científicos (descubrir) ______ muchas cosas nuevas.  

Ejercicio 3 — Futuro o presente

Elige qué tiempo verbal es el correcto según el contexto.

1. Prometo que ______ a tiempo. (llegar, yo) [llego / llegué / llegaré]
2. ¿Dónde ______ mis gafas? (estar) [suposición] [estaban / estarán / están]
3. ¿Cuál es el futuro de HACER para yo? [haceré / haré / hicé]
4. ¿Cuál es la forma correcta de TENER para ella en futuro? [tendrá / tenerá / tuvo]
5. El próximo mes nosotros ______ un nuevo compañero. (tener) [teníamos / tenemos / tendremos]

Clave de respuestas

Ejercicio 1: 1. hablaré 2. vivirán 3. podrás 4. haremos 5. tendrá 6. iréis

Ejercicio 2: 1. abrirá 2. viajaremos 3. vendrás 4. saldré 5. descubrirán

Ejercicio 3: 1. llegaré 2. estarán 3. haré 4. tendrá 5. tendremos
