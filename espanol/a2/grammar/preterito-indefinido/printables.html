Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Pretérito Indefinido

Concepto Forma Ejemplo
Verbos -AR (yo)-éYo hablé con mi madre.
Verbos -ER/-IR (yo)-íYo comí tarde. / Yo viví allí.
SER / IRfui, fuiste, fue, fuimos, fueronFui al cine. / Fue un buen día.
TENERtuve, tuviste, tuvo, tuvimos, tuvieronTuve un accidente ayer.
ESTARestuve, estuviste, estuvo, estuvimos, estuvieronEstuve en Lima tres días.
HACERhice, hiciste, hizo, hicimos, hicieronElla hizo la tarea sola.

Errores comunes

Incorrecto Correcto Explicación
yo habléyo hablé (con tilde)La primera persona singular de -AR lleva tilde: hablé, llegué, empecé. Sin tilde cambia el significado o es incorrecto.
yo hableyo habléSin tilde, «hable» es subjuntivo presente, no indefinido. La tilde es obligatoria en la forma yo.
él hicó / él hacéél hizoHACER es completamente irregular. La raíz cambia a hic- y en tercera persona singular se escribe hizo (con z).
nosotros hablaronnosotros hablamosLa forma nosotros de los verbos -AR es -amos, igual en presente e indefinido. -aron es solo para ellos/ellas.
yo fueyo fuiLa primera persona de SER/IR es fui, no fue. Fue corresponde a él/ella/usted.

Ejercicio 1 — Elige la forma correcta

Selecciona la forma de pretérito indefinido que corresponde.

1. Ayer yo ______ (hablar) con mi jefa. [hablo / hablé / hablaba]
2. El sábado, ellos ______ (comer) en un restaurante nuevo. [comieron / comían / comen]
3. ¿Dónde ______ (estar, tú) anoche? [estabas / estuviste / estás]
4. Nosotros ______ (ir) al museo el domingo. [íbamos / fuimos / vamos]
5. María ______ (hacer) un pastel para la fiesta. [hació / hacía / hizo]
6. Yo no ______ (poder) dormir bien esa noche. [podía / pude / puedo]

Ejercicio 2 — Escribe la forma correcta

Escribe el verbo entre paréntesis en pretérito indefinido.

1. Ayer (tener, yo) ______ mucho trabajo.  
2. El año pasado (vivir, ellos) ______ en Barcelona.  
3. ¿(Ser, vosotros) ______ estudiantes en esa universidad?  
4. Anoche (hacer, nosotros) ______ los deberes juntos.  
5. Ella (llamar) ______ tres veces pero no contesté.  

Ejercicio 3 — ¿Indefinido o no?

Elige la oración que usa correctamente el pretérito indefinido.

1. ¿Cuál usa el indefinido correctamente? [Ayer comía pizza. / Ayer comí pizza. / Ayer como pizza.]
2. ¿Cuál está en indefinido? [Siempre tuve hambre de niño. / Tuve hambre ayer a las tres. / Tengo hambre ahora.]
3. ¿Qué forma es incorrecta? [él habló / yo hable / tú hablaste]
4. ¿Cuál expresa una acción completa en el pasado? [Estudiaba cada día. / Estudié toda la noche. / Estudio mucho.]
5. ¿Cuál está bien formado? [yo fuí / yo fui / yo fue]

Clave de respuestas

Ejercicio 1: 1. hablé 2. comieron 3. estuviste 4. fuimos 5. hizo 6. pude

Ejercicio 2: 1. tuve 2. vivieron 3. fuisteis 4. hicimos 5. llamó
