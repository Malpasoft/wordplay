Referencia: Pretérito Perfecto

Concepto Forma Ejemplo
yo he + participioacción pasada recienteHe terminado el informe.
tú has + participio2.ª persona sing.¿Has probado el gazpacho?
él/ella ha + participio3.ª persona sing.Ana ha llegado tarde.
nosotros hemos + participio1.ª persona pl.Hemos visitado el museo.
participios irreg. clavehecho, visto, vuelto, escritoHe hecho los deberes.
ya / todavía nomarcadores típicos¿Ya has comido? — Todavía no.

Errores comunes

Incorrecto Correcto Explicación
He ido allí ayer.Fui allí ayer."Ayer" es un marcador de tiempo terminado → indefinido. En España, "He ido ayer" existe en algunos dialectos, pero el estándar es "fui".
Ha volvido a casa.Ha vuelto a casa."Volver" tiene participio irregular: vuelto. No existe "volvido".
He comido. He estado cansado.He comido. Estaba cansado.Los estados durativos pasados (cansado, enfermo) prefieren el imperfecto, no el perfecto.
¿Haz visto esta película?¿Has visto esta película?El auxiliar de segunda persona singular es "has", no "haz".
El año pasado he trabajado mucho.El año pasado trabajé mucho."El año pasado" es tiempo terminado → pretérito indefinido.

Ejercicio 1 — Elige la forma correcta del pretérito perfecto

Selecciona la opción correcta.

1. Yo ______ con mi jefe esta mañana. (hablar) [he hablado / hablé / había hablado]
2. ¿Tú ______ alguna vez gazpacho? (probar) [has probado / probaste / habías probado]
3. Ella ______ a las ocho. (llegar) [ha llegado / llegó / llega]
4. Nosotros ______ la reunión. (terminar) [hemos terminado / terminamos / habíamos terminado]
5. ¿Vosotros ______ la nueva película? (ver) [habéis visto / visteis / habéis vido]
6. Ellos ______ el problema. (resolver) [han resuelto / resolvieron / han resolvido]

Ejercicio 2 — Conjuga en pretérito perfecto

Escribe la forma correcta.

1. ¿Tú ______ (hacer) los deberes?  
2. Nosotros ______ (escribir) el informe.  
3. Ella ______ (volver) de sus vacaciones.  
4. Yo nunca ______ (estar) en Tokio.  
5. ¿Vosotros ya ______ (comer)?  

Ejercicio 3 — ¿Perfecto o indefinido?

Elige el tiempo verbal correcto.

1. ______ (yo, ir) al cine ayer. [Fui / He ido / Había ido]
2. Este mes ______ (nosotros, viajar) mucho. [hemos viajado / viajamos / viajábamos]
3. ¿______ (tú, alguna vez, probar) el mole? [Has probado / Probaste / Probabas]
4. La semana pasada ______ (ellos, trabajar) mucho. [trabajaron / han trabajado / trabajaban]
5. Todavía no ______ (yo, abrir) el correo. [he abierto / abrí / abría]

Clave de respuestas

Ejercicio 2: 1. has hecho 2. hemos escrito 3. ha vuelto 4. he estado 5. habéis comido
