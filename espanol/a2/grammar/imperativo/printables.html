Referencia: Imperativo

Concepto Forma Ejemplo
Imperativo afirmativo tú (reg.)3.ª pers. sg. presente indicativoHabla más despacio. / Come despacio.
Imperativo negativo túno + 2.ª pers. sg. subjuntivoNo hables tan rápido. / No comas tan rápido.
Imperativo ustedforma del presente de subjuntivoHable más alto, por favor.
Imperativo vosotrosinfinitivo con -r → -dHablad en voz alta. / Escribid vuestros nombres.
Irregulares de túven / ve / haz / di / pon / sal / ten / séHaz los deberes. / Di la verdad. / Ven aquí.
Pronombresafirm.: verbo+pron / neg.: pron+verboDímelo (af.) / No me lo digas (neg.).

Errores comunes

Incorrecto Correcto Explicación
No habla tan alto. (imperativo negativo de tú)No hables tan alto.El imperativo negativo de tú usa el presente de subjuntivo, no la tercera persona del presente de indicativo. La forma correcta es no hables.
Ven aquí, no vengas allí. → "No ven allí."No vengas allí.El imperativo negativo de tú de venir es no vengas. La forma irregular ven solo se usa en el afirmativo; el negativo sigue el patrón del subjuntivo.
Dime lo. / Dime-lo.Dímelo.Cuando se añaden dos pronombres al imperativo afirmativo, ambos se unen directamente al verbo formando una sola palabra, y se añade tilde si es necesario: dímelo.
Hacéd los deberes. (imperativo de vosotros)Haced los deberes.La forma de vosotros del imperativo afirmativo se forma sustituyendo la -r del infinitivo por -d. No lleva tilde y no tiene irregularidades: hablad, comed, escribid, haced.
Se lo digas. (prohibición)No se lo digas.Para expresar prohibición siempre se usa no + subjuntivo. Sin no, la frase es incompleta. Además, en el imperativo negativo el pronombre va antes del verbo: no se lo digas.

Ejercicio 1 — Ejercicio 1 — Selección múltiple: Imperativo afirmativo

Elige la forma correcta del imperativo afirmativo.

1. _____ (escuchar, tú) con atención la explicación. [Escucha / Escuche / Escuchad]
2. _____ (venir, tú) aquí un momento, necesito hablar contigo. [Viene / Ven / Venga]
3. _____ (leer, vosotros) el texto en voz alta. [Leed / Lean / Leen]
4. _____ (poner, tú) el abrigo — hace mucho frío fuera. [Pon / Pone / Ponga]
5. Si tiene problemas, _____ (llamar, usted) a este número de teléfono. [llame / llama / llamad]
6. _____ (ser, tú) más paciente con tus compañeros de trabajo. [Sé / Se / Es]

Ejercicio 2 — Ejercicio 2 — Escribe la forma del imperativo

Escribe el imperativo de tú o usted (afirmativo o negativo) según se indica.

1. (hacer, tú, afirmativo) _____ ejercicio todos los días para estar en forma.  
2. (hablar, tú, negativo) No _____ por teléfono durante la clase.  
3. (salir, usted, afirmativo) _____ por la puerta principal, por favor.  
4. (decir, tú, negativo) No _____ mentiras — siempre se descubren.  
5. (escribir, vosotros, afirmativo) _____ vuestros nombres al principio del examen.  

Ejercicio 3 — Ejercicio 3 — Corrección de errores en el imperativo

Cada oración tiene un error en el imperativo. Elige la versión correcta.

1. "No habla tan rápido — no te entiendo." (dirigido a tú) [No hables tan rápido. / No hablas tan rápido. / No hable tan rápido.]
2. "Vene aquí cuando puedas." (imperativo irregular de tú) [Ven aquí cuando puedas. / Viene aquí cuando puedas. / Venga aquí cuando puedas.]
3. "Dime lo todo." (imperativo con dos pronombres) [Dímelo todo. / Dime-lo todo. / Me lo di todo.]
4. "Hacéd los ejercicios para mañana, chicos." [Haced los ejercicios para mañana, chicos. / Hacen los ejercicios para mañana, chicos. / Haceis los ejercicios para mañana, chicos.]
5. "No se lo digas a nadie — es un secreto." (¿es correcta?) [Es correcta. / Di a nadie no lo. / No digas lo a nadie.]
