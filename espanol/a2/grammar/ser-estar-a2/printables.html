Referencia: SER y ESTAR avanzado

Concepto Forma Ejemplo
SER + adjetivo esencialCaracterística permanente de identidadJuan es inteligente y trabajador.
ESTAR + adjetivo de estadoEstado temporal o percibidoJuan está cansado hoy.
Adjetivos con doble significadoSER ≠ ESTAR con el mismo adjetivoEs malo (mala persona) / Está malo (enfermo).
SER + participioVoz pasiva: sujeto recibe la acciónEl puente fue construido en 1850.
ESTAR + participioEstado resultante de una acciónLa puerta está abierta.
LocalizaciónSiempre ESTAR, también edificiosLa catedral está en el centro.

Errores comunes

Incorrecto Correcto Explicación
Es cansado (persona) vs. Está cansadoEstá cansado para el estadoCuando describes cómo se siente alguien en este momento, usa ESTAR: está cansado. «Es cansado» significaría que esa persona es pesada/aburrida por naturaleza.
El museo es en el centroEl museo está en el centroLa localización, incluso de edificios permanentes, usa siempre ESTAR: «El museo está en el centro».
Soy listo → significado equivocadoEstoy listo para «estoy preparado»Cuidado con listo: «Soy listo» = soy inteligente. «Estoy listo» = estoy preparado/listo para hacer algo.
La puerta fue abierta vs. está abiertaSon construcciones distintas«Fue abierta» (SER + participio) es voz pasiva: alguien la abrió. «Está abierta» (ESTAR + participio) es el estado actual.
Este pastel es muy rico / bueno → ser buenoEstá muy rico para el saborPara comentar el sabor de la comida, usa siempre ESTAR: «Este pastel está muy rico/bueno».

Ejercicio 1 — SER o ESTAR: elige correctamente

Selecciona el verbo correcto según el contexto.

1. El Amazonas ______ el río más largo del mundo. [está / es / estaba]
2. ¿Dónde ______ el banco más cercano? [es / fue / está]
3. La película ______ muy aburrida. (característica de la película) [está / estuvo / es]
4. ¡Qué cansada ______ hoy! (estado temporal) [soy / eres / estás]
5. Este café ______ muy rico. (sabor) [es / está / era]
6. La puerta ______ cerrada con llave. (estado resultante) [es / fue / está]

Ejercicio 3 — SER + participio vs. ESTAR + participio

Elige la opción que corresponde al significado indicado.

1. La ventana ______ rota. (estado actual) [es rota / está rota / fue rota]
2. El edificio ______ construido en 1920. (voz pasiva histórica) [está construido / es construido / fue construido]
3. ¿Cuál significa «es una persona aburrida»? [Está aburrido. / Es aburrido. / Fue aburrido.]
4. ¿Cuál significa «siente aburrimiento ahora»? [Es aburrido. / Está aburrido. / Era aburrido.]
5. ¿Cuál usa correctamente ESTAR + participio? [Los deberes son hechos. / Los deberes están hechos. / Los deberes fueron hechos ahora.]
