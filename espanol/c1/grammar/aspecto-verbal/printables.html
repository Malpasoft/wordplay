El Aspecto Verbal: Perfectivo e Imperfectivo — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Aspecto Verbal

Concepto Forma Ejemplo
Aspecto perfectivoEvento visto como todo cerrado → indefinido, pluscuamperfectoAyer terminé el proyecto.
Aspecto imperfectivoProceso en curso, sin límite → imperfecto, progresivoCuando llegué, dormía.
Indefinido vs. imperfectoIndef. = evento; imperf. = escenario/estado/hábitoEntraron cuando hablaba por teléfono.
Estar + gerundioProgresivo: acción en curso en un momentoEstaban discutiendo cuando llegué.
Llevar + gerundioDurativo: acción que lleva X tiempo en cursoLlevo dos años viviendo en Bogotá.
Ponerse a / dejar de / seguir + inf./ger.Incoativo / cesativo / continuativoSe puso a hablar y no dejó de hacerlo.
AktionsartEstado / Actividad / Realización / LogroEncontré las llaves. (logro, perfectivo natural)

Errores comunes

Incorrecto Correcto Explicación
Cuando era niño, fui al parque cada día. (hábito pasado)Cuando era niño, iba al parque cada día.Los hábitos o acciones repetidas en el pasado sin límite temporal definido se expresan con el imperfecto (aspecto imperfectivo): iba, no «fui».
Llevo estudiando tres horas. (error de orden)Llevo tres horas estudiando.La perífrasis durativa es llevar + expresión de tiempo + gerundio. El gerundio va al final, después de la expresión de tiempo.
Acabé de llegar. (con indefinido)Acabo de llegar. (presente)La perífrasis de aspecto reciente acabar de + infinitivo se usa habitualmente en presente para indicar que la acción ocurrió hace muy poco.
El tren llegaba a las ocho. (para un evento puntual pasado concreto)El tren llegó a las ocho.Un evento puntual y concreto en el pasado usa el indefinido (perfectivo). El imperfecto «llegaba» indicaría un horario habitual, no un evento específico.
Supe la verdad desde siempre. (saber + desde siempre)Sabía la verdad desde siempre.Los verbos de estado (saber, conocer, tener, querer) expresan estados de fondo y suelen ir en imperfecto. El indefinido «supe» indica el momento en que se adquirió el conocimiento.

Ejercicio 1 — Elige el tiempo o la perífrasis aspectual adecuada

1. Cuando era pequeña, ______ (leer) un libro cada semana. (hábito pasado) [leyó / leía / ha leído / había leído]
2. Cuando llegué a la fiesta, mis amigos ya ______ (irse). [se fueron / se iban / se habían ido / se van]
3. ______ (llevar) dos años aprendiendo chino y todavía me cuesta la escritura. [Estoy llevando / He llevado / Llevo / Llevé]
4. De repente, ______ (ponerse) a nevar con mucha intensidad. [empezaba / se había puesto / se puso / estaba poniendo]
5. Perdona, no puedo hablar ahora: ______ (estar / comer). [como / comía / estoy comiendo / he comido]
6. ______ (dejar / fumar) hace tres años y desde entonces se siente mucho mejor. [Dejó de fumar / Dejaba de fumar / Estaba dejando de fumar / Había dejado fumar]

Ejercicio 2 — Explica la diferencia aspectual entre los dos enunciados

1. ¿Qué diferencia hay entre «Escribí la carta» y «Escribía la carta»?  
2. Transforma: «Hace dos horas que estudio.» → Perífrasis con LLEVAR.  
3. Transforma: «Empezó a correr de repente.» → Perífrasis incoativa equivalente.  
4. ¿Cómo se dice en español «She has just arrived»?  
5. ¿Cuál es el Aktionsart de «encontrar las llaves»: estado, actividad, realización o logro?  

Ejercicio 3 — Identifica el aspecto verbal correcto

1. ¿Cuál usa el imperfecto correctamente para un hábito pasado? [Cuando vivía en París, visité el Louvre cada semana. / Cuando vivía en París, visitaba el Louvre cada semana. / Cuando viví en París, visitaba el Louvre cada semana. / Cuando viví en París, visité el Louvre cada semana.]
2. ¿Cuál expresa que una acción había terminado antes de otra acción pasada? [Cuando llegué, empezaron. / Cuando llegué, empezaban. / Cuando llegué, habían empezado ya. / Cuando llegaba, empezaron.]
3. ¿Cuál usa correctamente la perífrasis incoativa? [Se comenzó a llover. / Se puso a llover. / Se ponía a llover. / Estaba poniéndose a llover.]
4. ¿Cuál corresponde a un verbo de «logro» (Aktionsart)? [hablar durante horas / saber la respuesta / construir una casa / llegar a la cima]
5. ¿Qué diferencia de aspecto hay entre «Supe la verdad» y «Sabía la verdad»? [Ninguna; son intercambiables. / «Supe» indica el momento en que se adquirió el conocimiento (perfectivo); «sabía» describe un estado de fondo (imperfectivo). / «Sabía» es más formal que «supe». / «Supe» se usa solo en oraciones negativas.]
