Referencia: El Sistema Pronominal

Concepto Forma Ejemplo
CD 3ª personalo, la, los, lasLo vi / Las compré
CI 3ª personale, lesLe dije la verdad
le/les + lo/lase lo/la/los/lasSe lo expliqué (= le expliqué el libro)
Posición proclíticaAntes del verbo conjugadoMe lo dijo
Posición enclíticaUnido tras inf/ger/imp. afirm.Decírmelo / Dímelo
Leísmo personalLe vi (a él) — RAE aceptaLe invité a la fiesta
Pronombre éticoImplicación afectiva sin función gramatical¡No te me vayas tan pronto!

Errores comunes

Incorrecto Correcto Explicación
*Le vi el coche (refiriéndose a la cosa).Lo vi / Lo compré (el coche).El leísmo de cosa no está aceptado. Para objetos inanimados, CD → lo/la/los/las.
*La dije que viniera.Le dije que viniera.El laísmo no está aceptado. CI siempre → le/les.
*Dámelo a mí primero.Dámelo. / Dámelo a mí.Con imperativo afirmativo el clítico va después del verbo unido: Dá-me-lo.
*Le + lo → lelo.Le + lo → se lo.La combinación CI le/les + CD lo/la se convierte en se lo/la/los/las.
*No diciéndole nada.No le digas nada. / Sin decirle nada.El imperativo negativo requiere no + verbo conjugado; el gerundio solo no forma negativo imperativo.

Ejercicio 1 — Ejercicio 1: Pronombres en contexto

1. — ¿Le has dado el regalo a María? — Sí, ___ ___ di ayer. [le lo / se lo / lo le / la le]
2. — ¿Puedes traerme el libro? — Claro, voy a traér___ ahora mismo. [lo te / te lo / me lo / le la]
3. No ___ digas nada a tus padres sobre esto. [les / los / las / se]
4. ¡Escúcha___ bien, que es importante! [me / le / lo / la]
5. ___ llamaron pero no estaba en casa. [Me / Le / Lo / La]
6. ¿Puedo ver ___ fotos de la boda? [lo / los / la / las]

Ejercicio 2 — Ejercicio 2: Reescribe con clíticos

2. Da el documento a nosotros. → _______.  
3. No expliques el problema a los estudiantes. → No _______ _______ expliques.  
4. Estoy contando una historia a ella. → _______.  
5. Manda las fotos a tus amigos. → _______.  

Ejercicio 3 — Ejercicio 3: Posición correcta

1. ¿Cuándo ___ vas a contar la verdad? [me lo / lo me / me la / la me]
2. Deja ___ en paz, por favor. [me / a me / a mí / conmigo]
3. No ___ hagas eso. [me lo / melo / lo me / me]
4. ___ la entregaron ayer mismo. [Se / Le / La / Lo]
5. Voy a explicár___ mañana. [telo / te lo / lo te / les lo]
