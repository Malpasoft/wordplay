Combinación de Dos Clíticos: Orden y Regla SE

Cuando hay dos clíticos, el CI va primero y el CD después. Le/les + lo/la/los/las → se lo/la/los/las.

CombinaciónForma correctaEjemplo
le + lose loSe lo doy a él
le + lase laSe la expliqué
les + losse losSe los mandé
me + lome loMe lo dijiste
te + late laTe la presto

Posición de los Clíticos

La posición depende del tipo de forma verbal.

Errores comunes

Los errores más frecuentes de los principiantes

Estos son los errores que los estudiantes cometen con más frecuencia.

*Le vi el coche (refiriéndose a la cosa).
→
Lo vi / Lo compré (el coche).
El leísmo de cosa no está aceptado. Para objetos inanimados, CD → lo/la/los/las.
*La dije que viniera.
→
Le dije que viniera.
El laísmo no está aceptado. CI siempre → le/les.
*Dámelo a mí primero.
→
Dámelo. / Dámelo a mí.
Con imperativo afirmativo el clítico va después del verbo unido: Dá-me-lo.
*Le + lo → lelo.
→
Le + lo → se lo.
La combinación CI le/les + CD lo/la se convierte en se lo/la/los/las.
*No diciéndole nada.
→
No le digas nada. / Sin decirle nada.
El imperativo negativo requiere no + verbo conjugado; el gerundio solo no forma negativo imperativo.
