Las Construcciones Pasivas en Profundidad — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Construcciones Pasivas

Concepto Forma Ejemplo
Pasiva perifrásticaSER + participio (+ por + agente) → acciónEl puente fue construido por ingenieros suizos.
Pasiva de estadoESTAR + participio → estado resultanteLa oficina está cerrada hasta las diez.
Pasiva reflejaSE + verbo (concuerda con sujeto)Se venden pisos en el centro histórico.
Impersonal con seSE + verbo en 3.ª sg. (sin sujeto)En esta región se trabaja mucho.
Pasiva de resultadoTENER + OD + participio (concuerda con OD)Tengo escritas las conclusiones del informe.
Verbos sin pasivagustar, doler, pertenecer, costar: no se pasivizánINCORRECTO: *Es gustado por todos.
RegistroFormal → SER; periodístico → SE; coloquial → voz activaEl decreto fue promulgado vs. Se promulgó el decreto.

Errores comunes

Incorrecto Correcto Explicación
La carta está escrita por Ana. (para expresar la acción)La carta fue escrita por Ana.Para expresar la acción pasada con agente explícito, se usa SER + participio: fue escrita por Ana. ESTAR describe el estado resultante, no la acción.
Se vende pisos en esta zona.Se venden pisos en esta zona.En la pasiva refleja, el verbo concuerda con el sujeto gramatical «pisos» (plural): se venden.
Me gusta fue dicho por muchas personas.Lo que dijeron muchas personas fue que les gustaba.Gustar no admite voz pasiva. Los verbos inversos (gustar, doler, encantar) ya tienen el objeto como sujeto gramatical y no se pasivizán.
Tengo escrito los informes. (con objeto plural)Tengo escritos los informes.En la construcción TENER + participio, el participio concuerda en género y número con el objeto directo: «los informes» es masculino plural → escritos.
El problema está resuelto por el ingeniero. (para acción)El problema fue resuelto por el ingeniero.Para mencionar al agente, se usa la pasiva con SER, no con ESTAR. ESTAR + participio no admite complemento agente con por.

Ejercicio 1 — Elige la construcción pasiva adecuada

1. La novela ______ por un autor desconocido en el siglo XIX. [está escrita / fue escrita / se escribió / tiene escrita]
2. La tienda ______ los domingos; no abren en festivos. [fue cerrada / se cierra / está cerrada / es cerrada]
3. ______ se entiende el español en toda América Latina. [Es hablado y / Se habla y / Está hablado y / Ha sido hablado y]
4. El director ya ______ los contratos; están listos para firmar. [fue firmado / está firmado / tiene firmados / se firmaron]
5. ¿Cuál es incorrecto con el verbo PERTENECER? [El inmueble pertenece al ayuntamiento. / El inmueble es pertenecido por el ayuntamiento. / El inmueble no es de propiedad privada. / El inmueble le pertenece al ayuntamiento.]
6. Los documentos ______ antes de la reunión de mañana. [estarán revisados / serán revisados / se revisarán / tendrán revisados]

Ejercicio 2 — Transforma la oración usando la construcción indicada

1. Los investigadores redactaron el informe. → El informe ______ por los investigadores. (pasiva perifrástica, pasado)  
2. Alguien ha cerrado la tienda. → La tienda ______. (estado actual)  
3. Aquí venden productos artesanales. → Aquí ______ productos artesanales. (pasiva refleja)  
4. He leído tres capítulos. → Tengo ______ tres capítulos. (pasiva de resultado)  
5. (Nadie en particular trabaja mucho.) → En esta empresa ______ mucho. (impersonal con se)  

Ejercicio 3 — Identifica el uso correcto

1. ¿Cuál describe un estado resultante con ESTAR? [El contrato fue firmado esta mañana. / El contrato está firmado y listo para entregarse. / El contrato se firmó esta mañana. / Tienen firmado el contrato.]
2. ¿Cuál usa la pasiva de resultado TENER correctamente? [Tengo escribiendo el informe. / Tengo el informe escrito. / Tengo escritos los informes. / Tanto B como C son correctas.]
3. ¿Cuál de estos verbos NO admite voz pasiva? [construir / redactar / doler / aprobar]
4. ¿Cuál es la diferencia entre «fue aprobada la ley» y «se aprobó la ley»? [No hay diferencia de significado. / «Fue aprobada» permite mencionar al agente con «por»; «se aprobó» omite al agente. / «Se aprobó» solo se usa en presente. / «Fue aprobada» es informal; «se aprobó» es formal.]
5. ¿Cuál combina correctamente la pasiva refleja con el número del verbo? [Se vende pisos nuevos. / Se venden piso nuevo. / Se venden pisos nuevos. / Se vende los pisos nuevos.]
