Referencia: Modalidad y Atenuación

Concepto Forma Ejemplo
certezasin duda, evidentemente, indudablemente + indicativoSin duda es la mejor solución
probabilidad altadeber de + inf, seguramente, probablementeDebe de tener razón
posibilidadquizás/tal vez + subj, puede que + subjPuede que venga mañana
atenuaciónen cierta medida, algo, al parecer, quisieraAl parecer, dimitirá pronto
contraste discursivosin embargo, no obstante, ahora bienSin embargo, los datos muestran lo contrario
consecuenciapor lo tanto, por consiguiente, de ahí que + subjPor lo tanto, debemos actuar
conclusiónen definitiva, en conclusión, en sumaEn definitiva, el proyecto es viable

Errores comunes

Incorrecto Correcto Explicación
*Debe ser tarde ya.Debe de ser tarde ya.Deber de + inf expresa probabilidad/deducción. Deber + inf sin de expresa obligación.
*Quizás viene mañana (con certeza hipotética).Quizás venga mañana (hipótesis) / Quizás viene mañana (más seguridad).Quizás/tal vez + subjuntivo = hipótesis; + indicativo = más probable. Ambas son aceptadas.
*Cabría decir que el texto es muy mal.Cabría decir que el texto adolece de ciertos problemas.En el discurso formal se prefieren expresiones elaboradas. Muy mal resulta demasiado directo y coloquial.
*Sin duda, podría ser que llegue tarde.Sin duda llegará tarde. / Podría ser que llegara tarde.No mezcles marcadores de certeza con marcadores de duda en la misma cláusula.
*Por lo tanto, no obstante mejoramos.No obstante, seguimos mejorando. / Por lo tanto, debemos mejorar.Por lo tanto (consecuencia) y no obstante (contraste) no se combinan en la misma oración.

Ejercicio 1 — Ejercicio 1: Marcadores de modalidad

1. El proyecto ___ tener algún fallo técnico — los resultados son inexplicables. [debe / debe de / tiene que / puede]
2. ___ sea necesario revisar todo el procedimiento desde el principio. [Quizás / Sin duda / Evidentemente / Claramente]
3. ___, las ventas han crecido un 15% este trimestre. [Al parecer / Sin embargo / Por lo tanto / A pesar de]
4. El enfoque es correcto; ___, los resultados tardarán en verse. [sin embargo / por lo tanto / además / es decir]
5. Estudió mucho; ___, aprobó con matrícula de honor. [sin embargo / no obstante / por consiguiente / al parecer]
6. ___, el texto presenta ciertas carencias metodológicas. [Hasta cierto punto / Sin duda alguna / Desde luego / Evidentemente]

Ejercicio 2 — Ejercicio 2: Completa con el marcador adecuado

1. _______ se ha producido un malentendido: nadie fue informado del cambio.  
2. La propuesta es innovadora; _______, requiere una mayor dotación presupuestaria.  
3. La empresa mejoró su eficiencia y, _______, redujo costes en un 20%.  
4. _______ llegar a tiempo si salimos ahora.  
5. El análisis es correcto; _______, no tiene en cuenta las variables externas.  

Ejercicio 3 — Ejercicio 3: Registro formal y marcadores

1. Para expresar probabilidad con deducción lógica, usamos: [sin duda / debe de + inf / quizás + subj / al parecer]
2. ¿Cuál de estos marcadores es un atenuador? [indudablemente / en cierta medida / desde luego / sin duda]
3. Para introducir una consecuencia lógica, usamos: [sin embargo / no obstante / por consiguiente / si bien]
4. ¿Cuál expresa contraste con lo anterior? [además / asimismo / no obstante / por lo tanto]
5. ¿Cuál de estos usos es incorrecto? [Debe de llover — deducción / Debe llover — obligación / Debe de llover — obligación / Debe salir — obligación]
