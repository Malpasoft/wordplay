Referencia: Subordinadas: Modo

Concepto Forma Ejemplo
certeza → indicativosaber/creer/ver/es cierto + que + indic.Creo que tiene razón
negación de certeza → subjuntivono creer/no pensar/no es cierto + que + subj.No creo que tenga razón
influencia → subjuntivoquerer/pedir/permitir/prohibir + que + subj.Pide que lo hagas
emoción → subjuntivoalegrarse/temer/sorprender + que + subj.Me sorprende que no lo sepa
mismo sujetoverbo principal + infinitivoEspero aprobar
diferente sujetoverbo principal + que + subjuntivoEspero que apruebes
duda → subjuntivodudar de / es posible / es probable + que + subj.Es posible que llueva

Errores comunes

Incorrecto Correcto Explicación
*Creo que venga mañana.Creo que viene mañana.Creer en afirmativo + indicativo: expresa certeza del hablante.
*No creo que viene.No creo que venga.No creer + subjuntivo: la negación convierte la certeza en duda.
*Quiero que voy al cine.Quiero ir al cine.Mismo sujeto → infinitivo, no que + subjuntivo.
*Es evidente que lo sepa.Es evidente que lo sabe.Expresiones de certeza (es evidente, es claro, es obvio) exigen indicativo.
*Pido que vas al médico.Pido que vayas al médico.Pedir exige subjuntivo en la subordinada (diferente sujeto).

Ejercicio 1 — Ejercicio 1: Indicativo o Subjuntivo

1. El médico dice que el paciente ___ mejor muy pronto. [estará / esté / estuvo / estaría]
2. No es verdad que nosotros ___ el dinero. [tomamos / tomáramos / tomemos / hayamos tomado]
3. Me sorprende que ella no ___ nada sobre el cambio. [sabe / supo / sepa / sabría]
4. Pienso que la propuesta ___ interesante. [es / sea / fuera / sería]
5. No pienso que la propuesta ___ tan buena como dicen. [es / sea / fuera / sería]
6. La directora pidió que todos ___ el informe antes del viernes. [entregan / entregaron / entreguen / entregarán]

Ejercicio 2 — Ejercicio 2: Completa con infinitivo o que + subjuntivo

1. Espero _______ (poder) asistir a la conferencia.  
2. Mi jefe exige que nosotros _______ (llegar) puntualmente.  
3. Es importante _______ (descansar) bien antes del examen.  
4. No creo que ellos _______ (tener) suficiente experiencia.  
5. Temo _______ (equivocarse) en el análisis.  

Ejercicio 3 — Ejercicio 3: Elige la forma correcta

1. Es evidente que el proyecto ___ un gran esfuerzo. [requiere / requiera / requiriera / requerirá]
2. Dudo mucho que ellos ___ a tiempo. [llegan / lleguen / llegaron / llegaban]
3. Me alegra que todo ___ bien. [sale / salga / salió / saldría]
4. Está claro que el mercado ___ una recuperación. [experimenta / experimente / experimentara / haya experimentado]
5. No está claro que la medida ___ los resultados esperados. [produce / produzca / produjo / produjera]

Clave de respuestas

Ejercicio 1: 1. estará 2. tomemos 3. sepa 4. es 5. sea 6. entreguen

Ejercicio 2: 1. poder 2. lleguemos 3. descansar 4. tengan 5. equivocarme

Ejercicio 3: 1. requiere 2. lleguen 3. salga 4. experimenta 5. produzca
