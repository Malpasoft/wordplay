Referencia: Las Perífrasis Verbales

Concepto Forma Ejemplo
tener que vs. haber quePersonal (tener que) vs. impersonal (haber que)Tengo que ir / Hay que ir
deber vs. deber deObligación (deber) vs. probabilidad (deber de)Debes llamar / Debe de ser tarde
incoativasPonerse a (voluntario), echarse/romper a (espontáneo)Se puso a estudiar / Rompió a llorar
acabar de vs. acabar + gerReciente (acabar de) vs. resultado (acabar + ger)Acaba de llegar / Acabó llorando
llevar + ger/partDurativo (ger: proceso en curso) / acumulativo (part: resultado)Lleva horas esperando / Lleva leídas tres
ser/estar + participioProceso (ser) vs. estado resultante (estar)Fue aprobado / Está aprobado
tener + participioResultado acumulado en la esfera del sujeto (activa)Tengo escritas tres cartas

Errores comunes

Incorrecto Correcto Explicación
*Debe llegar tarde (deducción).Debe de llegar tarde (deducción).Deber de + inf = probabilidad/deducción. Deber + inf sin de = obligación moral.
*Llevo dos horas esperado.Llevo dos horas esperando.Llevar + gerundio (durativo): la acción aún continúa → gerundio, no participio.
*Acabé llorando (reciente).Acabo de llegar (reciente). Acabé llorando (resultado).Acabar de + inf = reciente; acabar + gerundio = resultado final del proceso.
*Romper a trabajar (voluntario).Ponerse a trabajar (voluntario). Romper a llorar (espontáneo).Romper a y echarse a se usan para reacciones espontáneas y emotivas.
*Tengo escrita muchas cartas.Tengo escritas muchas cartas.El participio en tener + participio concuerda con el CD: tener escritas muchas cartas.

Ejercicio 1 — Ejercicio 1: Perífrasis modales

1. «___ de tener mucho dinero para llevar ese estilo de vida.» Deducción: [Tiene que / Debe de / Hay que / Puede]
2. «___ respetar las señales de tráfico.» Obligación impersonal: [Tienes que / Debes / Hay que / Puedes]
3. «He ___ entregar el proyecto antes del viernes.» Obligación formal: [de / que / a / para]
4. «___ llover mañana, así que lleva paraguas.» Posibilidad: [Debe / Hay que / Puede / Tiene que]
5. «___ estudiar más para aprobar.» Obligación personal: [Hay que / Tienes que / Debes de / Puedes]
6. «___ ser la una — todo el mundo está comiendo.» Deducción: [Tiene que / Debe de / Hay que / Va a]

Ejercicio 3 — Ejercicio 3: Elige la perífrasis correcta

1. Para indicar que una acción empezó hace tiempo y sigue ahora, usamos: [ponerse a + inf / llevar + gerundio / acabar de + inf / romper a + inf]
