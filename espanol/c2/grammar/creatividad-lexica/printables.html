Creatividad Léxica: Formación de Palabras — Imprimibles

Referencia, ejercicios y clave de respuestas para imprimir.

Referencia: Creatividad Léxica

Concepto Forma Ejemplo
DerivaciónFormación de palabras con prefijos y sufijos: re-, des-, -ción, -mente, -ismo, -istareleer, deshacer, globalización, claramente
ComposiciónUnión de dos o más bases léxicas: N+N, V+N, Adj+Nlavaplatos, cortacésped, mediodía, altavoz
ParasíntesisDerivación simultánea con prefijo y sufijo sin que existan las formas intermediasaterrizar (*tierra → *terrizar → aterrizar)
AcortamientosAbreviación de palabras largas: apócope, aféresis, acronimia, siglasfoto (fotografía), bus (autobús), DNI, ONG
Préstamos y neologismosAdopción de palabras extranjeras (anglicismos, galicismos) y creación de nuevas palabrasinternet, chatear, hackear, selfi
Creatividad en el español actualFormación por analogía, calcos, palabras-maleta (blended words)franquismo → kirchnerismo, infotainment → infoentretenimiento

Errores comunes

Incorrecto Correcto Explicación
*La parasíntesis es solo un tipo de composiciónLa parasíntesis combina prefijación y sufijación simultáneasaterrizar no es composición (no existe *terrizar) ni simple derivación; el prefijo y el sufijo se añaden a la vez.
*Selfi es un anglicismo no aceptado por la RAESelfi está en el diccionario de la RAE desde 2014La RAE acepta préstamos adaptados a la ortografía española; selfi es la forma recomendada (no selfie).
*Las siglas siempre se leen letra por letraLas siglas que forman sílabas se leen como palabras (acrónimos)OTAN se lee [ó-tan], no letra por letra; DNI sí se lee letra por letra.
*Chat-ear es derivación simpleChatear es un préstamo integrado mediante derivación verbal (-ear)La base es el anglicismo chat + el sufijo verbalizador -ear; es préstamo + sufijación, no derivación puramente hispánica.
*Los calcos semánticos son siempre galicismosLos calcos semánticos pueden venir de cualquier lenguaRatón (de mouse) y nube (de cloud) son calcos del inglés, no del francés.

Ejercicio 1 — Identifica el proceso de formación

1. ¿Qué proceso forma la palabra lavaplatos? [Derivación por sufijación / Composición V+N / Parasíntesis / Acrónimo]
2. ¿Qué proceso genera aterrizar? [Derivación simple con prefijo / Composición de dos sustantivos / Parasíntesis (a- + tierra + -izar) / Calco del inglés]
3. ¿Cuál de estas palabras es un acrónimo? [DNI / OTAN / ONG / URL]
4. ¿Qué proceso produce chatear? [Composición V+N / Calco semántico / Préstamo + sufijación verbal (-ear) / Apócope]
5. ¿Cuál es un ejemplo de calco semántico del inglés? [Chatear / Ratón (= mouse informático) / Selfi / OTAN]

Ejercicio 2 — Analiza morfológicamente

1. Descompón globalización en sus morfemas y explica el proceso.  
2. ¿Cuál es la palabra completa de la que viene bici? ¿Qué proceso es?  
3. Crea un neologismo en español para el concepto inglés «to google» (buscar en un buscador de internet) siguiendo el patrón de chatear.  

Ejercicio 3 — Formación de palabras en contexto

1. ¿Cuál de estos procesos no existe en español? [Parasíntesis / Composición / Infijación (inserción en la raíz) / Derivación]
2. ¿Cuál de estas palabras es una palabra-maleta? [lavaplatos / globalización / infoentretenimiento / aterrizar]
3. La RAE recomienda selfi en lugar de selfie. ¿Por qué? [Porque selfie es una palabra inventada / Para adaptar el préstamo a la ortografía española / Porque selfi viene del francés / Para evitar el anglicismo]
4. ¿Cuál de estos términos es un neologismo por analogía? [Foto (< fotografía) / Infoentretenimiento / Kirchnerismo / OTAN]

Clave de respuestas

Ejercicio 1: 1. Composición V+N 2. Parasíntesis (a- + tierra + -izar) 3. OTAN 4. Préstamo + sufijación verbal (-ear) 5. Ratón (= mouse informático)

Ejercicio 2: 1. global + -iz(ar) + -ación: sufijación doble (global → globalizar → globalización) 2. bicicleta; apócope (supresión del final) 3. googlear

Ejercicio 3: 1. Infijación (inserción en la raíz) 2. infoentretenimiento 3. Para adaptar el préstamo a la ortografía española 4. Kirchnerismo
