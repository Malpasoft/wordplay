Sobre el DELE B2

Formato, estructura y criterios de evaluación del Diploma de Español Nivel B2

¿Qué es el DELE B2?

El DELE B2 (Diploma de Español como Lengua Extranjera, nivel B2) es un certificado oficial expedido por el Instituto Cervantes en nombre del Ministerio de Educación de España. Acredita un dominio del español de nivel Intermedio-Alto según el Marco Común Europeo de Referencia para las Lenguas (MCER).

Con el DELE B2 demuestras que puedes entender textos complejos sobre temas concretos y abstractos, expresarte con fluidez y espontaneidad, y producir textos claros y detallados sobre una amplia gama de temas.

Comprensión de lectura — Tareas

Tarea 1: Texto de unos 450 palabras seguido de 6 afirmaciones. Debes indicar si cada afirmación es verdadera, falsa o no se dice en el texto.

Tarea 2: Texto con 6 huecos. Para cada hueco, elige la opción correcta (A, B o C) de entre tres palabras o expresiones.

Tarea 3: Texto de unos 600 palabras con 6 preguntas de opción múltiple (A, B o C).

Tarea 4: Ocho enunciados que debes relacionar con cuatro textos breves (A, B, C, D). Algunos textos pueden usarse más de una vez.

Tarea 5: Texto periodístico con 6 huecos. Elige para cada hueco la opción correcta entre cuatro posibilidades (A, B, C o D).

Puntuación y aprobado

Las pruebas escritas (comprensión de lectura y auditiva) se puntúan entre 0 y 25 cada una; la expresión escrita y oral, entre 0 y 25 también. Para aprobar, debes obtener al menos el 60 % en cada grupo: comprensión (lectura + auditiva) y expresión (escrita + oral). La nota mínima de aprobado es 30/50 en cada grupo.

Consejos para la prueba de lectura
  • Lee el enunciado de cada tarea antes de leer el texto.
  • Subraya las palabras clave en cada pregunta.
  • Busca sinónimos: el texto no repetirá exactamente las palabras de las opciones.
  • Para «no se dice», asegúrate de que la información no aparece en el texto — aunque parezca lógica.
  • Administra el tiempo: 70 minutos para 5 tareas = 14 minutos por tarea como máximo.
  • No dejes ninguna respuesta en blanco: las respuestas incorrectas no restan puntos.
