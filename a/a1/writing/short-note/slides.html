Short Note or Message

Write a simple informal note

Lesson Practice Game

What is a short note?

Purpose
Leave a quick message for someone you know
Length
20–30 words
Register
Informal — relaxed language
Where used
Leaving a message at home, at work, for a friend

Structure of a note

Greeting
Hi! / Hello! / Dear [Name],
Main message
I am at... / I went to... / I will be back at...
Closing
See you later! / Back soon! / Love, [Name]
Keep it short. Use simple sentences. You do not need to explain everything — just the key information.

Model answer

Example
Hi Tom! I went to the supermarket. I will be back at 5pm. There is food in the fridge. See you later! Ana
Check
Is the message clear? Does it say what, where, and when?
