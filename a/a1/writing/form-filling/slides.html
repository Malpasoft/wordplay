Form Filling

Complete a form with accurate personal information

Lesson Practice Game

Why form filling?

Purpose
Give accurate personal information in a structured format
Register
Neutral / formal — no slang
Where used
Registration forms, applications, government documents
Key rule
Use capital letters for names and proper nouns

Key conventions

Name order
LAST NAME, First name (or as instructed)
Date of birth
DD/MM/YYYY e.g. 15/03/2001
Nationality
Use the adjective: British, French, Japanese...
Occupation
Use a job title: teacher, engineer, student, unemployed
Read each field carefully before writing. Write N/A if a field does not apply to you.

Common mistakes to avoid

Date
Do not write the month as a word unless asked
Telephone
+[country code] [area code] [number]
Email
Write exactly as it appears — no spaces
Signature
Sign in the box — do not print your name here
