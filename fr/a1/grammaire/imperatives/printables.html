Forme Exemple

NégationDon't + base verbaleDon't run! / Don't be late.
PolitessePlease + impératifPlease close the window.
PropositionLet's + base verbaleLet's go to the beach!
Proposition négativeLet's not + base verbaleLet's not stay here.
Usagepanneaux · recettes · consignesPush. / Turn left. / Read the text.
