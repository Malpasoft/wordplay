Cap. 7

Demostrativos

this · that · these · those — señalar cosas cerca y lejos

Los cuatro demostrativos

Los demostrativos sirven para señalar cosas o personas según estén cerca o lejos, en singular o plural. El inglés solo tiene cuatro:

Cerca (here)Lejos (there)
Singularthis
este / esta / esto		that
ese / esa / aquel
Pluralthese
estos / estas		those
esos / aquellos
Diferencia clave con el español: El español tiene tres niveles (este/ese/aquel); el inglés solo dos: this (cerca) y that (lejos). Además, en inglés no hay género: this sirve para masculino y femenino.

Singular: this / that

Para una sola cosa o persona

Este libro es bueno. (cerca)
→
This book is good.
Esa casa es grande. (lejos)
→
That house is big.
Esta es mi hermana.
→
This is my sister.
¿Qué es eso?
→
What's that?
This is delicious!
algo que tienes cerca
Who's that?
alguien a distancia

Plural: these / those

Para dos o más cosas

Cuando hablamos de varias cosas, this se convierte en these y that en those. El sustantivo va en plural.

Estos zapatos son nuevos.
→
These shoes are new.
Esos coches son caros.
→
Those cars are expensive.
Estas son mis llaves.
→
These are my keys.
¿De quién son esos?
→
Whose are those?
Pronunciación: this /ðɪs/ tiene un sonido corto; these /ðiːz/ es más largo y termina en sonido de z. Es la pista para distinguir singular y plural al oído.

¿Con sustantivo o solos?

Determinante vs pronombre

Los demostrativos pueden ir antes de un sustantivo o solos, igual que en español:

+ sustantivo
este coche
→
this car
esas mesas
→
those tables
solos (pronombre)
Esto es difícil.
→
This is difficult.
Esos son caros.
→
Those are expensive.
Al teléfono: el inglés usa demostrativos de forma especial: "Hello, this is Anna" (yo soy Anna) e "Is that John?" (¿eres tú John?).

Errores frecuentes

Cuando el español interfiere

This shoes are new.
→
These shoes are new.
Sustantivo plural → demostrativo plural.
That are my books.
→
Those are my books.
Plural lejos → those, no that.
this is a problem (mayúscula)
→
This is a problem.
Cuidado: this ≠ "this is" — son cosas distintas.
these car
→
this car  /  these cars
El número del demostrativo y del sustantivo deben coincidir.

Resumen
