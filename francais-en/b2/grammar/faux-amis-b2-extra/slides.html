In Formal Contexts

How these false friends appear in the kinds of texts B2 learners encounter.

  • Il n'a pas réalisé l'ampleur du problème. (He didn't realise the scale.) — se rendre compte would be more precise
  • Ils ont réalisé ce projet en six mois. (They completed/achieved this project.) ✓
  • J'ai résumé le rapport. (I summarised the report.) ✓ NOT resumed
  • Il faut former le personnel. (The staff must be trained.) ✓
  • Elle a élaboré un plan détaillé. (She drew up/developed a detailed plan.) ✓
  • Exigeant ≠ existing. Le marché est très exigeant. (The market is very demanding.)

Common Mistakes

AvoidUseWhy
J'ai réalisé que je me suis trompé.Je me suis rendu compte que je m'étais trompé.Réaliser = to achieve/make. To realise (become aware) = se rendre compte de. (Note: réaliser is now widely used informally for 'realise', but avoid in formal writing.)
Nous allons achever notre objectif.Nous allons accomplir / atteindre notre objectif.Achever = to complete/finish (a task that's underway). To achieve a goal = accomplir / atteindre.
Je vais résumer le travail demain. (= resume)Je vais reprendre le travail demain.Résumer = to summarise. To resume = reprendre / recommencer.
Exigeant ≠ existingLe processus existant / actuel. Exigeant = demanding.Exigeant comes from exiger (to demand/require). The existing system = le système existant / actuel.
