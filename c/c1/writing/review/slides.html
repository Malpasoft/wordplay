The Review

Informed opinion · balanced evaluation · recommendation · semi-formal

LessonReviewGamePrintables

C1 reviews

At C1, a review must show sophisticated vocabulary, nuanced evaluation, and a clear personal voice. It should avoid both gushing praise and unsupported criticism.

Structure
Title → Brief description → Evaluation → Recommendation
Register
Semi-formal: engaging, informative, personal but not casual
Length
140–190 words
Key feature
Specific detail and evidence, not vague opinion

Advanced language for reviews

Nuanced praise
What is particularly striking is… / The standout feature is…
Qualified criticism
While the pacing is occasionally slow, the performances compensate.
Vocabulary
compelling / evocative / understated / uneven / riveting / derivative
Recommendation
It comes highly recommended to… / This is essential viewing for…
Avoid: amazing, awesome, terrible, horrible. Use: outstanding, thought-provoking, unconvincing, overrated, derivative, nuanced.

What markers look for

Content
Is the subject described clearly? Is the evaluation supported by specifics?
Range
Does the vocabulary go beyond good/bad/nice?
Register
Is the tone appropriately semi-formal throughout?
Recommendation
Is it clear who should read/watch/visit this?
