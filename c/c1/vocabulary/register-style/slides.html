Vocabulary

Register & Style

Formal vs informal equivalents · register awareness · neutral alternatives

Flashcards Lesson Game

Words 1–4

Ask → Enquire
ask is informal/neutral; enquire is formal
Get → Obtain
get is informal/neutral; obtain is formal
Use → Utilise
use is neutral; utilise is formal (but avoid overusing)
Find out → Ascertain
find out is informal; ascertain is formal
Examples
I would like to enquire about the position advertised.
Researchers were able to obtain the necessary funding.

Words 5–8

Start → Commence
start is neutral/informal; commence is formal
End → Conclude
end is neutral; conclude is formal
Say → State / Assert
say is neutral; state/assert are more formal and precise
Show → Demonstrate
show is neutral/informal; demonstrate is formal
Examples
The proceedings will commence at 9am.
The study concluded that further research was necessary.

Words 9–12

Think → Consider / Deem
think is informal; consider/deem are formal and more precise
Help → Facilitate
help is informal/neutral; facilitate is formal (enable/make easier)
Big → Substantial / Significant
big is informal/vague; substantial/significant are formal and precise
Bad → Detrimental / Adverse
bad is informal/vague; detrimental/adverse are formal
Examples
The committee considered the proposal unsuitable.
Technology can facilitate communication across borders.
