Part 8 — Multiple Matching

Match each statement to the correct section (A-D). A section may be used more than once.

A — Mountain biking: Offers a full-body workout while exploring trails. Requires significant equipment investment but provides unmatched contact with nature.

B — Swimming: Gentle on joints, suitable for all ages. Can be done year-round if you have access to an indoor pool. Improves cardiovascular health and flexibility.

C — Running: Requires minimal equipment — just a good pair of shoes. Can be done almost anywhere. However, the repetitive impact can cause knee and ankle injuries over time.

D — Yoga: Excellent for mental health and stress reduction. Builds core strength and flexibility. Can be practised at home with no equipment at all.

1. Which activity is described as particularly low-cost to start?

2. Which activity mentions a risk of physical injury?

3. Which activity highlights benefits for mental wellbeing?

4. Which activity requires the most expensive equipment?

5. Which activity is recommended for people with joint problems?

6. Which activity can be done without leaving the house?
