Part 3 — Word Formation

Eight gaps. A root word is given in capitals at the end of each line — you must change it (prefix, suffix, sometimes both) to fit the gap grammatically and semantically.

Strategy

  • Identify the word class needed: noun, verb, adjective, adverb
  • Check if a negative prefix is needed (un-, in-, dis-, mis-, ir-, im-)
  • Be aware of multiple changes — e.g. ABLE → INABILITY (prefix + suffix change)
  • Watch plural and verb forms — "decisions" not "decision" if needed
  • Time guide: 12 minutes
