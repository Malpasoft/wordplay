Part 7 — Multiple Matching

One long text (or several short texts) and 10 questions. Match each question to the section that contains the answer.

Strategy

  • Read the questions FIRST and underline keywords
  • Scan the text looking for paraphrases of those keywords
  • The same section can be the answer to multiple questions
  • The questions are NOT in text order
  • Don't get stuck — flag and return
  • Time guide: 15 minutes
