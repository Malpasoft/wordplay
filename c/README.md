# Directory `c`: CEFR C1 and C2

Advanced (C1) and Proficiency (C2) levels. Includes Cambridge CAE (C1 Advanced) and CPE (C2 Proficiency) exam preparation.

## Folder structure

- **c1/** — CEFR C1: 17 grammar + 14 vocabulary + 6 writing = **37 chapters** + CAE exam prep
- **c2/** — CEFR C2: 11 grammar + 12 vocabulary + 6 writing = **29 chapters** + CPE exam prep

Each chapter folder contains:
- `slides.html` — lesson deck
- `worksheet.html` — auto-graded practice
- `game.html` — 4-stage mastery game
- `printables.html` — print-ready A4 (grammar)
- `flashcards.html` — vocabulary chapters

## CAE / CPE exam prep

Both exams have strategy overviews and part-by-part lesson guides. Full practice papers and mock exams are planned but not yet built.

## Notes

- C1/C2 grammar and vocabulary chapters are scaffolded. Content enrichment is a current priority.
- C1 vocabulary expanded to 14 topics; C2 to 12 topics (May 2026).
