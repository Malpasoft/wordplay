The Report

Headings, formal register, recommendations.

Headings · Impersonal language · Recommendations
B2 · FCE Cambridge

What makes a report different?

Headings
Use clear section headings throughout
Introduction · Findings · Recommendations
Impersonal
Use formal, impersonal language
'It was found that…' not 'I think…'
Bullet points
Use lists under each heading for clarity
Makes information easy to scan
Recommendations LAST
Opinions go ONLY in the conclusion
Don't give recommendations in the body

Useful language

Introduction
The aim of this report is to…
This report is intended to…
Information was gathered by…
Reporting results
Most people seem to feel that…
Several respondents suggested…
It was generally agreed that…
Recommending
I would therefore recommend…
It would seem that … is the best idea.
It is recommended that…
Key differences from essay
No 'Dear Sir/Madam'
Use headings — essay doesn't
More impersonal throughout
