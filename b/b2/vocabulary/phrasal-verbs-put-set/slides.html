Phrasal Verbs — Put and Set

Put off, put up with, set up, set back, set aside.

Meaning · Examples · Sentence practice
B2 Vocabulary

Phrasal verbs with PUT AND SET

These phrasal verbs are tested in FCE Use of English Parts 1 and 4. Learn each as a fixed phrase with its meaning.
put off
postpone / delay
put up with
tolerate / accept something unpleasant
put forward
propose / suggest an idea
put through
connect a phone call / make someone experience something difficult
put aside
save money or time for a specific purpose
put out
extinguish a fire / publish / inconvenience

More put and set phrasal verbs

set up
establish / create something
set out
begin a journey / explain in detail
set back
delay progress
set aside
reserve time or money for a purpose
set off
begin a journey / trigger an alarm

Examples in context

The government decided to put off the meeting until next month.
She managed to put up with with the new regulations despite the complexity.
The entrepreneur decided to put forward her own consulting firm.
Some phrasal verbs are separable: put it off / put off the meeting. Others are inseparable: cope with the problem (NOT cope the problem with).
