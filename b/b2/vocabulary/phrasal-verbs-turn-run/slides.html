Phrasal Verbs — Turn and Run

Turn down, turn out, run out of, run into, run through.

Meaning · Examples · Sentence practice
B2 Vocabulary

Phrasal verbs with TURN

turn down
refuse an offer or request / reduce volume
turn up
arrive / increase volume / be found unexpectedly
turn out
prove to be the case / result in a particular way
turn into
become or transform into something
turn over
hand to authorities / generate revenue
turn around
reverse a bad situation / physically face the other way

Phrasal verbs with RUN

run out of
have no more of something
run into
meet someone by chance / encounter a problem
run over
go beyond a time limit / read through quickly
run through
practise / explain quickly
run up
accumulate a debt or bill
run by
mention an idea to get someone's opinion

Examples in context

She turned down the promotion — she didn't want the extra responsibility.
The project turned out to be far more complex than anyone had anticipated.
We are running out of renewable energy capacity in this region.
She ran into her former colleague at the conference on data privacy.
