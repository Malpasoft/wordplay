Mixed & Inverted Conditionals

Past if + present result · inverted had/were

Mixed conditionals

Mixed conditionals combine elements from different types when the time in the if-clause and result-clause differ.

Past if → present result
If I had studied medicine, I would be a doctor now.
Present if → past result
If I were braver, I would have asked her out.

Type 1: Past if → present result

If + past perfect, + would + base verb (now)
Example
If she had taken the job, she would be earning more now.
Reality
She didn't take the job → she isn't earning more now.

Type 2: Present if → past result

If + past simple (now), + would have + PP (then)
Example
If I were more organised, I would have finished on time.
Reality
I'm not organised (present) → I didn't finish (past).

Inverted conditionals (formal)

Had I known...
Had I known the truth, I would have acted differently. = If I had known...
Were I to...
Were I to resign, the project would collapse.
Should you need...
Should you need assistance, contact us. = If you need...
Were it not for...
Were it not for your help, I would have failed.

Common mistakes

Had I would known...
→
Had I known...
inversion: had + subject + PP (no "would")
If I had studied harder, I would have a doctor now.
→
If I had studied harder, I would be a doctor now.
present result → would + be + ...

Recap

Mixed 1
past perfect → would + base (now)
If I'd studied medicine, I'd be a doctor.
Mixed 2
past simple (now) → would have + PP
If I were braver, I'd have asked.
Inverted
Had I / Were I / Should you
Had I known, I would have helped.
