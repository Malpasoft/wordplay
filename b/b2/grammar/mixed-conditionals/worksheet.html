Exercise 3
Key word transformation
8 points
Complete the second sentence so it has the same meaning. Use the word given.
1
We didn't invest in renewables, so emissions are still high today. INVESTED
If we ______ in renewables, emissions would be lower today.
2
She is not ambitious, so she didn't apply for promotion. WERE
If she ______ more ambitious, she would have applied for promotion.
3
He lost his job last year, so he's struggling financially now. LOST
If he hadn't ______ his job, he wouldn't be struggling now.
4
Data privacy laws are weak, so companies have shared user data freely for years. STRONGER
If data privacy laws were ______, companies would not have shared user data so freely.
