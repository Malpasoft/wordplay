Present Tense — Practice — Answer Key

Practice

  1. hablo Explanation: For -AR verbs with "yo" (I), remove -ar and add -o. hablar → habl + o = hablo
  2. trabajamos Explanation: For "nosotros" (we) with -AR verbs, the ending is -amos. trabajar → trabaj + amos = trabajamos
  3. comen Explanation: "Ellos" (they) uses the -en ending for -ER verbs. comer → com + en = comen
  4. vives Explanation: For "tú" (informal you) with -IR verbs, the ending is -es. vivir → viv + es = vives
  5. estudio Explanation: "Yo" (I) with -AR verbs uses -o. estudiar → estudi + o = estudio
  6. Nosotros hablamos español Explanation: "Nosotros" (we) requires the -amos ending. The other options use wrong pronouns or endings. Remember: the subject pronoun and verb ending must agree!
  7. vive Explanation: "Tu familia" (your family) is singular, so it uses the él/ella form with -e ending. vivir → viv + e = vive
  8. bebo Explanation: "Yo" (I) with -ER verbs uses -o. beber → beb + o = bebo
