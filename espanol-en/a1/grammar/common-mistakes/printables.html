Common Mistakes — Error Reference
A1 Grammar · Word Play
Error TypeWrongCorrectRule
Subject-VerbYo soy hablar inglés.Yo hablo inglés.Conjugate the verb; never use the infinitive alone
Adjective GenderEl niño es bonita.El niño es bonito.Adjectives must match noun gender (-o masc, -a fem)
Past TensesAyer iba al mercado.Ayer fui al mercado.Completed past event → pretérito; habitual → imperfecto
Object PronounsVeo ella mañana.La veo mañana.Object pronouns go BEFORE the conjugated verb
Ser vs Estar¿Dónde eres?¿Dónde estás?ESTAR = location/state; SER = identity/permanent
Article GenderLas problema.El problema.Most -ma nouns (problema, sistema) are masculine
Common Mistakes — Error Correction Exercises
A1 Grammar · Word Play
Exercise 1 — Spot the error (6 pts)
Each sentence has one error. Underline it and write the correct version.
  1. Ella soy de México.
  2. El libro es interesanta.
  3. ¿Dónde eres el restaurante?
  4. Yo veo ella en el parque.
  5. Ayer iba mucho al gimnasio.
  6. Las sistema no funciona.
Exercise 2 — Choose the correct form (5 pts)
Circle the correct option.
  1. Mi hermana es / está profesora.
  2. ¿Es / Está la leche en la nevera?
  3. Las mesas es / son redondas.
  4. Ayer comí / comía pizza (one specific meal).
  5. Cuando era niño, siempre jugué / jugaba al fútbol.
Exercise 3 — Correct the sentence (5 pts)
Rewrite each sentence correctly.
  1. Él tiene hablar inglés.
  2. La camisa es azulo.
  3. Te doy lo mañana.
  4. La programa es buena.
  5. Mi amigo estar cansado.
Common Mistakes — Answer Key
A1 Grammar · Word Play
Exercise 1 — Answers
  1. soy → es: Ella es de México. (3rd person singular = es)
  2. interesanta → interesante: El libro es interesante. (-nte endings don't change gender)
  3. eres → está: ¿Dónde está el restaurante? (location = estar)
  4. veo ella → la veo: Yo la veo en el parque. (pronoun before verb)
  5. iba → fui (if specific past): Or keep iba if meaning "used to go habitually". Context determines choice.
  6. Las → El: El sistema no funciona. (sistema is masculine)
Exercise 2 — Answers
  1. es — profession = ser
  2. Está — location = estar
  3. son — plural subject (las mesas)
  4. comí — single completed past event
  5. jugaba — habitual past action
Exercise 3 — Answers
  1. Él habla inglés. (conjugate the verb)
  2. La camisa es azul. (azul has no gender change)
  3. Te lo doy mañana. (both pronouns before verb)
  4. El programa es bueno. (programa is masculine)
  5. Mi amigo está cansado. (cansado is a state = estar)
