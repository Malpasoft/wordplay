Regular Past Tense (Preterite) — Practice

1. Conjugate "hablar" for "yo" in past:
Answer: hablé

Explanation: For "yo" with -AR verbs, remove -ar and add -é. hablar → habl + é = hablé

2. Complete: "Ellos _______ (comer) en el restaurante."
Answer: comieron

Explanation: For "ellos" with -ER verbs, add -ieron. comer → com + ieron = comieron

3. Conjugate "escribir" for "yo" in past:
Answer: escribí

Explanation: For "yo" with -IR verbs, remove -ir and add -í. escribir → escrib + í = escribí

4. Complete: "Tú _______ (trabajar) ayer."
Answer: trabajaste

Explanation: For "tú" with -AR verbs, add -aste. trabajar → trabaj + aste = trabajaste

5. Conjugate "vivir" for "nosotros" in past:
Answer: vivimos

Explanation: For "nosotros" with -IR verbs, add -imos. vivir → viv + imos = vivimos

6. Which is the correct past tense for "ella / estudiar"?
Answer: estudió

Explanation: For "ella" with -AR verbs, add -ó. estudiar → estudi + ó = estudió

7. Complete: "Yo _______ (beber) agua esta mañana."
Answer: bebí

Explanation: For "yo" with -ER verbs, add -í. beber → beb + í = bebí

8. Complete: "Nosotros _______ (hablar) con el profesor."
Answer: hablamos

Explanation: For "nosotros" with -AR verbs, the -amos ending is the same in present and past! Context (ayer, anoche) makes clear it's past tense.
