Regular Past Tense (Preterite) — Answer Key

Practice

  1. hablé Explanation: For "yo" with -AR verbs, remove -ar and add -é. hablar → habl + é = hablé
  2. comieron Explanation: For "ellos" with -ER verbs, add -ieron. comer → com + ieron = comieron
  3. escribí Explanation: For "yo" with -IR verbs, remove -ir and add -í. escribir → escrib + í = escribí
  4. trabajaste Explanation: For "tú" with -AR verbs, add -aste. trabajar → trabaj + aste = trabajaste
  5. vivimos Explanation: For "nosotros" with -IR verbs, add -imos. vivir → viv + imos = vivimos
  6. estudió Explanation: For "ella" with -AR verbs, add -ó. estudiar → estudi + ó = estudió
  7. bebí Explanation: For "yo" with -ER verbs, add -í. beber → beb + í = bebí
  8. hablamos Explanation: For "nosotros" with -AR verbs, the -amos ending is the same in present and past! Context (ayer, anoche) makes clear it's past tense.
