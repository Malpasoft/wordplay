Ser & Estar — Practice

1. Which is correct: "Ella ___ mexicana"?
Answer: es

Explanation: Nationality is a permanent identity, so we use SER. "Ella es mexicana" = She is Mexican.

2. Fill in: "El libro _____ en la mesa."
Answer: está

Explanation: Location requires ESTAR. "El libro está en la mesa" = The book is on the table.

3. Which is correct: "Yo ____ profesor"?
Answer: soy

Explanation: Profession is a permanent identity, so we use SER. "Yo soy profesor" = I am a teacher.

4. Fill in: "¿Cómo ______ hoy?" (How are you today?)
Answer: estás

Explanation: Asking about someone's condition/state uses ESTAR. "¿Cómo estás?" = How are you?

5. Which sentence is correct?
Answer: Nosotros somos de España.

Explanation: Origin (where you are FROM) is permanent and uses SER. ESTAR is only for temporary location (where you ARE NOW).

6. Fill in: "La puerta _____ cerrada." (The door is closed.)
Answer: está

Explanation: A current condition or state uses ESTAR. The door is IN A CLOSED STATE now, so "está cerrada".

7. Which is correct: "Mi padre _____ inteligente"?
Answer: es

Explanation: A personality trait or characteristic is permanent, so we use SER. "Mi padre es inteligente" = My father is intelligent.

8. Fill in: "Yo _____ en Barcelona ahora." (I am in Barcelona now.)
Answer: estoy

Explanation: Current location uses ESTAR. "Yo estoy en Barcelona" = I am in Barcelona (right now, but I might leave).
