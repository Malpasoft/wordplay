Gustar — Practice — Exercises

Practice

  1. Fill in: "Me _____ la pizza."
  2. Which is correct: "Te gustan las frutas"?
    (Yes, it's correct / No, should be "Te gusta las frutas") 
  3. Fill in: "A Juan le _____ el fútbol."
  4. Fill in: "Nos _____ viajar." (We like to travel.)
  5. Which pronoun is correct? "_____ gusta la música." (She likes music.)
    (Me / Te / Le) 
  6. Fill in: "Me gustan _____ películas."
  7. Which sentence is correct?
    (Nos gusta cantando / Nos gusta cantar) 
  8. Fill in: "Les _____ los libros." (They like books.)

Gustar — Practice — Answer Key

Practice

  1. gusta Explanation: "Pizza" is singular, so we use gusta (not gustan). "Me gusta la pizza" = I like pizza (pizza is pleasing to me).
  2. Yes, it's correct Explanation: "Frutas" is plural, so gustan is correct. "Te gustan las frutas" = You like fruits (fruits are pleasing to you).
  3. gusta Explanation: "Fútbol" (soccer) is singular. The prepositional phrase "A Juan" clarifies who we're talking about. "A Juan le gusta el fútbol" = Juan likes soccer.
  4. gusta Explanation: With infinitives (viajar), always use gusta (singular), even with nosotros. "Nos gusta viajar" = We like to travel.
  5. Le Explanation: For "she" (ella), we use le. "Le gusta la música" = She likes music. Me = I, Te = you, Le = he/she/you formal.
  6. las Explanation: "Películas" (movies) is plural feminine. You need the definite article "las" to make the sentence complete. "Me gustan las películas" = I like movies.
  7. Nos gusta cantar Explanation: With gustar, you use the infinitive (cantar), not the gerund (cantando). "Nos gusta cantar" = We like to sing.
  8. gustan Explanation: "Libros" (books) is plural, so we use gustan. "Les gustan los libros" = They like books.
