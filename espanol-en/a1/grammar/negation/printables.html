Negation — Practice — Exercises

Practice

  1. Make this negative: "Hablo español."
  2. Complete: "No tengo _____ hermanos." (I don't have any brothers.)
    (ningunos / ningún / ninguno) 
  3. Which sentence is correct?
    (Nunca no voy. / No voy nunca.) 
  4. Complete: "No conozco a _____." (I don't know anybody.)
  5. Which negation word means "never"?
    (nada / nunca / ninguno) 
  6. Complete: "No como _____ por la mañana." (I don't eat anything in the morning.)
  7. Make negative: "Tengo algunos amigos." (I have some friends.)
  8. Which word order is correct for starting with "Nunca"?
    (Nunca voy al cine. / No nunca voy al cine.) 

Negation — Practice — Answer Key

Practice

  1. No hablo español. Explanation: Put "no" right before the verb. "No hablo" = I don't speak.
  2. ningún Explanation: Use "ningún" before masculine plural. It means "not any" or "no".
  3. No voy nunca. Explanation: "No voy nunca" (I never go) is the correct double negation. "Nunca no voy" is incorrect word order.
  4. nadie Explanation: "Nadie" means "nobody/anybody" and requires "no" before the verb in standard Spanish.
  5. nunca Explanation: "Nunca" = never. "Nada" = nothing. "Ninguno" = none.
  6. nada Explanation: "Nada" = nothing. "No como nada" is a correct double negation.
  7. No tengo ningún amigo. Explanation: "Algunos" becomes "ningún" in negation. "No tengo ningún amigo" = I don't have any friends.
  8. Nunca voy al cine. Explanation: When a negative word starts the sentence, "no" is optional. "Nunca voy" = I never go.
