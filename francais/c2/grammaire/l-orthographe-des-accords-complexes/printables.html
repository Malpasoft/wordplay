L'Orthographe des Accords Complexes — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : Accords complexes

Concept Forme Exemple
PP avec avoir : accord si COD antéposé (sauf 'en' partitif et 'fait' + inf.).
PP pronominal : accord si SE = COD ; invariable si SE = COI ('se sont parlé').
Adjectifs de couleur composés : invariables (bleu marine, gris clair).
'Tout' adverbe : invariable sauf devant adj. féminin commençant par consonne.
'Ci-joint' : invariable en tête ; variable après le nom ou le verbe.

Erreurs fréquentes

Incorrect Correct Explication
Accord de PP avec 'en'*'Des pommes ? J'en ai mangées.' — faute classique. Après 'en' partitif, le PP est toujours invariable : 'J'en ai mangé'. Le 'en' représente une quantité indéfinie et ne peut pas être antéposé au sens syntaxique requis pour l'accord.
Accord de 'fait' + infinitif*'Les pièces qu'il a faites réparer.' — faute fréquente. 'Fait' + infinitif est toujours invariable : 'Les pièces qu'il a fait réparer.' Seule exception : 'elle s'est faite belle' (faire attributif).
Accord de 'tout' adverbe devant adjectif féminin*'Elles sont toutes timides' — faux, car 'timide' commence par une consonne au féminin → accord : 'elles sont toutes timides'. Mais : '*elles sont toutes étonnées' → 'tout étonnées' (adjectif à initiale vocalique → pas d'accord de l'adverbe 'tout'). Règle d'euphonie souvent mal appliquée.
'Ci-joint' invariable ou variable ?En tête de phrase ou directement avant le nom : invariable ('Ci-joint la facture'). Après le nom : s'accorde ('La facture ci-jointe'). Après un verbe : s'accorde ('Vous trouverez ci-jointe la facture'). Cette règle est systématiquement testée dans les concours administratifs.

Exercice 1 —

1. Choisissez la forme correcte : 'Les lettres que j'ai _____ (envoyer).' [envoyé / envoyés / envoyées / envoyer]

Exercice 2 —

1. Expliquez la différence d'accord : 'Elle s'est lavée.' vs 'Elle s'est lavé les mains.'  

Exercice 3 —

1. Quelle est la règle d'accord pour les adjectifs de couleur composés ('bleu marine', 'gris clair') ? [Ils s'accordent normalement avec le nom / Ils sont toujours invariables / Ils s'accordent uniquement au féminin / Ils s'accordent au pluriel uniquement]

Clé des réponses

Exercice 1 : 1. envoyées

Exercice 2 : 1. 'Elle s'est lavée' : SE est COD (= elle a lavé elle-même), donc PP s'accorde avec SE (féminin singulier). 'Elle s'est lavé les mains' : SE est COI (= elle a lavé les mains à elle-même), le COD est 'les mains' (après le verbe) → pas d'accord du PP.

Exercice 3 : 1. Ils sont toujours invariables
