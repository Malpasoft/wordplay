L'Orthographe des Accords Complexes

Maîtriser les accords difficiles du participe passé et de l'adjectif

Récapitulatif des accords du participe passé

Avec avoir : accord avec le COD si celui-ci est antéposé au verbe. 'La lettre que j'ai écrite.' Avec être : accord avec le sujet. 'Elle est partie.' Verbes pronominaux réfléchis/réciproques : accord si le pronom SE est COD. 'Elle s'est lavée' (COD = se = elle) vs 'Elle s'est lavé les mains' (COD = les mains, après).

Les verbes pronominaux et l'accord du PP

Se verbe : si SE est COD → accord. 'Ils se sont vus.' (= ils ont vu eux-mêmes) Si SE est COI → pas d'accord. 'Ils se sont parlé.' (= ils ont parlé à eux-mêmes) Verbes essentiellement pronominaux (toujours se) : accord systématique. 'Elle s'est enfuie / souvenue.'

Le participe passé avec 'faire' + infinitif

Toujours invariable : 'Elle les a fait venir.' / 'Les tableaux qu'il a fait peindre.' Le COD ne déclenchera jamais l'accord car 'fait' gouverne l'infinitif, non le COD directement. Exception : 'elle s'est faite belle' (faire = rendre, attributif, non causatif).

Le participe passé des verbes impersonnels

Toujours invariable : 'Les inondations qu'il y a eu.' / 'La chaleur qu'il a fait.' Le sujet grammatical est toujours 'il' impersonnel — jamais de vrai COD antéposé au participe.

Accord de l'adjectif avec plusieurs sujets

Sujets coordonnés avec 'et' : accord au pluriel. 'Sa prudence et sa générosité étaient remarquables.' Sujets coordonnés avec 'ou/ni' : accord selon le sens. Si exclusion mutuelle → singulier : 'Ni l'un ni l'autre n'est arrivé.' Si les deux sont possibles → pluriel : 'Ni sa prudence ni sa bonté ne manquaient.'

L'accord des adjectifs de couleur

Adjectif simple : accord normal. 'Des robes vertes.' Adjectif composé ou nom utilisé comme adjectif : invariable. 'Des robes bleu marine / des yeux noisette / des tenues kaki.' Exception : 'rose', 'mauve', 'écarlate', 'pourpre' s'accordent car ce sont des adjectifs à part entière.

Demi-, semi-, mi-, nu-

Toujours invariables avant un nom ou adjectif : 'à demi nu', 'des semi-remorques', 'mi-temps'. Après un nom relié par un tiret : 'des bras nus' mais 'nu-pieds' (invariable composé figé). 'À demi' = invariable ; 'demi' devant un nom + tiret = invariable. 'Deux heures et demie' = s'accorde quand après le nom.

Participe passé après 'en'

Après 'en' pronom, le participe passé est toujours invariable : 'Des pommes ? J'en ai mangé.' Le COD 'en' est partitif et ne déclenche jamais l'accord, même antéposé.

Accord de 'tout'

Adjectif : accord en genre et nombre. 'Toutes les fleurs.' Adverbe devant un adjectif : invariable, SAUF devant un adjectif féminin commençant par une consonne ou un h aspiré (euphonie). 'Elles sont toutes honteuses / tout étonnées.' Pronom : 'Tout est bien qui finit bien.' Nom : 'Le tout est de comprendre.'

Accords dans les locutions figées

Certaines locutions au participe passé sont invariables par convention : 'Ci-joint les documents.' / 'Ci-annexé la liste.' (invariables en tête de phrase ou avant le nom). 'Les documents ci-joints.' (s'accorde après le nom). 'Étant donné les circonstances' (invariable) vs 'les circonstances étant données' (s'accorde).

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Accord de PP avec 'en'
→
*'Des pommes ? J'en ai mangées.' — faute classique. Après 'en' partitif, le PP est toujours invariable : 'J'en ai mangé'. Le 'en' représente une quantité indéfinie et ne peut pas être antéposé au sens syntaxique requis pour l'accord.
Accord de 'fait' + infinitif
→
*'Les pièces qu'il a faites réparer.' — faute fréquente. 'Fait' + infinitif est toujours invariable : 'Les pièces qu'il a fait réparer.' Seule exception : 'elle s'est faite belle' (faire attributif).
Accord de 'tout' adverbe devant adjectif féminin
→
*'Elles sont toutes timides' — faux, car 'timide' commence par une consonne au féminin → accord : 'elles sont toutes timides'. Mais : '*elles sont toutes étonnées' → 'tout étonnées' (adjectif à initiale vocalique → pas d'accord de l'adverbe 'tout'). Règle d'euphonie souvent mal appliquée.
'Ci-joint' invariable ou variable ?
→
En tête de phrase ou directement avant le nom : invariable ('Ci-joint la facture'). Après le nom : s'accorde ('La facture ci-jointe'). Après un verbe : s'accorde ('Vous trouverez ci-jointe la facture'). Cette règle est systématiquement testée dans les concours administratifs.

Résumé : Accords complexes

PP avec avoir : accord si COD antéposé (sauf 'en' partitif et 'fait' + inf.).
PP pronominal : accord si SE = COD ; invariable si SE = COI ('se sont parlé').
Adjectifs de couleur composés : invariables (bleu marine, gris clair).
'Tout' adverbe : invariable sauf devant adj. féminin commençant par consonne.
'Ci-joint' : invariable en tête ; variable après le nom ou le verbe.
