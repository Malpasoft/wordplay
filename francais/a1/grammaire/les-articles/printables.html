Référence, exercices et clé des réponses à imprimer.

Référence : Les Articles

Concept Forme Exemple
le / la / l'défini singulierle chat, la table, l'ami
lesdéfini plurielles chats, les tables
un / uneindéfini singulierun chien, une fleur
desindéfini plurieldes chiens, des fleurs
l' (élision)le/la devant voyelle ou h muetl'ami, l'école, l'hôpital

Erreurs fréquentes

Incorrect Correct Explication
le amil'amiLe s'élide devant une voyelle : le + ami = l'ami.
J'ai un informations.J'ai des informations.Informations est toujours pluriel : utilise des.
Je suis une professeure.Je suis professeure.Après être, le métier s'emploie sans article indéfini (sauf avec adjectif).
un eauune eauEau est féminin : une eau (ou l'eau avec l'article défini).
des bon livresde bons livresDevant un adjectif pluriel, des devient de.

Exercice 1 — Articles définis ou indéfinis ?

Choisis le bon article pour compléter chaque phrase.

1. Je vois _____ chien dans le parc. [le / un / une / des]
2. _____ Tour Eiffel est à Paris. [Une / La / Le / L']
3. Elle a _____ amie à Lyon. [le / un / une / des]
4. _____ enfants jouent dans le jardin. [Le / Un / Des / Les]
5. Il mange _____ pomme. [le / la / une / un]

Exercice 3 — Genre des noms

Choisis l'article défini correct selon le genre du nom.

1. _____ voiture est rouge. [Le / La / L' / Les]
2. _____ avion est en retard. [Le / La / L' / Les]
3. Il y a _____ arbres dans le parc. [le / la / l' / les]
4. _____ eau est froide. [Le / La / L' / Les]
5. C'est _____ beau film. [le / la / un / une]

Clé des réponses

Exercice 1 : 1. un 2. La 3. une 4. Les 5. une

Exercice 2 : 1. l'arbre 2. l'école 3. le hibou 4. l'université 5. l'homme
