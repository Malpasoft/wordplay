Le Présent Irrégulier — Aller, Faire, Pouvoir, Vouloir — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : Le Présent Irrégulier

Concept Forme Exemple
je vais / tu vas / il vaaller singulierJe vais bien.
nous allons / vous allez / ils vontaller plurielIls vont en France.
je/tu fais / il fait / ils fontfaire présentIl fait beau.
je/tu peux / il peut / ils peuventpouvoir présentJe peux venir.
je/tu veux / il veut / ils veulentvouloir présentElle veut partir.
aller + infinitiffutur procheJe vais manger.

Erreurs fréquentes

Incorrect Correct Explication
vous faisezvous faitesFaire est irrégulier : vous faites (pas faisez).
Je vas bien.Je vais bien.Aller : je vais (pas je vas).
ils peuvent (ils peuves)ils peuventPouvoir : ils peuvent (irrégulier).
Je veux manger. / Je veux de manger.Je veux manger.Vouloir + infinitif sans préposition : je veux manger (pas de manger).
Il va à faire du sport.Il va faire du sport.Futur proche : aller + infinitif sans préposition : il va faire.

Exercice 1 — Conjugaison des verbes irréguliers

Choisis la forme correcte.

1. Tu _____ au cinéma ? (aller) [vais / vas / va / allons]
2. Il _____ ses devoirs. (faire) [fais / faites / fait / font]
3. Vous _____ du sport ? (faire) [fait / fais / faites / font]
4. Je _____ venir ce soir. (pouvoir) [peux / peut / pouvez / peuvent]
5. Ils _____ partir maintenant. (vouloir) [veut / voulons / veulent / voulez]

Exercice 2 — Futur proche

Forme le futur proche avec aller + infinitif.

1. Je / manger (demain)  
2. Elle / partir (ce soir)  
3. Nous / visiter (Paris)  
4. Ils / arriver (bientôt)  
5. Tu / faire (quoi ?)  

Exercice 3 — Pouvoir ou Vouloir ?

Choisis le bon verbe selon le contexte.

1. J'ai la permission. → Je _____ venir. [peux / veux / vais / fais]
2. C'est mon désir. → Je _____ un café. [peux / veux / vais / fais]
3. Il n'a pas la capacité. → Il ne _____ pas nager. [peut / veut / va / fait]
4. Nous avons envie. → Nous _____ aller au restaurant. [pouvons / voulons / allons / faisons]
5. Il fait beau. → On _____ faire du vélo. [peut / veut / va / fait]

Clé des réponses

Exercice 1 : 1. vas 2. fait 3. faites 4. peux 5. veulent

Exercice 2 : 1. je vais manger 2. elle va partir 3. nous allons visiter 4. ils vont arriver 5. tu vas faire

Exercice 3 : 1. peux 2. veux 3. peut 4. voulons 5. peut
