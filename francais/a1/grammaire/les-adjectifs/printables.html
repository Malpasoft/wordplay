Erreurs fréquentes

Incorrect Correct Explication

Elle est grand.Elle est grande.L'adjectif s'accorde : sujet féminin → adjectif féminin (grand → grande).

Exercice 1 — Accord de l'adjectif

Choisis la forme correcte de l'adjectif.

1. C'est une _____ fille. (petit) [petit / petite / petits / petites]
2. Ce sont des _____ garçons. (grand) [grand / grande / grands / grandes]
3. Elle est _____. (heureux) [heureux / heureuse / heureux / heureuses]
4. C'est une _____ idée. (bon) [bon / bonne / bonnes / bons]
5. Les maisons sont _____. (blanc) [blanc / blanche / blancs / blanches]

Exercice 3 — Position de l'adjectif

L'adjectif va avant ou après le nom ?

1. C'est une _____ maison. (petit) [petite maison / maison petite / maison petits / petits maison]
2. J'ai un sac _____. (rouge) [rouge sac / rouge / sacs rouge / sac rouge]
3. C'est un _____ film. (bon) [bon film / film bon / films bon / bons film]
4. Il a les yeux _____. (bleu) [bleus yeux / bleu yeux / yeux bleus / yeux bleu]
5. Voilà une _____ personne. (beau) [belle personne / personne belle / beau personne / belle personnes]

Clé des réponses

Exercice 1 : 1. petite 2. grands 3. heureuse 4. bonne 5. blanches

Exercice 2 : 1. grande 2. heureuse 3. belle 4. légère 5. vieille

Exercice 3 : 1. petite maison 2. sac rouge 3. bon film 4. yeux bleus 5. belle personne
