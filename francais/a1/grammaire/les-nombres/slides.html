Les nombres composés : 21 à 99

Pour les nombres entre les dizaines, on ajoute un trait d'union :

21 : vingt et un (avec "et")

22 : vingt-deux (avec trait d'union)

71 : soixante et onze

72 : soixante-douze

81 : quatre-vingt-un

91 : quatre-vingt-onze

Note : 80 = quatre-vingts (avec s), mais 81 = quatre-vingt-un (sans s).

Les nombres ordinaux

Les nombres ordinaux (1er, 2e, 3e...) :

1er/1re premier / première
2e deuxième
3e troisième
4e quatrième
5e cinquième
6e sixième
7e septième
10e dixième

Règle générale : nombre cardinal + -ième (deuxième, troisième...).

Exception : 1er premier / 1re première.

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

septante
→
soixante-dix
En France, 70 = soixante-dix (pas septante, qui est utilisé en Belgique et Suisse).
huitante / octante
→
quatre-vingts
En France, 80 = quatre-vingts (pas huitante ni octante).
quatre-vingt-un (quatre-vingts un)
→
quatre-vingt-un
Devant un autre nombre, quatre-vingts perd son -s : quatre-vingt-un.
vingt-et-deux
→
vingt-deux
Et ne s'utilise qu'avec les unités 1 : vingt et un, trente et un, etc. Pour les autres : vingt-deux.
le deuxième étage (2ème)
→
le deuxième étage ✓
Cette phrase est correcte. 2ème et 2e sont tous les deux acceptés en abréviation.
