Référence : Remplir un Formulaire

Concept Forme Exemple
Nom / PrénomNom = famille (MAJUSC.) / Prénom = first nameNOM : DUPONT · PRÉNOM : Marie
DateJJ/MM/AAAA — mois en minuscule15/03/1998 = le 15 mars 1998
Adressenuméro + rue + code postal + ville12, rue des Fleurs · 75008 Paris
Nationalitéminuscule, accordé en genrefrançais / française
Situationcélibataire / marié(e) / divorcé(e)Situation de famille : célibataire
SignatureFait à + ville, le + dateFait à Lyon, le 15 juin 2026

Erreurs fréquentes

Incorrect Correct Explication
Nom : Marie / Prénom : DupontNom : Dupont / Prénom : MarieSur un formulaire français, Nom = famille (Dupont), Prénom = premier prénom (Marie).
Date : 03/15/1998Date : 15/03/1998En France, l'ordre est JJ/MM/AAAA, pas MM/DD/YYYY comme en anglais américain.
Nationalité : FrançaisNationalité : français(e)La nationalité prend la minuscule en français, même sur un formulaire.
Adresse : Paris 75008Adresse : 75008 ParisLe code postal (5 chiffres) précède le nom de la ville en France.
Mois : JanvierMois : janvierLes noms de mois prennent la minuscule en français.

Exercice 1 — Comprendre le formulaire

Choisissez la réponse correcte.

1. Sur un formulaire français, « Nom » signifie : [votre prénom / votre nom de famille / votre surnom / votre nationalité]
2. Comment s'écrit la date 3 juillet 2001 sur un formulaire français ? [07/03/2001 / 03/07/2001 / 2001/07/03 / 3-JUL-2001]
3. Quel est l'ordre correct d'une adresse française ? [Ville + code postal + rue / Code postal + ville + rue / Rue + code postal + ville / Rue + ville]
4. Comment écrire la nationalité sur un formulaire ? [Français / français / FRANÇAIS / FR]

Exercice 2 — Remplir les champs

Complétez avec le mot manquant.

1. ___ : MARTIN (votre nom de famille en majuscules)  
2. ___ : Thomas (votre premier prénom)  
3. Date de naissance : ___/06/2000 (le 8 juin)  
4. ___ de famille : célibataire  
5. Fait à Paris, le 15 ___ 2026  

Exercice 3 — Identifier l'erreur

Quelle réponse contient une erreur ?

1. Quel formulaire est correctement rempli ? [NOM : PETIT · PRÉNOM : Julie / Date : 25/12/1995 / Code postal : 69001 Lyon / Nationalité : Espagnol]
2. Quelle date est au format français ? [12/31/1990 / 31/12/1990 / 1990-31-12 / 31.DEC.1990]
3. Quelle adresse est au format français ? [5, avenue Victor Hugo · 13001 Marseille / Marseille 13001 · avenue Victor Hugo 5 / 13001 · avenue Victor Hugo · 5 / avenue Victor Hugo, 5, Marseille]

Clé des réponses

Exercice 1 : 1. votre nom de famille 2. 03/07/2001 3. Rue + code postal + ville 4. français

Exercice 2 : 1. Nom 2. Prénom 3. 08 4. Situation 5. juin

Exercice 3 : 1. Nationalité : Espagnol 2. 31/12/1990 3. Marseille 13001 · avenue Victor Hugo 5
