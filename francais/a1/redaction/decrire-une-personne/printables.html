Décrire une Personne — Apparence et Caractère — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : Décrire une Personne

Concept Forme Exemple
Tailleêtre + grand(e) / petit(e) / de taille moyenneIl est grand et mince.
Cheveuxavoir les cheveux + couleur + formeElle a les cheveux longs et blonds.
Yeuxavoir les yeux + couleurIl a les yeux marron. (invariable)
Caractèreêtre + adjectif accordéElle est sympa et généreuse.
Accord adj.masc. → fém. : +e / -eux→-euse / -il→-illesérieux → sérieuse · gentil → gentille
Ordrenom → âge → taille → cheveux → yeux → caractèreMon ami Lucas a 30 ans. Il est grand…

Erreurs fréquentes

Incorrect Correct Explication
Elle a grand.Elle est grande.La taille s'exprime avec être, pas avoir.
Il a les yeux marrons.Il a les yeux marron.Marron est invariable en français : pas de -s au pluriel.
Elle est une gentille.Elle est gentille.Après être + adjectif de caractère, pas d'article.
Il a les cheveux le brun.Il a les cheveux bruns.La couleur des cheveux est un adjectif, pas un nom avec article.
Elle est sérieux.Elle est sérieuse.L'adjectif s'accorde avec le sujet : sérieux (m.) → sérieuse (f.).

Exercice 1 — Choisir la bonne description

Choisissez la réponse correcte.

1. Comment décrit-on la taille en français ? [Il a grand. / Il est grand. / Il fait grand. / Il a de la grandeur.]
2. Laquelle est correcte ? [Elle a les yeux marrons. / Elle a les yeux marron. / Elle est les yeux marron. / Elle a yeux marron.]
3. Comment dit-on qu'elle est sympa ? [Elle a sympa. / Elle est sympa. / Elle fait sympa. / Elle a de la sympa.]
4. Quelle est la forme féminine de « sérieux » ? [sérieuse / sérieusse / sérieuxe / sérieux]

Exercice 2 — Compléter la description

Complétez avec le mot manquant.

1. Il ___ les cheveux blonds.  
2. Elle ___ grande et mince.  
3. Il a les yeux ___. (couleur invariable)  
4. Elle est ___. (féminin de « gentil »)  
5. Il a les cheveux courts et ___. (couleur : brun au pluriel)  

Exercice 3 — Identifier l'erreur

Quelle phrase contient une erreur ?

1. Choisissez la phrase correcte. [Il est grand. / Elle a les yeux bleus. / Il a les cheveux roux. / Elle a petite.]
2. Trouvez l'erreur. [Il est sympa et drôle. / Elle est sérieuse. / Il est un gentil. / Elle est timide.]
3. Quelle description est incorrecte ? [Elle a les yeux verts. / Il a les cheveux bruns. / Elle a les yeux marron. / Il a les cheveux le noir.]

Clé des réponses

Exercice 1 : 1. Il est grand. 2. Elle a les yeux marron. 3. Elle est sympa. 4. sérieuse

Exercice 2 : 1. a 2. est 3. marron 4. gentille 5. bruns

Exercice 3 : 1. Elle a petite. 2. Il est un gentil. 3. Il a les cheveux le noir.
