Les Propositions Infinitives

Infinitif comme substitut de proposition subordonnée

Infinitif après verbes de perception

Voir, entendre, sentir, regarder, écouter + nom + infinitif (COD de la perception).
J'entends les oiseaux chanter.
Je vois les enfants jouer.
Le nom est à la fois COD du verbe principal et sujet logique de l'infinitif.

Infinitif après faire, laisser, envoyer

Faire + infinitif = faire réaliser par quelqu'un d'autre (causatif) :
Je fais réparer ma voiture. (par un mécanicien)
Elle fait travailler ses étudiants.
Laisser + inf. = permettre : Il laisse les enfants jouer.

Place des pronoms avec faire + infinitif

Le COD se place avant le verbe faire, pas avant l'infinitif :
Je la fais réparer. (= je fais réparer la voiture; 'la' = la voiture)
Il les fait travailler.
Exception : se faire + infinitif = subir : Je me suis fait voler mon sac.

Infinitif passé : après avoir / après être

Après avoir + PP = action antérieure au verbe principal (même sujet) :
Après avoir mangé, il est sorti.
Après être parti, elle a appelé.
Ne jamais utiliser après que + subjonctif — l'indicatif s'impose après après que.

Pour + infinitif : but et cause négative

But : Il travaille dur pour réussir.
Cause négative (reproche) : Il a été puni pour avoir triché.
Simultanéité avec le verbe principal : infinitif présent.
Antériorité : infinitif passé (pour avoir + PP).

Sans, avant de, afin de + infinitif

Sans + inf. : Il est parti sans dire au revoir.
Avant de + inf. : Réfléchissez avant de répondre.
Afin de + inf. : but formel. Elle a étudié afin de réussir.
Ces prépositions précèdent toujours l'infinitif (pas une proposition finie).

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Après que + subjonctif
→
'Après que' prend l'indicatif en français standard, pas le subjonctif. *Après qu'il soit parti → Après qu'il est parti / After leaving → Après être parti (même sujet).
Faire + infinitif : ordre du COD
→
Le pronom va avant faire, pas avant l'infinitif. *Je fais la réparer → Je la fais réparer.
Après avoir vs Après que
→
'Après avoir + PP' = même sujet que le verbe principal (infinitif passé). 'Après que' = sujets différents (indicatif). Après être parti (même sujet) / Après qu'il est parti (sujets différents).
Pour avoir + PP = cause passée
→
'Pour avoir triché' = parce qu'il a triché (cause passée). Ne pas confondre avec le but 'pour tricher' (but présent/futur).

Résumé : Les Propositions Infinitives

Verbes de perception + nom + infinitif (voir, entendre, sentir).
Faire + infinitif = causatif (faire faire par qqn d'autre).
Pronoms avant faire, pas avant l'infinitif.
Après avoir/être + PP = antériorité (même sujet).
Sans, avant de, afin de, pour + infinitif (présent ou passé).
