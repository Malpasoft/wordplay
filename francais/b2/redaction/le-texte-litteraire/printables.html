Le Texte Littéraire — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : Le Texte Littéraire

Concept Forme Exemple
Genres littéraires : narratif, descriptif, lyrique, argumentatif, satirique.
Narration au passé : passé simple (actions), imparfait (description), plus-que-parfait (antériorité).
Figures de style au service du sens et de l'émotion.
Description sensorielle et style varié (alternance courtes/longues phrases).

Erreurs fréquentes

Incorrect Correct Explication
Passé composé dans la narration littéraireutiliser le passé composé dans un texte littéraire au lieu du passé simple ; les grands textes français utilisent le passé simple pour les actions narratives.
Description sans ancrage sensoriellister des éléments visuels sans faire appel aux autres sens ; une description riche mobilise vue, ouïe, odorat, toucher.
Figures de style plaquéesinsérer des métaphores sans lien avec le propos ; les figures de style doivent servir le sens et l'émotion.
Dialogues mal ponctusoublier le tiret cadratin (—) pour les répliques et le guillemet français (« ») pour l'introduction du dialogue.

Exercice 1 —

1. Quel temps verbal est utilisé pour les actions principales dans la narration littéraire française ? [Le passé composé / L'imparfait / Le passé simple / Le plus-que-parfait]

Exercice 2 —

1. Identifiez la figure de style : 'Le soleil s'endormit derrière la colline.'  

Exercice 3 —

1. Quelle technique de description littéraire est recommandée pour rendre une scène vivante ? [Lister tous les éléments visuels de la scène / Mobiliser les cinq sens et ancrer dans la perception d'un personnage / Utiliser uniquement des adjectifs précis / Citer des sources historiques]

Clé des réponses

Exercice 1 : 1. Le passé simple

Exercice 2 : 1. personnification
