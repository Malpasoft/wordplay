Récapitulatif des Temps du Passé

PC, imparfait, PQP — choisir le bon temps

Les quatre temps du passé en usage courant

Passé composé (PC) : action accomplie, datée ou délimitée dans le passé.
Imparfait (IMP) : état, description, habitude ou action en cours dans le passé.
Plus-que-parfait (PQP) : action antérieure à une autre action passée.
Passé simple (PS) : écrit littéraire — narration formelle.

Passé composé vs Imparfait

PC = événement ponctuel, séquence d'actions, résultat visible.
IMP = décor, contexte, état mental/physique, habitude (souvent/toujours/chaque jour).
Il pleuvait (IMP) quand je suis sorti (PC).

Plus-que-parfait : l'antécédent du passé

PQP = action déjà terminée AVANT l'action principale au passé.
Quand il est arrivé, elle avait déjà préparé le repas.
Marqueurs : déjà, avant de, parce que, comme.

Storytelling : les trois temps ensemble

Dans un récit au passé, les trois temps se combinent :
IMP = décor / états → Il faisait nuit, les rues étaient désertes.
PQP = contexte antérieur → Il avait travaillé toute la journée.
PC = action narrative → Soudain, il a entendu un bruit.

Expressions de temps associées

PC : soudain, tout à coup, un jour, puis, ensuite, finalement.
IMP : toujours, souvent, chaque jour, à cette époque, d'habitude, quand (+ état).
PQP : déjà, avant que, parce que (+ raison antérieure).

Passé simple — lecture uniquement

Le PS est réservé à la littérature et l'écrit formel. On le reconnaît mais ne le produit pas en oral. Il parla (PC : il a parlé), il prit (PC : il a pris), ils vinrent (PC : ils sont venus).

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Avant / Après
→
'Avant de + inf.' = action non encore faite. 'Après + inf. passé' = action déjà faite. Pour l'antériorité dans le passé → PQP, pas PC seul.
Habitude = toujours imparfait
→
Quand on décrit une habitude passée (souvent, chaque matin, tous les jours…), utiliser l'imparfait, même pour des actions qui semblent ponctuelles.
Confusion PC / PQP
→
PC marque une action dans la ligne narrative principale. PQP marque un flash-back antérieur. Il était parti avant que j'arrive (PQP), pas il est parti.
Passé simple en lecture
→
À l'écrit, -a/-it/-ut/-int sont souvent du passé simple. Les reconnaître mais les reformuler en PC à l'oral.

Résumé : Récapitulatif des Temps du Passé

PC = action accomplie, ponctuelle ou délimitée dans le passé.
Imparfait = état, habitude, contexte, action en cours.
PQP = action antérieure à une autre action passée.
Dans un récit : IMP (décor) + PQP (contexte) + PC (événements).
Passé simple : uniquement à l'écrit formel — reconnaître, pas produire.
