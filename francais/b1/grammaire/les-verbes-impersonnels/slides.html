Les Verbes Impersonnels

Il faut, il s'agit de, il vaut mieux…

Qu'est-ce qu'un verbe impersonnel ?

Un verbe impersonnel s'emploie uniquement à la 3e personne du singulier avec le sujet il grammatical (pas de référent réel). La liste inclut des verbes météo, des expressions de nécessité/opinion, et des constructions avec d'autres verbes.

Météo et existence

il pleut, il neige, il fait chaud/froid/beau/mauvais, il y a, il vente.
Ces verbes n'ont pas d'autre sujet possible : *'il' est toujours le sujet.

Nécessité et obligation

il faut + infinitif : Il faut partir maintenant.
il faut que + subjonctif : Il faut que tu viennes.
il est nécessaire de + inf. · il est indispensable de + inf.

Opinion et probabilité

il semble que + subjonctif : Il semble qu'il soit absent.
il paraît que + indicatif : Il paraît qu'il est malade.
il est possible que + subjonctif · il est probable que + indicatif · il est évident que + indicatif.

Il s'agit de et il vaut mieux

Il s'agit de + nom/inf. : Il s'agit d'un problème grave. / Il s'agit de trouver une solution.
Il vaut mieux + inf. : Il vaut mieux partir tôt.
Il vaut mieux que + subjonctif : Il vaut mieux qu'il parte.

Constructions impersonnelles avec d'autres verbes

Tout verbe peut devenir impersonnel avec construction passive/neutre :
Il se passe quelque chose d'étrange.
Il existe plusieurs solutions.
Il reste deux places.
Il manque un détail.

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Il paraît que vs il semble que
→
'Il paraît que' + indicatif (information entendue). 'Il semble que' + subjonctif (impression personnelle).
Il est probable vs il est possible
→
'Il est probable que' + indicatif (forte probabilité). 'Il est possible que' + subjonctif (simple possibilité).
Il faut sans que vs avec que
→
'Il faut partir' (sujet de la principale et de l'infinitif identique ou indéterminé). 'Il faut que tu partes' (nécessité pour un sujet spécifique → subjonctif).
Il reste / Il manque
→
Ces constructions impersonnelles ont un sujet logique postposé : Il reste deux billets (= deux billets restent). L'accord du verbe est toujours avec 'il' (3e sg).

Résumé : Les Verbes Impersonnels

Météo : il pleut, il neige, il fait beau/chaud/froid.
Nécessité : il faut + inf. / il faut que + subjonctif.
Opinion : il semble que + subj.; il paraît que + ind.; il est possible que + subj.
Il s'agit de + nom/inf.; il vaut mieux + inf.
Constructions neutres : il existe, il reste, il manque + nom.
