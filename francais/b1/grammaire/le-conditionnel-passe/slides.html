Le Conditionnel Passé

Regrets, reproches et conditions irréelles passées

Formation

Auxiliaire (être/avoir) au conditionnel présent + participe passé.
j'aurais mangé · tu serais parti(e) · il aurait voulu · nous aurions compris · vous seriez restés · elles auraient su

Emploi 1 : condition irréelle dans le passé

Si + plus-que-parfait → conditionnel passé.
Si j'avais étudié, j'aurais réussi l'examen.
La condition n'est pas remplie : il n'a pas étudié, donc il n'a pas réussi.

Emploi 2 : regrets

J'aurais voulu être chanteur. (mais je ne l'ai pas été)
Tu aurais pu me prévenir ! (mais tu ne l'as pas fait)
Souvent avec pouvoir, vouloir, devoir, aimer.

Emploi 3 : reproches

Conditionnel passé de devoir, pouvoir + infinitif = reproche.
Tu aurais dû appeler. (tu n'as pas appelé — c'est une critique)
Elle aurait pu arriver à l'heure.

Emploi 4 : fait non confirmé

Dans les médias : Le président aurait déclaré… = l'information n'est pas vérifiée.
Même emploi qu'au conditionnel présent mais pour le passé.

Mêmes auxiliaires et accords qu'au PC

Être pour mouvement/état + pronominaux; avoir pour les autres. Accords identiques : sujet (être), COD antéposé (avoir).

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Si + CP ≠ Si + futur
→
Après si conditionnel, on n'utilise jamais le futur ni le conditionnel. Si + imparfait → cond. présent. Si + PQP → cond. passé.
Conditionnel passé ≠ Futur antérieur
→
Formes similaires : CP = aurais/serais + PP; FA = aurai/serai + PP. L'auxiliaire est au conditionnel (CP) ou au futur (FA).
Accord participe passé
→
Mêmes règles qu'au PC. Avec être → accord sujet. Avec avoir + COD antéposé → accord COD.
Regret vs Reproche
→
Regret = je parle de moi-même : 'J'aurais voulu…'. Reproche = je parle à/de quelqu'un : 'Tu aurais dû…', 'Il aurait pu…'.

Résumé : Le Conditionnel Passé

Formation : avoir/être au conditionnel présent + participe passé.
Condition irréelle passée : Si + PQP → conditionnel passé.
Regrets : j'aurais voulu, tu aurais aimé…
Reproches : tu aurais dû, il aurait pu…
Fait non confirmé dans les médias.
