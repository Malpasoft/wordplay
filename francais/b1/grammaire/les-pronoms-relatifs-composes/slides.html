Les Pronoms Relatifs Composés

Lequel, laquelle, lesquels, lesquelles

Rappel : pronoms relatifs simples

qui (sujet) · que (objet direct) · où (lieu/temps) · dont (de + antécédent).
Les pronoms composés s'utilisent après une préposition (à, avec, pour, sur, dans, par…).

Formes : lequel, laquelle…

Accordent avec le nom antécédent :
masculin singulier → lequel
féminin singulier → laquelle
masculin pluriel → lesquels
féminin pluriel → lesquelles

Contractions avec à et de

à + lequel → auquel
à + lesquels → auxquels
à + lesquelles → auxquelles
à + laquelle → laquelle (pas de contraction)
de + lequel → duquel
de + lesquels → desquels
de + lesquelles → desquelles

Emploi après préposition

C'est le problème auquel je pense. (penser à)
La table sur laquelle j'ai posé mon livre.
Les collègues avec lesquels je travaille.
L'entreprise pour laquelle il travaille.

Dont vs duquel

Dont remplace 'de + antécédent' dans une structure directe.
Mais après une préposition complexe (à côté de, au sujet de, près de…), on utilise duquel/de laquelle/desquels/desquelles.
La maison à côté de laquelle il y a un parc.

Quoi : pronom relatif composé pour les choses abstraites

Après préposition, quoi peut référer à une chose indéfinie ou une proposition :
C'est ce à quoi je pensais.
Je ne sais pas sur quoi compter.

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Accord en genre/nombre
→
'Lequel' s'accorde avec l'antécédent, pas avec le sujet de la proposition relative. La réunion à laquelle j'ai assisté (réunion = féminin → laquelle).
Dont ne suit pas une préposition
→
On ne dit pas *pour dont, *avec dont. 'Dont' remplace 'de + antécédent' directement. Après une autre préposition, utiliser duquel/de laquelle.
Contraction à + laquelle
→
À + laquelle ne se contracte pas : la table à laquelle (pas : *aulaquelle).
Lequel pour les personnes ?
→
Pour les personnes, on préfère qui après une préposition : la personne avec qui je travaille (plus courant que avec laquelle). Pour les objets, on utilise toujours lequel/laquelle…

Résumé : Les Pronoms Relatifs Composés

Pronoms composés = lequel/laquelle/lesquels/lesquelles après préposition.
Accord en genre et nombre avec l'antécédent.
Contractions : à + lequel → auquel; de + lequel → duquel.
Après préposition complexe (à côté de…) → duquel/de laquelle.
Pour personnes + préposition, préférer qui; pour objets → lequel.
