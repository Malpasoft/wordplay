Argumenter Pour et Contre

Structure d'un texte argumentatif équilibré

Structure du texte pour/contre

1. Introduction : Présenter le sujet et annoncer le plan.
2. Arguments pour : 2–3 arguments avec exemples.
3. Arguments contre : 2–3 arguments avec exemples.
4. Conclusion : Synthèse et prise de position personnelle.

Introduire le sujet

De nos jours, … est un sujet qui suscite beaucoup de débats.
La question de … divise l'opinion publique.
Nous allons examiner les arguments pour et contre …
Dans quelle mesure … est-il/elle bénéfique ?

Présenter les arguments pour

Premièrement / D'abord / En premier lieu… · Deuxièmement / Ensuite / De plus…
L'un des principaux avantages est que…
Il convient de souligner que…
Force est de constater que…

Présenter les arguments contre

Cependant / Néanmoins / Toutefois / En revanche…
L'un des inconvénients majeurs est que…
Il faut également noter que…
On peut cependant objecter que…

Illustrer avec des exemples

Par exemple · C'est le cas de · Ainsi · Tel est le cas lorsque…
Chiffres et statistiques : Selon les études, … Pour cent des personnes…
Citations : Comme le dit l'expert X, …

Conclure et prendre position

En définitive / En conclusion / Pour conclure / En somme…
Après avoir examiné les différents aspects…
Il ressort de cette analyse que…
Prise de position : Je considère que… / Il me semble que… / Il est indéniable que…

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Plan en deux parties sans conclusion
→
Un texte pour/contre doit toujours avoir une conclusion avec prise de position. Sans elle, le texte reste incomplet.
Connecteurs répétés
→
Evitez de répéter toujours 'et', 'mais', 'donc'. Variez : d'abord/ensuite, cependant/néanmoins, ainsi/par exemple.
Exemples sans argument
→
Un exemple seul n'est pas un argument. Structure : argument → exemple → analyse de l'exemple.
Introduction = copier-coller du sujet
→
L'introduction doit reformuler et contextualiser le sujet, pas le recopier mot pour mot.

Résumé : Argumenter Pour et Contre

Structure : intro → arguments pour → arguments contre → conclusion.
Intro : De nos jours… · La question de… divise l'opinion.
Arguments pour : premièrement, de plus, l'un des avantages est que.
Arguments contre : cependant, néanmoins, l'un des inconvénients est que.
Conclusion : en définitive, il ressort que + prise de position.
