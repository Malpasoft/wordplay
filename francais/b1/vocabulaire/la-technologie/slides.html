La Technologie

Internet, numérique, innovations et intelligence artificielle

Mots 1–4

la technologie [/la tɛk.nɔ.lɔ.ʒi/]
ensemble des techniques et outils développés par la science
La technologie transforme notre façon de vivre.
le numérique [/lə ny.me.ʁik/]
qui fonctionne avec des données codées en chiffres (bits)
La transformation numérique touche toutes les entreprises.
l'intelligence artificielle [/lɛ̃.tɛ.li.ʒɑ̃s aʁ.ti.fi.sjɛl/]
simulation de l'intelligence humaine par des machines
L'intelligence artificielle révolutionne la médecine.
le réseau social [/lə ʁe.zo sɔ.sjal/]
plateforme en ligne permettant de partager du contenu avec d'autres
Elle passe trop de temps sur les réseaux sociaux.

Mots 5–8

la cybersécurité [/la si.bɛʁ.se.ky.ʁi.te/]
protection des systèmes informatiques contre les attaques
La cybersécurité est cruciale pour les entreprises.
le logiciel [/lə lɔ.ʒi.sjɛl/]
programme informatique exécuté sur un ordinateur
Ce logiciel de traduction est très précis.
la base de données [/la baz də don.ne/]
ensemble structuré d'informations stockées sur un ordinateur
La base de données client est très large.
le cloud [/lə klawd/]
stockage et traitement des données sur Internet
Mes fichiers sont sauvegardés dans le cloud.

Mots 9–12

l'algorithme [/la.lgɔ.ʁitm/]
suite d'instructions permettant de résoudre un problème
L'algorithme de YouTube choisit les vidéos recommandées.
l'application [/la.pli.ka.sjɔ̃/]
logiciel conçu pour un usage spécifique sur mobile ou ordinateur
J'utilise une application pour apprendre le français.
le pirate informatique [/lə pi.ʁat ɛ̃.fɔʁ.ma.tik/]
personne qui s'introduit illégalement dans des systèmes informatiques
Un pirate informatique a volé des données.
la vie privée [/la vi pʁi.ve/]
droit à garder certaines informations personnelles confidentielles
Les réseaux sociaux menacent la vie privée.

Mots 13–15

le streaming [/lə stʁi.miŋ/]
lecture de contenu audio ou vidéo en ligne sans téléchargement
Je regarde des films en streaming.
l'innovation [/li.nɔ.va.sjɔ̃/]
introduction d'une nouveauté technique ou commerciale
L'innovation est au cœur de la croissance.
les données personnelles [/le dɔ.ne pɛʁ.sɔ.nɛl/]
informations relatives à une personne identifiable
Les données personnelles sont protégées par la loi.
