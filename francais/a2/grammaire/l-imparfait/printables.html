L'Imparfait — Descriptions et Habitudes du Passé — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : L'Imparfait

Concept Forme Exemple
Rôledescription, habitude, fond du récitIl faisait beau. · Je jouais au foot.
Formationbase nous-présent + -ais/-ais/-ait/-ions/-iez/-aientparler → parl- → je parlais
Exceptionêtre → étais/étais/était/étions/étiez/étaientJ'étais fatigué.
vs. PCimparfait = fond / PC = actionJe dormais quand il est arrivé.
Marqueurstoujours, souvent, d'habitude → imparfaitD'habitude, il prenait le bus.
Marqueurs PChier, soudain, une fois → passé composéHier, j'ai vu un film.

Erreurs fréquentes

Incorrect Correct Explication
Hier, je travaillais.Hier, j'ai travaillé.« Hier » signale une action unique/terminée → passé composé.
Quand j'étais enfant, j'ai joué au foot tous les jours.Quand j'étais enfant, je jouais au foot tous les jours.« Tous les jours » = habitude → imparfait.
Il a été fatigué.Il était fatigué.Un état ou une description dans le passé s'exprime à l'imparfait.
Je dormissais.Je dormais.Dormir : nous dormons → base dorm- → je dormais. Pas de -iss- à l'imparfait.
Soudain, il pleuvait.Soudain, il a plu.« Soudain » (tout à coup) = événement ponctuel → passé composé.

Exercice 1 — Choisir le bon temps

Passé composé ou imparfait ?

1. D'habitude, elle ___ (prendre) le train. [a pris / prenait / prend / prendra]
2. Hier, il ___ (pleuvoir) toute la journée. [pleuvait / a plu / pluvait / plu]
3. Quand j'étais petit, j'___ (adorer) les glaces. [ai adoré / adorais / adore / adoré]
4. Soudain, le téléphone ___ (sonner). [sonnait / a sonné / sonne / sonnera]

Exercice 2 — Conjuguer à l'imparfait

Donnez la forme correcte.

1. Quand il ___ (être) enfant, il habitait à Nice.  
2. Nous ___ (manger) souvent des pâtes.  
3. Elle ___ (finir) toujours ses devoirs avant le dîner.  
4. Ils ___ (parler) français à la maison.  
5. Tu ___ (avoir) les cheveux longs autrefois.  

Exercice 3 — Identifier l'erreur

Quelle phrase est incorrecte ?

1. Laquelle est incorrecte ? [Je jouais au foot quand j'étais enfant. / Hier, j'ai travaillé. / Soudain, il pleuvait. / D'habitude, elle prenait le bus.]
2. Quelle forme est mal conjuguée ? [je parlais / tu finissais / il dormissait / nous mangions]
3. Quel marqueur ne va PAS avec l'imparfait ? [d'habitude / toujours / soudain / souvent]

Clé des réponses

Exercice 1 : 1. prenait 2. a plu 3. adorais 4. a sonné

Exercice 2 : 1. était 2. mangions 3. finissait 4. parlaient 5. avais

Exercice 3 : 1. Soudain, il pleuvait. 2. il dormissait 3. soudain
