L'Imparfait — Descriptions et Habitudes du Passé

Formation de l'imparfait et distinction avec le passé composé

Qu'est-ce que l'imparfait ?

L'imparfait exprime :

  • Une description dans le passé : Il faisait beau. · Elle était fatiguée.
  • Une habitude passée : Quand j'étais enfant, je jouais au foot.
  • Une action en cours quand une autre arrive : Je dormais quand le téléphone a sonné.
  • Une situation de fond (décor) d'un récit.

L'imparfait n'est jamais utilisé pour une action unique et terminée — c'est le rôle du passé composé.

Formation de l'imparfait

Base : 1re personne du pluriel du présent (nous) — on enlève -ons, on ajoute les terminaisons :

Exception : être

Le seul verbe vraiment irrégulier à l'imparfait est être :
base = ét- (pas étions)

Tous les autres verbes suivent la règle régulière.

Passé composé vs. imparfait

La distinction clé :

Passé composéImparfait
Action terminée, uniqueDescription, habitude, fond
Rupture, événementDurée, continuité
Hier, j'ai travaillé 8 h.Avant, je travaillais à Paris.
Il a sonné à la porte.Il faisait nuit.

Dans un récit : imparfait = décor, passé composé = action.

Erreurs fréquentes

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Hier, je travaillais.
→
Hier, j'ai travaillé.
« Hier » signale une action unique/terminée → passé composé.
Quand j'étais enfant, j'ai joué au foot tous les jours.
→
Quand j'étais enfant, je jouais au foot tous les jours.
« Tous les jours » = habitude → imparfait.
Il a été fatigué.
→
Il était fatigué.
Un état ou une description dans le passé s'exprime à l'imparfait.
Je dormissais.
→
Je dormais.
Dormir : nous dormons → base dorm- → je dormais. Pas de -iss- à l'imparfait.
Soudain, il pleuvait.
→
Soudain, il a plu.
« Soudain » (tout à coup) = événement ponctuel → passé composé.

Résumé : L'Imparfait

Rôle
description, habitude, fond du récit
Il faisait beau. · Je jouais au foot.
Formation
base nous-présent + -ais/-ais/-ait/-ions/-iez/-aient
parler → parl- → je parlais
Exception
être → étais/étais/était/étions/étiez/étaient
J'étais fatigué.
vs. PC
imparfait = fond / PC = action
Je dormais quand il est arrivé.
Marqueurs
toujours, souvent, d'habitude → imparfait
D'habitude, il prenait le bus.
Marqueurs PC
hier, soudain, une fois → passé composé
Hier, j'ai vu un film.
