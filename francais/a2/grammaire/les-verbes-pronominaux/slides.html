Les Verbes Pronominaux — Se Lever, Se Coucher…

Formation, conjugaison et usage des verbes réfléchis et réciproques

Qu'est-ce qu'un verbe pronominal ?

Un verbe pronominal se conjugue avec un pronom réfléchi (me, te, se, nous, vous, se) qui renvoie au sujet :

  • Je me lève à 7 h. (je me lève moi-même)
  • Tu te brosses les dents.
  • Il s'appelle Thomas.

Catégories : réfléchis (action sur soi-même) · réciproques (action mutuelle) · idiomatiques (sens différent du verbe simple).

Conjugaison au présent

SujetPronomse lever
jemeje me lève
tutetu te lèves
il/elleseil/elle se lève
nousnousnous nous levons
vousvousvous vous levez
ils/ellesseils/elles se lèvent

Élision : me/te/se → m'/t'/s' devant voyelle : je m'habille.

Verbes pronominaux courants

Routine quotidienne :

  • se lever · se coucher · se réveiller · s'endormir
  • se laver · se doucher · se brosser les dents · s'habiller
  • se maquiller · se raser · se coiffer

Autres courants :

  • s'appeler · se trouver · s'asseoir · se dépêcher
  • se souvenir · s'inquiéter · se sentir · se tromper

Au passé composé — avec être

Tous les verbes pronominaux forment leur passé composé avec être :

sujet + pronom réfléchi + être (conjugué) + participe passé

  • Je me suis levé(e) à 7 h.
  • Elle s'est habillée rapidement.
  • Nous nous sommes dépêchés.

Le participe s'accorde avec le sujet (sauf si COD après le verbe) :
Elle s'est lavé les mains. (les mains = COD après → pas d'accord)

La négation et les verbes réciproques

Négation : ne encadre le pronom réfléchi + verbe :
Je ne me lève pas tard. / Elle ne s'est pas coiffée.

Réciproques (action mutuelle entre deux sujets ou plus) :

  • Ils se téléphonent tous les soirs. (ils téléphonent l'un à l'autre)
  • Nous nous aimons beaucoup.
  • Elles se connaissent depuis dix ans.

Les erreurs les plus courantes des apprenants

Voici les erreurs que les étudiants font le plus souvent.

Je me suis levé à 7 h.
→
Correct !
Se lever au passé composé → être + participe accordé (levé pour masc.).
Elle s'est lavé.
→
Elle s'est lavée.
Pas de COD après le verbe → le participe s'accorde avec le sujet (féminin) : lavée.
Il se est rasé.
→
Il s'est rasé.
Se + est → s'est. Élision obligatoire devant voyelle.
Nous nous sommes rencontré.
→
Nous nous sommes rencontrés.
Verbe réciproque + sujet masculin pluriel → accord : rencontrés.
Je me lève pas.
→
Je ne me lève pas.
La négation encadre pronom + verbe : je ne me lève pas.

Résumé : Les Verbes Pronominaux

Pronoms
me/te/se/nous/vous/se (élision m'/t'/s')
Je me lève. / Je m'habille.
Présent
pronom réfléchi + verbe conjugué
Tu te couches tôt.
PC
être + participe accordé avec sujet
Elle s'est levée. / Ils se sont levés.
Accord
accord sauf si COD après le verbe
Elle s'est lavé les mains. (les mains = COD)
Négation
ne + pronom + verbe + pas
Je ne me souviens pas. / Il ne s'est pas rasé.
Réciproque
action mutuelle (nous/vous/ils)
Ils se téléphonent chaque soir.
