Les Pronoms Compléments — COD et COI — Imprimables

Référence, exercices et clé des réponses à imprimer.

Référence : Les Pronoms COD/COI

Concept Forme Exemple
COD sing.me/te/le/la/l' — devant verbeJe le vois. / Je l'aime.
COD pl.nous/vous/les — devant verbeJe les connais.
COI sing.me/te/lui — verbe + à qqnJe lui téléphone. (à lui/elle)
COI pl.nous/vous/leur — verbe + à qqnJe leur écris. (à eux/elles)
Positionavant le verbe (ou auxiliaire)Je l'ai vu. / Je vais le faire.
Impératif +après le verbe avec tiretRegarde-le ! / Dis-lui !

Erreurs fréquentes

Incorrect Correct Explication
Je lui vois.Je le/la vois.« Voir » prend un COD (sans préposition) → le/la, pas lui.
Je les téléphone.Je leur téléphone.« Téléphoner à » prend un COI → leur (pluriel), pas les.
Je l'ai le mangé.Je l'ai mangé.Un seul pronom COD suffit : je l'ai mangé, pas je l'ai le mangé.
Mange-le pas !Ne le mange pas !À l'impératif négatif, le pronom se replace avant le verbe : ne le mange pas.
Je lui connais.Je le/la connais.« Connaître » prend un COD → le/la, pas lui.

Exercice 1 — Choisir le bon pronom

COD ou COI ?

1. Je vois Marie. → Je ___ vois. [lui / la / leur / les]
2. Je parle à mes amis. → Je ___ parle. [les / le / leur / lui]
3. Il invite ses collègues. → Il ___ invite. [leur / lui / les / le]
4. Elle écrit à son frère. → Elle ___ écrit. [le / la / les / lui]

Exercice 2 — Remplacer par un pronom

Réécrivez en remplaçant le complément souligné.

1. Je mange la pizza. → Je ___ mange.  
2. Il appelle ses parents. → Il ___ appelle.  
3. Elle téléphone à sa mère. → Elle ___ téléphone.  
4. Nous envoyons un message à nos amis. → Nous ___ envoyons un message.  
5. Tu vois le film ? → Tu ___ vois ?  

Exercice 3 — Identifier l'erreur

Quelle phrase est incorrecte ?

1. Laquelle est incorrecte ? [Je le vois. / Je lui parle. / Je lui vois. / Je leur écris.]
2. Quelle phrase a une erreur de position ? [Je le mange. / Je l'ai mangé. / Mange-le ! / Je le mange pas.]
3. Quel pronom est faux ? [Je les invite. (amis, pl.) / Je lui téléphone. (à Marie) / Je les téléphone. (à mes amis) / Je lui écris. (à Thomas)]

Clé des réponses

Exercice 1 : 1. la 2. leur 3. les 4. lui

Exercice 2 : 1. la 2. les 3. lui 4. leur 5. le

Exercice 3 : 1. Je lui vois. 2. Je le mange pas. 3. Je les téléphone. (à mes amis)
